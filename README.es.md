# Hyperledger Iroha

[![Licencia](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Hyperledger Iroha es un libro mayor de blockchain simple y eficiente construido
sobre **distributed ledger technology (DLT)**. Su diseño adopta la mentalidad
Kaizen japonesa de eliminar el exceso (*muri*) para ofrecer una implementación
robusta.

Iroha te ayuda a gestionar cuentas, activos y almacenamiento de datos en cadena
mediante contratos inteligentes eficientes, manteniendo tolerancia tanto a
fallos bizantinos como a fallos por caída.
