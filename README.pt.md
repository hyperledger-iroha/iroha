# Hyperledger Iroha

[![Licença](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Hyperledger Iroha é um livro‑razão de blockchain simples e eficiente construído
sobre **distributed ledger technology (DLT)**. O design segue a mentalidade
japonesa de Kaizen para eliminar excessos (*muri*) e oferecer uma
implementação robusta.

O Iroha ajuda você a gerenciar contas, ativos e armazenamento de dados on‑chain
por meio de contratos inteligentes eficientes, mantendo tolerância tanto a
falhas bizantinas quanto a falhas por crash.
