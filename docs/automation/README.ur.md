# دستاویزات آٹومیشن کے بنیادی خطوط

<div dir="rtl">

یہ ڈائریکٹری اُن آٹومیشن سطحوں کو محفوظ کرتی ہے جن کا حوالہ روڈمیپ کے AND5/AND6
(Android Developer Experience + Release Readiness) اور DA-1 (Data Availability
تھریٹ ماڈل آٹومیشن) کے آئٹمز آڈٹ کے قابل دستاویزی شواہد کے لیے دیتے ہیں۔
کمانڈ حوالوں اور متوقع آؤٹ پٹس کو ریپو میں رکھنا کمپلائنس ریویوز کے لیے درکار
شرائط کو اس وقت بھی دستیاب رکھتا ہے جب CI پائپ لائنز یا ڈیش بورڈز آف لائن ہوں۔

## ڈائریکٹری ساخت

| راستہ | مقصد |
|-------|------|
| `docs/automation/android/` | Android ڈاکیومنٹیشن اور لوکلائزیشن آٹومیشن کے بیس لائنز (AND5)، جن میں i18n stubs کے سنک لاگز، پیریٹی خلاصے، اور AND6 منظوری سے پہلے درکار SDK پبلشنگ شواہد شامل ہیں۔ |
| `docs/automation/da/` | Data Availability تھریٹ ماڈل آٹومیشن کے آؤٹ پٹس جن کا حوالہ `cargo xtask da-threat-model-report` اور رات کی ڈاکیومنٹیشن ریفریش دیتے ہیں۔ |

ہر ذیلی ڈائریکٹری اُن کمانڈز کی وضاحت کرتی ہے جو شواہد تیار کرتی ہیں اور اس
فائل لے آؤٹ کی بھی نشاندہی کرتی ہے جسے ہم چیک اِن کرنا چاہتے ہیں (عموماً JSON
خلاصے، لاگز یا مینفیٹس)۔ جب آٹومیشن رن شائع شدہ ڈاکس میں اہم تبدیلی کرے تو ٹیمیں
متعلقہ فولڈر میں نئے آؤٹ پٹس رکھتی ہیں، پھر status/roadmap کی متعلقہ اینٹری میں
کمٹ کا حوالہ دیتی ہیں۔

## استعمال

1. **آٹومیشن چلائیں** اور ذیلی README میں دی گئی کمانڈز استعمال کریں
   (مثلاً `ci/check_android_fixtures.sh` یا `cargo xtask da-threat-model-report`)۔
2. **حاصل ہونے والے JSON/لاگ آؤٹ پٹس کو کاپی کریں** اور انہیں `artifacts/…` سے
   متعلقہ `docs/automation/<program>/…` فولڈر میں رکھیں، فائل نام میں ISO-8601
   ٹائم اسٹیمپ شامل کریں تاکہ آڈیٹرز گورننس منٹس سے شواہد جوڑ سکیں۔
3. **کمٹ کا حوالہ دیں** اور روڈمیپ گیٹ بند کرتے وقت `status.md`/`roadmap.md` میں
   ریکارڈ کریں تاکہ ریویورز استعمال شدہ آٹومیشن بیس لائن کی تصدیق کر سکیں۔
4. **فائلیں ہلکی رکھیں۔** توقع ساختہ میٹا ڈیٹا، مینفیٹس یا خلاصوں کی ہے، بڑے
   بائنری بلاپس کی نہیں۔ بڑے ڈمپس آبجیکٹ اسٹوریج میں رہیں اور اس کی دستخط شدہ
   حوالہ جاتی اندراج یہاں درج کریں۔

ان آٹومیشن نوٹس کو مرکزی بنانے سے AND6 کی "docs/automation بیس لائنز آڈٹ کے لیے
دستیاب" شرط پوری ہوتی ہے اور DA تھریٹ ماڈل فلو کے لیے رات کے رپورٹس اور دستی
اسپاٹ چیکس کا ایک متعین گھر فراہم ہوتا ہے۔
