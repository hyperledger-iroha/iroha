# MOCHI དཀའ་ངལ་སེལ་ཐབས་ལམ་སྟོན།

ཉེ་གནས་ MOCHI ཀླད་ཀོར་ཚུ་འགོ་བཙུགས་མ་བཏུབ་པའི་སྐབས་ རན་བུ་འདི་ལག་ལེན་འཐབ།
ལོག་འགོ་བཙུགས་ལོག་, ཡང་ན་ སྡེབ་ཚན་/བྱུང་ལས་/གནས་སྟངས་དུས་མཐུན་བཟོ་ནི་ཚུ་ རྒྱུན་སྤེལ་འབད་ནི་ཚུ་བཀག་བཞག། འདི་གིས་རྒྱ་སྐྱེད་འབདཝ་ཨིན།
ལམ་སྟོན་གྱི་རྣམ་གྲངས་ “Documentation & brout” ཟེར་མི་འདི་ ལྟ་རྟོག་པ་གི་སྤྱོད་ལམ་ཚུ་ ༢༠༡༢ ལུ་ བསྒྱུར་བཅོས་འབད་དེ་ཨིན།
`mochi-core` དངོས་གནས་སླར་གསོའི་རིམ་པ་ནང་།

## 1. དང་པོ་ལན་འདེབས་ཐོ་ཡིག

༡ MOCHI ལག་ལེན་འཐབ་བཞིན་པའི་ གནད་སྡུད་རྩ་བ་འདི་ འཛིན་དགོ། སྔོན་སྒྲིག་འོག་གི་ཚུ།
   `$TMPDIR/mochi/<profile-slug>`; སྲོལ་སྒྲིག་འགྲུལ་ལམ་ཚུ་ ཡུ་ཨའི་ མགོ་མིང་ཕྲ་རིང་ནང་འབྱུངམ་ཨིན།
   བརྒྱུད་དེ་ `cargo run -p mochi-ui-egui -- --data-root ...`.
2. ལཱ་གི་ས་སྒོ་རྩ་བ་ལས་ `./ci/check_mochi.sh` གཡོག་བཀོལ། འདི་གིས་ ཀོར་འདི་བདེན་དཔྱད་འབདཝ་ཨིན།
   ཁྱོད་ཀྱིས་རིམ་སྒྲིག་ཚུ་ལེགས་བཅོས་འབད་ནི་འགོ་མ་བཙུགས་པའི་ཧེ་མ་ ཡུ་ཨའི་དང་ མཉམ་བསྡོམས་ཀེརེཊི།
༣ སྔོན་སྒྲིག་ (`single-peer` ཡང་ན་ `four-peer-bft`) ལུ་དྲན་འཛིན་འབད། བ ར་བའི་ ས་ ི་ ས་ ་ ས།
   གནད་སྡུད་རྩ་བའི་འོག་ལུ་ ཁྱོད་ཀྱིས་ མཉམ་རོགས་སྣོད་འཛིན་/དྲན་ཐོ་ག་དེམ་ཅིག་རེ་བ་བསྐྱེད་དགོཔ་ཨིན་ན་ གཏན་འབེབས་བཟོཝ་ཨིན།

## 2. དྲན་ཐོ་ཚུ་བསྡུ་སྒྲིག་དང་ ཊེ་ལི་མེ་ཊི་ར་ཊི་གི་སྒྲུབ་བྱེད།

`NetworkPaths::ensure` (`mochi/mochi-core/src/config.rs`) གིས་གཏན་ཏོག་ཏོ་གསར་བསྐྲུན་འབདཝ་ཨིན།
སྒྲིག༌བཀོད:

```
<data_root>/<profile>/
  peers/<alias>/...
  logs/<alias>.log
  genesis/
  snapshots/
```

བསྒྱུར་བཅོས་ཚུ་མ་འབད་བའི་ཧེ་མ་ རིམ་པ་འདི་ཚུ་ རྗེས་སུ་འཇུག་ནི།

- མཇུག་འདི་བཟུང་ནི་ལུ་ **ལོག་** མཆོང་ལྡེ་ ཡང་ན་ `logs/<alias>.log` ཐད་ཀར་དུ་ཁ་ཕྱེ་དགོ།
  མཉམ་རོགས་རེ་ལུ་ ཐིག་ ༢༠༠ རེ། ལྟ་རྟོག་པ་དེ་ stdout/stderr/རིམ་ལུགས་ཀྱི་ རྒྱུན་ལམ་ཚུ་ མཇུག་བསྡུཝ་ཨིན།
  བརྒྱུད་དེ་ `PeerLogStream`, དེ་འབདཝ་ལས་ ཡིག་སྣོད་འདི་ཚུ་ ཡུ་ཨའི་ཨའུཊི་པུཊི་དང་མཐུན་སྒྲིག་འབདཝ་ཨིན།
- **རྒྱུན་སྐྱོང་ → པར་ལེན་འཕྲུལ་ཆས་བརྒྱུད་དེ་ པར་ལེན་ཅིག་ཕྱིར་འདྲེན་འབད།
  `Supervisor::export_snapshot`). པར་ཆས་བསག་བཞག་དང་ རིམ་སྒྲིག་ དེ་ལས་ དང་།
  `snapshots/<timestamp>-<label>/` ནང་ལུ་དྲན་ཐོ་བཀོད།
- གནད་དོན་དེ་གིས་ རྒྱུན་ལམ་ཝིཌི་གེཊི་ཚུ་ཚུདཔ་ཨིན་པ་ཅིན་ `ManagedBlockStream` འདྲ་བཤུས་རྐྱབ་པ་ཅིན་,
  Kagami, དང་ `ManagedStatusStream` གསོ་བའི་བརྡ་སྟོན།
  ཌེཤ་བོརཌ་. ཡུ་ཨའི་གིས་ མཇུག་གི་ལོག་མཐུད་ནིའི་དཔའ་བཅམ་མི་དང་ འཛོལ་བ་གི་རྒྱུ་མཚན་ཚུ་ ཁ་ཐོག་ལུ་བཏོནམ་ཨིན། བཟུང་བ་
  བྱུང་རྐྱེན་དྲན་ཐོ་གི་དོན་ལུ་ གསལ་གཞི་ཅིག་བཏོན་ཡོདཔ།

## 3. མཉམ་རོགས་འགོ་བཙུགས་ཀྱི་གནད་དོན་ཚུ་སེལ་ཐབས་འབད་ནི།

མཉམ་རོགས་ཀྱི་འགོ་འབྱེད་ཀྱི་འཐུས་ཤོར་མང་ཤོས་རང་ བཱ་ཀེཊ་གསུམ་ལུ་འགྱུརཝ་ཨིན།

### གཉིས་ལྡན་ཡང་ན་ ངན་ལྷད་ངན་པ་ཚུ།

`SupervisorBuilder` `irohad`, `kagami`, དང་ (མ་འོངས་པ) `iroha_cli` ལུ་ ཕྱིར་ཐོན་འབདཝ་ཨིན།
གལ་སྲིད་ ཡུ་ཨའི་སྙན་ཞུ་འདི་གིས་ “བཤུབ་མ་བཏུབ་པའི་བྱ་རིམ་ལུ་འཐུས་ཤོར་བྱུང་ཡོདཔ་” ཡང་ན་ “གནང་བ་བཀག་ཆ་འབད་ཡོདཔ་”
at ཤེས་རྟོགས་ཡོད་པའི་གཉིས་ལྡན་ཚུ།

```bash
cargo run -p mochi-ui-egui -- \
  --irohad /path/to/irohad \
  --kagami /path/to/kagami \
  --iroha-cli /path/to/iroha_cli
```

ཁྱོད་ཀྱིས་ `MOCHI_IROHAD`, `MOCHI_KAGAMI`, དང་ `MOCHI_IROHA_CLI` འདི་ བཀག་ཐབས་ལུ་གཞི་སྒྲིག་འབད་ཚུགས།
རྒྱལ་དར་ཚུ་ བསྐྱར་ལོག་སྦེ་ཡིག་དཔར་རྐྱབ། རྐྱེན་སེལ་གྱི་བཱན་ཌལ་བཟོ་བསྐྲུན་འབད་བའི་སྐབས་ ག་བསྡུར་འབད།
`BundleConfig` ནང་ `mochi/mochi-ui-egui/src/config/` ནང་ ནང་འགྲུལ་ལམ་ཚུ་ལུ་རྒྱབ་འགལ་འབདཝ་ཨིན།
`target/mochi-bundle`.

### འདྲེན་བཀོལ།

`PortAllocator` རིམ་སྒྲིག་ཚུ་མ་འབྲི་བའི་ཧེ་མ་ ལོག་བེག་ངོས་འདྲ་བ་འདི་ཞིབ་དཔྱད་འབདཝ་ཨིན། མཐོང་བ་ན།
`failed to allocate Torii port` ཡང་ན་ `failed to allocate P2P port`, གཞན་ཡང་།
བྱ་རིམ་འདི་ ཧེ་མ་ལས་རང་ སྔོན་སྒྲིག་ཁྱབ་ཚད་ (༨༠༨༠/༡༣༣༧) གུ་ཉན་བཞིན་ཡོདཔ་ཨིན། Reanch MOCHI
གསལ་སྡུད་གཞི་རིམ་དང་མཉམ་དུ།

```bash
cargo run -p mochi-ui-egui -- --torii-start 12000 --p2p-start 19000
```

བཟོ་བསྐྲུན་པ་གིས་ གཞི་རྟེན་དེ་ཚུ་ལས་ རིམ་སྒྲིག་འདྲེན་ལམ་ཚུ་ རླུང་འཚུབ་འབད་འོང་ དེ་འབདཝ་ལས་ ཁྱབ་ཚད་ཅིག་ བཀག་བཞག་དགོ།
ཁྱོད་ཀྱི་སྔོན་སྒྲིག་དོན་ལུ་ཚད་གཞི། (`peer_count` མཉམ་རོགས་ → `peer_count` འདྲེན་ལམ་ཚུ་སྐྱེལ་འདྲེན་རེ་ལུ་)།

### བསགས་བཟོའི་ངན་ལྷད་དང་བསྒྲིགས།གལ་སྲིད་ Kagami འདི་ གསལ་སྟོན་མ་འབད་བའི་ཧེ་མ་ ཕྱིར་འཐོན་འབད་བ་ཅིན་ མཉམ་རོགས་ཚུ་ དེ་འཕྲོ་ལས་ བརྡབ་འགྱོ་འོང་། དབྱེ༌ཞིབ
`genesis/*.json`/`.toml` གནད་སྡུད་རྩ་བ་ནང་། བསྐྱར་དུ་བླངས།
`--kagami /path/to/kagami` ཡང་ན་ གཡས་ཕྱོགས་གཉིས་ལྡན་གྱི་ **སྒྲིག་སྟངས་*** ཌའི་ལོག་འདི་ དཔག་བྱེད་འབད།
བསག་བཞག་གི་ངན་ལྷད་ཀྱི་དོན་ལུ་ བདག་འཛིན་དབྱེ་ཚན་གྱི་ **Wipe & re-genesis** ལག་ལེན་འཐབ།
ལགཔ་གིས་སྣོད་འཛིན་ཚུ་བཏོན་གཏང་ནིའི་ཚབ་ལུ་ ཨེབ་རྟ་(འོག་ལུ་བཀབ་ཡོདཔ་)། འདི་ལོག་གསར་བསྐྲུན་འབདཝ་ཨིན།
ལས་སྦྱོར་ཚུ་ ལོག་འགོ་མ་བཙུགས་པའི་ཧེ་མ་ མཉམ་རོགས་སྣོད་ཐོ་དང་ པར་ལེན་རྩ་བ།

### སྒྲིག་བཀོད་རང་འགུལ་ལོག་འགོ་བཙུགས།

`[supervisor.restart]` ནང་ `config/local.toml` (ཡང་ན་ སི་ཨེལ་ཨའི་དར་ཚད།
