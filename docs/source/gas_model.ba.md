# IVM газ моделе

Был документта Iroha виртуаль машина өсөн канон газ графикын билдәләй.
(IVM) һәм график нисек хешировать һәм ҡулланыу аңлата. Дөрөҫлөк сығанағы
өсөн сығымдар `crates/ivm/src/gas.rs`; график таблицаһы түбәндә күрһәтелгән
ҡараш шул канонлы карта төҙөү.

## Масштаб

- IVM байткод башҡарыуына ҡағыла (Һуғыш::Ivm).
- ISI газын иҫәпләү `crates/iroha_core/src/gas.rs`-та айырым билдәләнә.
- ISO 20022 opcodes ABI v1-ҙә һаҡлана һәм әлегә газ яҙмаларын йөрөтмәй.

## Детерминизм һәм график хеш

Газ графигы детерминистик һәм опкодтан алынған → сығымдар парҙары. 1990 й.
канонлы һеңдерелгән опкодтар исемлеге аша һәр яҙма менән иҫәпләнә.
сериялы:

- опкод байт (u8)
- хаҡы (u64, аҙ-андочный)

Хэш фашлана:

- `ivm::gas::schedule_hash()` (канон график хеш)
- `ivm::limits::schedule_hash()` (эшкә ҡараған псевдоним)

Был distest ҡулланыу өсөн раҫлау өсөн, бөтә тиҫтерҙәре бер үк график менән бүлешергә, ҡасан проводка .
конфиг йәки телеметрия тикшерелеүҙәре.

## Вектор масштаблау һәм HTM ретия

- Вектор опстары (`VADD*`, `VAND`, `VXOR`, `VOR`, `VROT32`) логик менән
  вектор оҙонлоғо `SETVL` тарафынан ҡуйылған. Таблицала база сығымдары 2012 йылға тиклем масштаблана.
  `min(vector_len, VECTOR_BASE_LANES) / VECTOR_BASE_LANES` (база = 2 һыҙат).
- HTM реттары `(retries + 1)` X-ға сығымдарҙы ҡабатлай; күпселек консенсус юлдары юҡ
  ретиялар үҙ өҫтөнә ала.
