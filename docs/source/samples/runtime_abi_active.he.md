# ‏ABI בזמן ריצה — גרסאות פעילות (Torii)

נקודת קצה
- `GET /v1/runtime/abi/active`

תגובה (מהדורה ראשונה; ABI יחיד)
```json
{
  "abi_version": 1
}
```

הערות
- במהדורה הראשונה גרסת ה-ABI קבועה ל-1, ולכן נקודת הקצה הזאת תמיד מחזירה `1`.
