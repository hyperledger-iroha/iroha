# Kotodama компилятор хаталар кодтары

Kotodama компиляторы тотороҡло хата кодтарын сығара, шулай итеп, инструменттар һәм CLI ҡулланыусылар ала
тиҙ арала уңышһыҙлыҡтың сәбәбен аңлай. Ҡулланыу `koto_compile --explain <code>`
тейешле кәңәште баҫтырыу өсөн.

| Код | Тасуирлама | Типик төҙәтеү |
|------|-------------|-------------||
| `E0001` | Блика маҡсатлы диапазоны өсөн IVM һикерергә кодлау. | Бик ҙур функциялар йәки кәметергә inlining шулай төп блок арауыҡтары эсендә ҡала ±1MiB. |
| `E0002` | Шылтыратыу сайттары һылтанма функцияһы, тип, бер ҡасан да билдәләнмәгән. | Типос тикшерергә, күренеш модификаторҙары, йәки флагтар функцияһы, улар шылтыратыусыны алып ташланы. |
| `E0003` | Ҡаты дәүләт syscalls ABI v1 эшләмәйенсә сығарылған. | `CompilerOptions::abi_version = 1` комплекты йәки `seiyaku`X контракт эсендә `meta { abi_version: 1 }` өҫтәй. |
| `E0004` | Активтар менән бәйле syscalls туранан-тура булмаған күрһәткестәр ала. | Ҡулланыу `account_id(...)`, `asset_definition(...)`, һ. |
| `E0005` | `for`-цикл инициализаторы бөгөнгөгә ҡарағанда ҡатмарлыраҡ. | Ҡатмарлы ҡоролма иллюзия алдында күсерергә; тик ябай `let`/экспрессия инициаторҙары әлеге ваҡытта ҡабул ителә. |
| `E0006` | `for`-цикл аҙым пункты бөгөнгөгә ҡарағанда ҡатмарлыраҡ. | Яңыртыу иллюминатор счетчик менән ябай һүҙбәйләнеш (мәҫәлән, `i = i + 1`). |
