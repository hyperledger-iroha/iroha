# Ағымдағы дәүләт (v1)
- Идара итеү тәҡдимдәре буйынса эшләй: тәҡдим итеүсе → референдум → иҫәп → ҡабул итеү. Референдум тәҙрәләре һәм һайлаусылар һаны/раҫлау сиктәре `gov.md`-та һүрәтләнгәнсә үтәлә; йоҙаҡтары ғына һуҙылған һәм срогы буйынса блокировканы аса.
- Парламент һайлау VRF-нигеҙендә һыҙмаларҙы ҡуллана, детерминистик тәртип һәм термин сиктәре менән; ҡасан бер ниндәй ҙә һаҡланған исемлек юҡ, Torii `gov.parliament_*` конфиг ҡулланып, fallback ала. Совет ҡапҡаһы һәм кворум тикшерелгән `gov_parliament_bodies` / `gov_pipeline_sla` тестары.
- Тауыш биреүҙең режимдары: З.К. (ғәҙәти, `Active` ВК рәт-байттары менән) һәм Ябай (квадрат ауырлығы) талап итә. Режим тап килмәүҙәре кире ҡағыла; блокировка булдырыу/оҙайтыу ике режимда ла монотонлы, ZK һәм ябай яңынан тауыш биреүҙең регрессия һынауҙары менән.
- Валидатор тәртипһеҙлектәре дәлилдәр торбаһы аша (`/v2/sumeragi/evidence*`, CLI ярҙамсылары) аша үткәрелгән, улар `NextMode`X + `ModeActivationHeight` тарафынан үткәрелгән берлектәге консенсус ҡул-оффтары менән.
- Һаҡланған исемдәр киңлеге, йөрөү ваҡытлы яңыртыу ҡармаҡтары, һәм идара итеү манифест ҡабул итеү документлаштырылған `governance_api.md` һәм телеметрия менән ҡапланған (`governance_manifest_*`, `governance_protected_namespace_total`).

# Осоу / артта ҡалған
- Баҫма VRF артефакттарҙы (орлоҡ, иҫбатлау, тәртипкә килтерелгән исемлек, алмаштар) һыҙырыу һәм күрһәтеү өсөн алмаштырыу ҡағиҙәләрен кодификациялау; йәрәбә һәм алмаштырыу өсөн алтын ҡорамалдар өҫтәй.
- Парламент органдары өсөн сәхнә-SLA үтәү (ҡағиҙәләр → көн тәртибе → өйрәнеү → тикшерергә → жюри → ҡабул итеү) кәрәк асыҡ таймерҙар, эскалация юлдары, һәм телеметрия иҫәпләүселәр.
- Сәйәсәт-присяжный сер/коммит-разведка тауыш биреүҙе һәм улар менән бәйле ришүәтселек-ҡаршылыҡ аудиты әлегә тормошҡа ашырырға тейеш.
- Роль-бәйләнеш множитель, тәртипһеҙлектәр өсөн ҡырҡылған юғары хәүефле тәндәр, һәм хеҙмәтләндереүҙең слоттар араһында һыуытыу конфигурация сантехника плюс һынауҙар талап итә.
- Идара итеү һыҙаты герметизацияһы һәм референт тәҙрәһе/йөкләү ҡапҡалары `gov.md`/`status.md`-та күҙәтелә; юл картаһы яҙмаларын яңыртыу, ҡалған ҡабул итеү һынауҙары ерҙәре.
