དེ་ཚུ་ `fastpq.metallib` ནང་ལུ་འབྲེལ་མཐུད་འབད་དེ་ གོང་ལས་ཐོ་བཀོད་འབད་ཡོད་པའི་ འཛུལ་སྒོ་ག་ར་ཕྱིར་འདྲེན་འབདཝ་ཨིན།
`FASTPQ_METAL_LIB` འགྲུལ་ལམ་དེ་ལུ་གཞི་སྒྲིག་འབད་དོ་ (བཟོ་བསྐྲུན་ཡིག་ཚུགས་འདི་གིས་ འདི་གིས་འབདཝ་ཨིན།
རང་བཞིན་གྱིས་ རང་བཞིན་གྱིས་) གིས་ དཔེ་མཛོད་འདི་ མ་བལྟ་བར་ གཏན་འབེབས་བཟོ་ནི་ལུ་ འབད་བཅུགཔ་ཨིན།
གང་དུ་ `cargo` གིས་ བཟོ་བསྐྲུན་གྱི་ཅ་ལག་ཚུ་ བཙུགས་ཡོདཔ་ཨིན།

སི་ཨའི་གཡོག་བཀོལ་མི་དང་གཅིག་ཁར་ མཉམ་བསྡོམས་དོན་ལུ་ ཁྱོད་ཀྱིས་ དཔེ་མཛོད་འདི་ ལག་ཐོག་ལས་ བསྐྱར་བཟོ་འབད་ཚུགས།

```bash
export OUT_DIR=$PWD/target/metal && mkdir -p "$OUT_DIR"
xcrun metal -std=metal3.0 -O3 -c crates/fastpq_prover/metal/kernels/ntt_stage.metal -o "$OUT_DIR/ntt_stage.air"
xcrun metal -std=metal3.0 -O3 -c crates/fastpq_prover/metal/kernels/poseidon2.metal -o "$OUT_DIR/poseidon2.air"
xcrun metallib "$OUT_DIR/ntt_stage.air" "$OUT_DIR/poseidon2.air" -o "$OUT_DIR/fastpq.metallib"
export FASTPQ_METAL_LIB="$OUT_DIR/fastpq.metallib"
```
