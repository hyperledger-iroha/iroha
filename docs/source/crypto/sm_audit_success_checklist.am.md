# ዓላማ

ይህ የማረጋገጫ ዝርዝር ለስኬት የሚያስፈልጉትን ተጨባጭ መመዘኛዎች ይይዛል
የ SM2/SM3/SM4 የውጭ ኦዲት ማጠናቀቅ። ወቅት መከለስ አለበት
መጀመር፣ በእያንዳንዱ የሁኔታ ፍተሻ ነጥብ ላይ በድጋሚ የተጎበኘ እና መውጣቱን ለማረጋገጥ ጥቅም ላይ ይውላል
ለምርት አረጋጋጮች SM መፈረም ከማንቃት በፊት ሁኔታዎች።
