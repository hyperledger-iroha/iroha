# መመሪያ

በባህር ዳርቻ ላይ የማሰማራት አጠቃቀምን ከ SCA ቢሮ ጋር ሲያስገቡ ይህንን አብነት ይጠቀሙ
ኦፕሬተሮች. በአንድ የማሰማሪያ ክላስተር ወይም የውሂብ ቦታ አንድ ግቤት ያቅርቡ። አዘምን
ቦታ ያዢዎቹ ከዋኝ-ተኮር ዝርዝሮች ጋር እና የተዘረዘሩትን ማስረጃዎች ያያይዙ
በማረጋገጫ ዝርዝር ውስጥ.

# 3. ክሪፕቶግራፊክ መቆጣጠሪያዎች

- የተፈቀዱ ስልተ ቀመሮች፡ {{ ALGORITHMS }} (ኤስኤምኤስ ማዋቀሩን ከውቅር ጋር እንደሚዛመድ ያረጋግጡ)።
- ቁልፍ የሕይወት ዑደት ማጠቃለያ:
  | መድረክ | መግለጫ |
  |-------|-----------|
  | ትውልድ | {{KEY_generation}} |
  | ማከማቻ | {{ ቁልፍ_ማከማቻ }} |
  | ማሽከርከር | {{ ቁልፍ_ROTATION }} |
  | መሻር | {{ ቁልፍ_መሻሻያ}} |
- የተለየ ማንነት (`distid`) ፖሊሲ፡ {{ DISTID_POLICY }}
- የማዋቀር ቅንጭብ (`crypto` ክፍል)፡ Norito/JSON ቅጽበተ ፎቶ ከሃሽ ጋር ያቅርቡ።

# 4. ቴሌሜትሪ እና የኦዲት መንገዶች

- የመጨረሻ ነጥቦችን መከታተል፡ {{METRICS_ENDPOINTS }} (`/metrics`፣ ዳሽቦርዶች)።
- የተመዘገቡ መለኪያዎች፡ `crypto.sm.verification_total`፣ `crypto.sm.sign_total`፣
  የዘገየ ሂስቶግራም, የስህተት ቆጣሪዎች.
- የማቆያ መመሪያ፡ {{LOG_RETENTION }} (≥ ሶስት አመት ይመከራል)።
- የኦዲት ምዝግብ ማስታወሻ ማከማቻ ቦታ፡ {{ AUDIT_STORAGE }}

# 6. አባሪዎች ዝርዝር- [ ] የማዋቀር ቅጽበታዊ ገጽ እይታ (Norito + JSON) ከሃሽ ጋር።
- [] የመወሰን ግንባታ ማረጋገጫ (hashes፣ SBOM፣ reproducibility notes)።
- [] ቴሌሜትሪ ዳሽቦርድ ወደ ውጭ መላክ እና የማንቂያ መግለጫዎች።
- [ ] የአደጋ ምላሽ እቅድ እና የጥሪ ማዞሪያ ሰነድ።
- [] የኦፕሬተር ማሰልጠኛ እውቅና ወይም runbook ደረሰኝ።
- [ ] ወደ ውጭ የሚላኩ-ቁጥጥር መግለጫ ቅርሶችን ማንጸባረቅ።
- [ ] ተዛማጅነት ያላቸው የውል ስምምነቶች ቅጂዎች ወይም የፖሊሲ ውድቀቶች።

# 7. የኦፕሬተር መግለጫ

> ከላይ የተዘረዘረው ስምሪት ከPRC ንግድ ጋር የሚስማማ መሆኑን አረጋግጠናል።
> የስክሪፕቶግራፊ ደንቦች፣ በኤስኤምኤስ የነቁ አገልግሎቶች የተመዘገቡትን ይከተላሉ
> የአደጋ ምላሽ እና የቴሌሜትሪ ፖሊሲዎች፣ እና ያ የኦዲት ቅርሶች ይሆናሉ
> ቢያንስ ለሦስት ዓመታት ተይዟል.
