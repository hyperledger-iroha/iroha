## རིལ་ཚང

- སི་པི་ཡུ་ zstd དང་འདྲ་མཚུངས་སྦེ་ ཌི་ཀོཌི་འབད་མི་ ཚད་ལྡན་ zstd གཞི་ཁྲམ་ཚུ་ བཏོན་གཏང་། བཱའི་ཊི་ ཆ་ཏོམ།
  དང་གཅིག་ཁར་ སི་པི་ཡུ་ བསྡམ་བཞག་འཕྲུལ་ཆས་འདི་ དགོཔ་མེདཔ་ཨིན།
- མཐུན་རྐྱེན་དང་ འདྲེན་བྱེད་ དེ་ལས་ ཐགསཔ་གི་ལས་རིམ་ནང་ གཏན་འབེབས་ཐོན་འབྲས་ཚུ་ གཏན་འབེབས་བཟོཝ་ཨིན།
- མཐའ་མཚམས་ཞིབ་དཔྱད་དང་ སྔོན་དཔག་འབད་བཏུབ་པའི་ བཱ་ཕར་ཚེ་སྲོག་ཚུ་ གསལ་སྟོན་འབད།

## ལག་ལེན་གྱི་དྲན་ཐོ།

- ཇི་པི་ཡུ་གུ་ མཐུན་སྒྲིག་འཚོལ་ཞིབ་དང་ གོ་རིམ་བཟོ་ནི།
- གཞི་ཁྲམ་དང་ཨེན་ཊོ་པི་ཀོ་ཌིང་ (ཧྥ་མཱན་/ཨེཕ་ཨེསི་ཨི་) ད་ལྟོ་ ཧོསིཊི་གུ་གཡོག་བཀོལཝ་ཨིན།
  ཨིན་-ཀེར་ཊི་ཨིན་ཀོ་ཌར་ལག་ལེན་འཐབ་སྟེ། GPU Huffman/FSE ཀར་ནེལ་ཚུ་ ཆ་སྙོམས་བརྟག་དཔྱད་འབད་ཡོདཔ་ཨིན་རུང་མེན།
  དེ་འབདཝ་ད་ གཞི་ཁྲམ་ཆ་ཚང་ནང་ལུ་ གློག་ཐག་བཏང་ཡོདཔ་ཨིན།
- ཌི་ཀོཌི་གིས་ རྒྱབ་སྐྱོར་མ་འབད་བའི་ གཞི་ཁྲམ་ཚུ་གི་དོན་ལུ་ སི་པི་ཡུ་ zstd ཕོལབེག་དང་གཅིག་ཁར་ ཀེར་ཐིག་གཞི་ཁྲམ་ཌི་ཀོཌར་ལག་ལེན་འཐབ་ཨིན།
  ཆ་ཚང་ཇི་པི་ཡུ་བཀག་ཆའི་ཌི་ཀོཌི་ལུས་ཡོདཔ།

## ཨིན་ཀོ་ཌིང་མདོང་ལམ་ (མཐོ་རིམ)།

1. ཨིན་པུཊ་གི་གནས་སྟངས།
   - ཨིན་པུཊི་འདི་ ཐབས་འཕྲུལ་བཱ་ཕར་ཅིག་ནང་ འདྲ་བཤུས་རྐྱབས།
   - གཏན་བཟོས་ཚད་ཚད་ཀྱི་ཆ་ཤས་ (རིམ་པ་བཟོ་ནི་གི་དོན་ལུ་) དང་སྡེབ་ཚན་ (for for the for) ལུ་དབྱེ་སེལ་འབད།
     zstd གཞི་ཁྲམ་བསྡུ་སྒྲིག།
2. མཐུན་སྒྲིལ་དང་གོ་རིམ་བཏོན་པ།
   - ཇི་པི་ཡུ་ཀར་ནེལ་གྱིས་ ཅངཀ་རེ་རེ་ལུ་ པར་ལོག་བཏབ་ནི་དང་ བཀོ་བཞག་གི་རིམ་པ་ཚུ་ (ངོ་མ་གི་རིང་ཚད་, མཐུན་སྒྲིག, མཐུན་སྒྲིག།
     རིང་ཚད་, ཨོཕ་སེཊི་)།
   - གོ་རིམ་བཀོད་སྒྲིག་འདི་ བརྟན་ཏོག་ཏོ་དང་ གཏན་འབེབས་བཟོཝ་ཨིན།
3. ཚིག་གི་གྲ་སྒྲིག 3.
   - གོ་རིམ་ཚུ་གིས་ གཞི་བསྟུན་འབད་མི་ ཚིག་དོན་ཚུ་ བསྡུ་སྒྲིག་འབད།
   - ཡིག་ཆའི་སྡེབ་ཚན་ཚུ་བཟོ་བསྐྲུན་འབད་ཞིནམ་ལས་ ཡིག་ཆའི་སྡེབ་ཚན་(raw, RLE, ཡང་ན་ ཡང་ན་ ཡང་ཅིན་ ཡང་ཅིན་ སེལ་འཐུ་འབད།
     Huffman) གཏན་འཁེལ་སྦེ་ གཏན་འབེབས་བཟོ་ནི།
4. ཧྥ་མཱན་ཐིག་ཁྲམ་ (ཡིག་ཆ།)
   - ཧིསི་ཊོ་གཱརམ་ལས་ གསང་གྲངས་རིང་ཚད་ཚུ་ བཟོ་བསྐྲུན་འབད།
   - སི་པི་ཡུ་དང་མཐུན་སྒྲིག་འབད་མི་ གཏན་འབེབས་ཐགཔ་བཀག་ཆ་འབད་མི་དང་གཅིག་ཁར་ ཀེ་ནོ་ནིཀ་ཐིག་ཁྲམ་ཚུ་བཟོ་བསྐྲུན་འབད།
     zstd ཐོན་འབྲས་.
༥ ཨེཕ་ཨེསི་ཨི་ཐིག་ཁྲམ་ཚུ་ (ཨེལ་ཨེལ་/ཨེམ་ཨེལ་/ཨོ་ཨེཕ་)
   - བསྐྱར་འབྱུང་གྱངས་ཁ་ཚུ་ སྤྱིར་བཏང་ཨིན།
   - ཨེཕ་ཨེསི་ཨི་ ཌིསི་ཀོ་ཌིང་/ཨེན་ཀོ་ཌིང་ཐིག་ཁྲམ་ཚུ་ གཏན་འབེབས་བཟོ་ཐོག་ལས་ བཟོ་བསྐྲུན་འབད།
6. བིཊ་སི་ཊིམ་རྩོམ་པ་པོ།
   - བིཊི་ཆུང་ཀུ་ཚུ་ ཐུམ་སྒྲིལ་འབད་ (LSB-first).
   - བཱའིཊི་ས་མཚམས་གུ་ ཕྲངམ་ཕྲངམ་ཨིན། ཀླད་ཀོར་དང་བཅས་པའི་པེཌ་ཨིན།
   - བིཊི་རྒྱ་ཚད་གསལ་བསྒྲགས་འབད་ནི་དང་ ལྕོགས་གྲུབ་ཞིབ་དཔྱད་ཚུ་ བསྟར་སྤྱོད་འབད་ནི་ལུ་ གནས་གོང་ཚུ།
7. བཀག་ཆ་དང་གཞི་ཁྲམ་བསྡུ་སྒྲིག།
   - སྡེབ་ཚན་མགོ་ཡིག་ཚུ་ (དབྱེ་བ་, ཚད་, མཇུག་གི་བཀག་ཆ་དར་ཆ་) ལུ་གཞི་བསྟུན་འབད།
   - བསྡམ་བཞག་ཡོད་པའི་སྡེབ་ཚན་ཚུ་ནང་ ཚིག་དོན་དང་གོ་རིམ་ཚུ་ རིམ་སྒྲིག་འབད།
   - ཚད་ལྡན་ zstd གཞི་ཁྲམ་མགོ་ཡིག་ཚུ་དང་ གདམ་ཁ་ཅན་གྱི་ཞིབ་དཔྱད་སམ་ཚུ་ བཏོན་གཏང་།

## ཌི་ཀོ་ཌིང་མདོང་ལམ་ (མཐོ་རིམ)

1. གཞི་ཁྲམ་དཔྱད་ཞིབ།
   - མགུ་སྐོར་བཱའིཊི་དང་ སྒོ་སྒྲིག་སྒྲིག་སྟངས་ དེ་ལས་ གཞི་ཁྲམ་མགོ་ཡིག་ས་སྒོ་ཚུ་ བདེན་དཔྱད་འབད།
2. བིཊ་སི་ཊིམ་ལྷག་མི།
   - མཐའ་མཚམས་ཞིབ་དཔྱད་དམ་དམ་ཚུ་དང་གཅིག་ཁར་ ཨེལ་ཨེསི་བི་-དང་པ་བིཊི་རིམ་པ་ཚུ་ལྷག།
3. ཚིག་དོན་གསལ་གཞི།
   - ཌི་ཀོཌ་ ཚིག་དོན་བཀག་ཆ་ (raw, RLE, ཡང་ན་ Huffman) ནང་ ཚིག་དོན་བཱ་ཕར་ནང་།
4. གོ་རིམ་གསལ་བཤད།
   - ཨེཕ་ཨེསི་ཨི་ཐིག་ཁྲམ་ཚུ་ལག་ལེན་འཐབ་ཐོག་ལས་ ཌི་ཀོཌི་ཨེལ་ཨེལ་/ཨེམ་ཨེལ་/ཨོ་ཨེཕ་གནས་གོང་ཚུ།
   - བཤུད་སྒྲིལ་སྒོ་སྒྲིག་ལག་ལེན་འཐབ་ཐོག་ལས་ བསྐྱར་བཟོ་ཚུ་མཐུན་སྒྲིག་འབད།
5. ཐོན་འབྲས་དང་ ཅེག་སམ།
   - ཐོན་འབྲས་བཱ་ཕར་ནང་ལུ་ བསྐྱར་བཟོ་འབད་ཡོད་པའི་བཱའིཊི་ཚུ་བྲིས།
