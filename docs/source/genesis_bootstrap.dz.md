# བློ་གཏད་ཅན་གྱི་མཉམ་རོགས་ལས་ ཇེ་ནི་སིསི་བུཊ་སི་ཊར།

Iroha ས་གནས་ཀྱི་ `genesis.file` གིས་ བློ་གཏད་ཅན་གྱི་མཉམ་རོགས་ཀྱིས་ མཚན་རྟགས་བཀོད་ཡོད་པའི་ རིགས་མཚན་བཀག་ཆ་ཅིག་ ལེན་ཚུགས།
Norito-encoded བུཊི་སི་ཊརཔ་ མཐུན་གྲོས་ལག་ལེན་འཐབ་སྟེ།

- **Protocol:** མཉམ་རོགས་ཀྱིས་ `GenesisRequest` (`Preflight` མེ་ཊ་ཌེ་ཊ་, `Fetch` ལུ་སྤྲོད་དགོཔ་ཨིན།) དང་།
  `GenesisResponse` གཞི་ཁྲམ་ཚུ་ `request_id` གིས་ལྡེ་མིག་བརྐྱབ། ལན་འདེབས་འབད་མི་ཚུ་ཡང་ རིམ་སྒྲིག་ཨའི་ཌི་ མཚན་རྟགས་བཀོད་མི་ ཆང་ཁང་ཚུ་ཚུདཔ་ཨིན།
  hash, དང་གདམ་ཁ་ཅན་གྱི་ཚད་ཀྱི་བརྡ་སྟོན་ཅིག། པེ་ལོཌ་ཚུ་ `Fetch` ལུ་རྐྱངམ་ཅིག་སླར་ལོག་འབདཝ་ཨིནམ་དང་ འདྲ་བཤུས་ཞུ་བ་ ids ལུ་ཨིན།
  `DuplicateRequest` ཐོབ་ཡོད།
- **གཱརཌསི་:** ལན་འདེབས་འབད་མི་ཚུ་གིས་ གནང་བ་ཐོ་ཡིག་ (`genesis.bootstrap_allowlist` ཡང་ན་ བློ་གཏད་ཅན་གྱི་མཉམ་རོགས་ཚུ་ བསྟར་སྤྱོད་འབདཝ་ཨིན།
  set),-id/pubkey/has མཐུན་སྒྲིག་, ཚད་ཚད་ (`genesis.bootstrap_response_throttle`), དང་།
  ཚད་གཞི། (`genesis.bootstrap_max_bytes`). གནང་བ་ཐོ་ཡིག་ལས་ཕྱི་ཁར་ཞུ་བ་ཚུ་ལུ་ `NotAllowed` ཐོབ་ཨིན།
  ལྡེ་མིག་འཛོལ་བ་གིས་ མཚན་རྟགས་བཀོད་མི་ གླ་ཆ་ཚུ་ལུ་ `MismatchedPubkey` ཐོབ་ཨིན།
- **གསོག་འཇོག་འདི་སྟོངམ་སྦེ་ཡོད་པའི་སྐབས་ལུ་ བསྐྱར་གསོའི་རྒྱུན་འབབ་དང་ `genesis.file` འདི་ གཞི་སྒྲིག་མ་འབད་བས།
  `genesis.bootstrap_enabled=true`) མཐུད་མཚམས་དང་གཅིག་ཁར་ བློ་གཏད་ཅན་གྱི་མཉམ་རོགས་ གདམ་ཁ་ཅན་ཚུ་དང་གཅིག་ཁར་ མཛུབ་གནོན་འབད་བའི་སྔོན་བརྡ་ཚུ།
  `genesis.expected_hash`, དེ་ལས་ `validate_genesis_block` བརྒྱུད་དེ་ མིང་རྟགས་ཚུ་ བདེན་དཔྱད་འབདཝ་ཨིན།
  དང་ བཀག་ཆ་འདི་ མ་བཙུགས་པའི་ཧེ་མ་ ཀུ་ར་དང་གཅིག་ཁར་ `genesis.bootstrap.nrt` གིས་ གནས་ཏེ་ཡོདཔ་ཨིན། བུཊ་སི་ཊརཔ་ ཉམས་གསོ་འབད་ནི།
  གུས་བཏུད་ `genesis.bootstrap_request_timeout`, `genesis.bootstrap_retry_interval`, དང་།
  `genesis.file`.
- **འཐུས་ཤོར་གྱི་ཐབས་ལམ་ཚུ་:** ཞུ་བ་ཚུ་ གནང་བ་ཐོ་ཡིག་བརླག་སྟོར་ཞུགས་ནི་དང་ རྒྱུན་རིམ་/པབ་ཀི་/ཧ་ཤི་མ་མཐུནམ་ ཚད་གཞི།
  མགུ་ཏོག་ལས་འགལ་བའི་ཚད་གཞི་ཚད་གཞི་ ཉེ་གནས་ཀྱི་རིགས་མཚན་མེད་མི་ ཡང་ན་ ཞུ་བ་འདྲ་མིན་གྱི་ ids. ཧེ་ཤེ་ཚུ་ འཁྲུག་འཛིང་འབད་དོ།
  ཕར་ཚུར་མཉམ་རོགས་ཚུ་གིས་ ཕིཆ་བཀག་ཆ་འབད་ནི། ལན་འདེབས་འབད་མི་/དུས་ཚོད་ཚུ་ ཉེ་གནས་རིམ་སྒྲིག་ལུ་ལོག་མི་འགྱོ།
- **Operator step:** བློ་གཏད་ཅན་གྱི་མཉམ་རོགས་ གཅིག་འདི་ ནུས་ཅན་རིགས་མཚན་དང་གཅིག་ཁར་ ལྷོད་ཚུགསཔ་སྦེ་ངེས་གཏན་བཟོ། རིམ་སྒྲིག་འབད། རིམ་སྒྲིག་འབད།
  `bootstrap_allowlist`/`bootstrap_max_bytes`/`bootstrap_response_throttle` དང་ བསྐྱར་ལོག་འབད་རྩོལ་གྱི་མདུད་པ་དང་།
  གདམ་ཁ་ཅན་གྱི་པིན་ `expected_hash` གིས་ མཐུན་སྒྲིག་མེད་པའི་ པེ་ལོཌི་ཚུ་ ངོས་ལེན་འབད་ནི་ལས་ བཀག་ཐབས་ལུ། བརྩོན་ཤུགས་ཅན་གྱི་སྤྲོད་ལེན་ཚུ་འབད་ཚུགས།
  `genesis.file` ལུ་ `genesis.bootstrap.nrt` ལུ་སྟོན་ཐོག་ལས་ ཤུལ་མམ་གྱི་བུཊི་ཚུ་གུ་ལོག་སྟེ་ལག་ལེན་འཐབ་ཡོདཔ་ཨིན།
