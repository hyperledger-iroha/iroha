Этот файл создан `python3 scripts/fastpq/update_benchmark_history.py`.
Он соответствует требованиям FASTPQ Stage 7 WP5-B, отслеживая каждый завернутый графический процессор.

## Область применения и процесс обновления

- Создание или перенос новых захватов графического процессора (через `scripts/fastpq/wrap_benchmark.py`),
  добавьте их в матрицу захвата и перезапустите этот генератор, чтобы обновить
  столы.
- При наличии данных микробэнча Poseidon экспортируйте их с помощью
  `scripts/fastpq/export_poseidon_microbench.py` и перестройте манифест, используя
  `scripts/fastpq/aggregate_poseidon_microbench.py`.
- Записывайте пороговые значения Меркла, сохраняя выходные данные JSON в
  `benchmarks/merkle_threshold/`; этот генератор перечисляет известные файлы для проверки
  может перекрестно ссылаться на доступность процессора и графического процессора.
