## 1. လုပ်ဆောင်နေသည့် ဖော်ပြချက်

- **ရည်ရွယ်ချက်-** Rust node schema (`telemetry_redaction.md` ၏ §2) ကို ထင်ဟပ်နေချိန်တွင် AND7 ၏ မြင်နိုင်စွမ်း (latency၊ ထပ်စမ်းခြင်း၊ သက်သေပြခြင်း ကျန်းမာရေး) ကို ပံ့ပိုးရန်အတွက် လိုအပ်သော လုပ်ငန်းဆောင်ရွက်မှု တယ်လီမီတာကို ပံ့ပိုးပေးပါ။
- **စနစ်များ-** Android SDK ကိရိယာတန်ဆာပလာ -> OTLP တင်ပို့သူ -> SRE မှ စီမံခန့်ခွဲသော မျှဝေထားသော တယ်လီမီတာစုဆောင်းသူ (ပံ့ပိုးမှု Playbook §8 ကိုကြည့်ပါ)။
- **ဒေတာဘာသာရပ်များ-** Android SDK-based အက်ပ်များကို အသုံးပြုသည့် အော်ပရေတာဝန်ထမ်းများ၊ မြစ်အောက်ပိုင်း Torii အဆုံးမှတ်များ (တယ်လီမီတာမူဝါဒတစ်ခုလျှင် အာဏာပိုင်လိုင်းများကို ဖျက်ထားသည်)။

## 3. Lawful Basis & Rights

- **တရားဝင်အခြေခံ-** တရားဝင်အကျိုးစီးပွား (Art. 6(1)(စ)) — ထိန်းညှိထားသော စာရင်းဇယားဖောက်သည်များ၏ ယုံကြည်စိတ်ချရသော လည်ပတ်မှုကို အာမခံပါသည်။
- ** လိုအပ်သော စမ်းသပ်မှု-** လုပ်ငန်းလည်ပတ်မှု ကျန်းမာရေးအတွက် ကန့်သတ်ထားသော မက်ထရစ်များ (အသုံးပြုသူ အကြောင်းအရာ မရှိပါ); hashed authority သည် ခွင့်ပြုထားသော ပံ့ပိုးရေးဝန်ထမ်းများအတွက်သာ ရရှိနိုင်သော Rest node များနှင့် တန်းတူညီမျှမှုကို သေချာစေသည် (အလုပ်အသွားအလာကို လွှမ်းမိုးခြင်းမှတဆင့်)။
- ** ဟန်ချက်ညီခြင်းစမ်းသပ်ခြင်း-** တယ်လီမီတာကို အော်ပရေတာထိန်းချုပ်ထားသော စက်များတွင် အသုံးပြုသူဒေတာမဟုတ်ဘဲ အဆုံးစွန်သောအသုံးပြုသူဒေတာများထံတွင် ကန့်သတ်ထားသည်။ အစားထိုးမှုများသည် ပံ့ပိုးကူညီမှု + လိုက်နာမှု (Support Playbook §3 + §9) မှ ပြန်လည်သုံးသပ်ထားသော Norito လက်မှတ်ရေးထိုးထားသော ပစ္စည်းများ လိုအပ်သည်။
- **ဒေတာဘာသာရပ်ဆိုင်ရာအခွင့်အရေး-** အော်ပရေတာများသည် တယ်လီမီတာထုတ်ယူခြင်း/ဖျက်ခြင်းအား တောင်းဆိုရန် ပံ့ပိုးအင်ဂျင်နီယာ (ပလေးစာအုပ်§2) ကို ဆက်သွယ်ပါ။ ပြန်လည်ပြုပြင်ခြင်းအား ထပ်ကာထပ်ကာနှင့် မှတ်တမ်းများ (telemetry doc §Signal Inventory) သည် ရက် 30 အတွင်း ပြည့်စုံမှုကို ဖွင့်ပေးသည်။

## 5. လုပ်ဆောင်ချက်များနှင့် နောက်ဆက်တွဲများ

1. **သုံးလပတ်ပြန်လည်သုံးသပ်ခြင်း-** တယ်လီမီတာစနစ်များ၊ ဆားလည်ပတ်မှုများနှင့် မှတ်တမ်းများကို အစားထိုးအတည်ပြုပါ။ `docs/source/sdk/android/telemetry_redaction_minutes_YYYYMMDD.md` တွင် စာရွက်စာတမ်း
2. **Cross-SDK ချိန်ညှိမှု-** တစ်သမတ်တည်း ဟက်ခြင်း/ပုံးတင်ခြင်းဆိုင်ရာ စည်းမျဉ်းများကို ထိန်းသိမ်းရန် (လမ်းပြမြေပုံ AND7 တွင် ခြေရာခံသည်)။
3. **ပါတနာ ကွန်မန့်များ-** ပါတနာစတင်အသုံးပြုသည့်ကိရိယာများ (Support Playbook §9) တွင် DPIA အကျဉ်းချုပ်ကို ထည့်သွင်းပြီး `status.md` မှ ဤစာရွက်စာတမ်းသို့ လင့်ခ်ချိတ်ပါ။
