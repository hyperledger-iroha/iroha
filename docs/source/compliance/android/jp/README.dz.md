# ཇཱ་པཱན་གྱི་ བསྟུན་བཅོས་རྣམ་གཞག་ཚུ།

ཨེཕ་ཨའི་ཨེསི་སི་ཚད་འཛིན་ཞིབ་དཔྱད་ཐོ་ཡིག་ཚུ་གི་དོན་ལུ་ སྣོད་ཐོ་འདི་ལག་ལེན་འཐབ།
ཇ་པཱན་གྱི་མཉམ་འབྲེལ་པ་ཚུ་ལུ་ དམིགས་བསལ་གྱི་སྒྲུབ་བྱེད་ག་ཅི་རང་ཡོད་རུང་།
བསྒྱུར་བཅོས་འབད་མ་ཚུགས་པའི་འབྱུང་ཁུངས་ཡིག་ཆ་ལུ་འབྲེལ་ལམ་དང་ བསྐྱར་ཞིབ་འབད་མི་/མིང་རྟགས་-ཨོཕ་མེཊ་ཌེ་ཊ་ཚུ་ བཙུགས།
