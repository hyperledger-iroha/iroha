# Жапонияның сәйкестік артефактілері

Бұл каталогты FISC бақылау тізімдері, екі тілді StrongBox аттестаттау үшін пайдаланыңыз
пакеттер және жапон серіктестері үшін талап етілетін кез келген реттеушіге тән дәлелдер.
Өзгермейтін бастапқы құжатқа сілтеме және шолушы/шығу метадеректерін қосыңыз.
