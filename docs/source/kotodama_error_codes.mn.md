# Kotodama Хөрвүүлэгчийн алдааны кодууд

Kotodama хөрвүүлэгч нь тогтвортой алдааны кодуудыг гаргадаг бөгөөд ингэснээр багаж хэрэгсэл болон CLI хэрэглэгчид
бүтэлгүйтлийн шалтгааныг хурдан ойлгох. `koto_compile --explain <code>` ашиглана уу
харгалзах зөвлөмжийг хэвлэх.

| Код | Тодорхойлолт | Ердийн засвар |
|-------|-------------|-------------|
| `E0001` | Салбарын зорилт нь IVM үсрэлтийн кодчилолын хүрээнээс гадуур байна. | Маш том функцуудыг хуваах эсвэл доторлогоог багасгахын тулд блокийн үндсэн зай ±1МБ дотор байх болно. |
| `E0002` | Дуудлагын сайтууд хэзээ ч тодорхойлогдоогүй функцийг иш татдаг. | Дуудлага хүлээн авагчийг устгасан үсгийн алдаа, харагдац тохируулагч эсвэл онцлог шинж тэмдгийг шалгана уу. |
| `E0003` | ABI v1-ийг идэвхжүүлэхгүйгээр удаан эдэлгээтэй төлөвийн системийн дуудлагууд гарсан. | `CompilerOptions::abi_version = 1`-г тохируулах эсвэл `seiyaku` гэрээний дотор `meta { abi_version: 1 }`-г нэмнэ үү. |
| `E0004` | Өмчтэй холбоотой системүүд нь шууд бус заагч хүлээн авсан. | `account_id(...)`, `asset_definition(...)` гэх мэтийг ашиглах эсвэл хостын өгөгдмөл байдалд 0 харуул дамжуулна. |
| `E0005` | `for` давталт эхлүүлэгч нь өнөөдрийн дэмжигдсэнээс илүү төвөгтэй юм. | Нарийн төвөгтэй тохиргоог давталтын өмнө шилжүүлэх; Одоогоор зөвхөн энгийн `let`/илэрхийлэл эхлүүлэгчийг хүлээн авч байна. |
| `E0006` | `for` давталтын алхам нь өнөөдрийн дэмжигдсэнээс илүү төвөгтэй юм. | Давталтын тоолуурыг энгийн илэрхийллээр шинэчилнэ үү (жишээ нь, `i = i + 1`). |
