# Iroha v2 གནས་སྡུད་དཔེ་ཚད་ – གཏིང་ཟབ་པའི་མཆོང་ལྡིང་།

ཡིག་ཆ་འདི་གིས་ `iroha_data_model` ཀེརེཊི་ནང་ལག་ལེན་འཐབ་ཡོདཔ་དང་ ལཱ་གི་ས་སྒོ་ནང་ལུ་ལག་ལེན་འཐབ་མི་ Iroha v2 གནད་སྡུད་དཔེ་ཚད་བཟོ་མི་ གཞི་བཀོད་དང་ངོས་འཛིན་འབད་མི་ རང་གཤིས་ དེ་ལས་ མཐུན་སྒྲིག་ཚུ་ འགྲེལ་བཤད་རྐྱབ་ཨིན། ཁྱོད་ཀྱིས་བསྐྱར་ཞིབ་འབད་དེ་ དུས་མཐུན་བཟོ་ནི་གི་གྲོས་འཆར་བཀོད་ཚུགས་པའི་ གཞི་བསྟུན་གཏན་གཏན་ཅིག་སྦེ་ གོ་དགོཔ་ཨིན།

## ཁྱབ་ཁོངས་དང་རྨང་གཞི།

- དམིགས་ཡུལ་: མངའ་ཁོངས་དངོས་པོ་ཚུ་གི་དོན་ལུ་ ཀེ་ནོ་ནིཀ་དབྱེ་བ་ཚུ་བྱིནམ་ཨིན།
- རིམ་སྒྲིག་འབད་ནི་: མི་མང་དབྱེ་བ་ཆ་མཉམ་གྱིས་ Norito ཀོ་ཌེགསི་ (`norito::codec::{Encode, Decode}`) དང་ ལས་རིམ་ (`iroha_schema::IntoSchema`) ཚུ་འབྱུངམ་ཨིན། JSON འདི་ ཁྱད་རྣམ་གྱི་དར་ཆ་ཚུ་གི་རྒྱབ་ཁར་ སེལ་འཐུ་འབད་དེ་ལག་ལེན་འཐབ་ཨིན་ (དཔེར་ན་ HTTP དང་ `Json` payloads གི་དོན་ལུ་)
- IVM དྲན་འཛིན་: ཧོསིཊི་གིས་ གན་རྒྱ་ཚུ་ འབོད་བརྡ་མ་འབད་བའི་ཧེ་མ་ བདེན་དཔྱད་འབདཝ་ལས་ རིམ་སྒྲིག་མེད་པའི་དུས་ཚོད་བདེན་དཔྱད་ལ་ལུ་ཅིག་ ལྕོགས་མིན་བཟོཝ་ཨིན།
- FFI gates: དབྱེ་བ་ལ་ལུ་ཅིག་ FFI གི་དོན་ལུ་ `iroha_ffi` བརྒྱུད་དེ་ `ffi_export`/`ffi_import` གི་རྒྱབ་ཁར་ FFI དགོཔ་མེད་པའི་སྐབས་ མགུ་ཏོ་གུ་བཀལ་ནི་ལས་ བཀག་ཐབས་ལུ་ གནས་སྟངས་དང་འཁྲིལ་ཏེ་ མཆན་བཀོད་འབད་ཡོདཔ་ཨིན།

## གཞི་རྩའི་ཁྱད་ཆོས་དང་རོགས་རམ།- `Identifiable`: ངོ་བོ་ཚུ་ལུ་ བརྟན་ཏོག་ཏོ་ `Id` དང་ `fn id(&self) -> &Self::Id` ཡོདཔ་ཨིན། སབ་ཁྲ་/ཆ་ཚན་མཐུན་འབྲེལ་གྱི་དོན་ལུ་ `IdEqOrdHash` དང་ཅིག་ཁར་ བཏོན་དགོ།
- `Registrable`/`Registered`: ངོ་བོ་མང་ཤོས་ཅིག་གིས་ (དཔེར་ན་ `Domain`, `AssetDefinition`, `Role`, `Role`) བཟོ་བསྐྲུན་པ་དཔེ་རིས་ཅིག་ལག་ལེན་འཐབ་ཨིན། `Registered` གིས་ ཐོ་བཀོད་ཚོང་འབྲེལ་གྱི་དོན་ལུ་འོས་འབབ་ཡོད་པའི་ ལྗིད་ཚད་མར་ཕབ་བཟོ་བསྐྲུན་པ་དབྱེ་བ་ (`With`) ལུ་ རན་ཊའིམ་དབྱེ་བ་འདི་མཐུདཔ་ཨིན།
- `HasMetadata`: ལྡེ་མིག་/གནས་གོང་ `Metadata` སབ་ཁྲ་ཅིག་ལུ་ མཉམ་བསྡོམས་འཛུལ་སྤྱོད་འབད།
- `IntoKeyValue`: འདྲ་བཤུས་མར་ཕབ་འབད་ནི་ལུ་ `Key` (ID) དང་ `Value` (གནད་སྡུད་) སོ་སོ་སྦེ་གསོག་འཇོག་འབད་ནི་ལུ་ གསོག་འཇོག་བགོ་བཤའ་རྐྱབ་མི་གྲོགས་རམ་འབདཝ་ཨིན།
- `Owned<T>`/`Ref<'world, K, V>`: དགོས་མཁོ་མེད་པའི་འདྲ་བཤུས་ཚུ་ བཀག་ཐབས་ལུ་ གསོག་འཇོག་དང་ འདྲི་དཔྱད་ཚགས་མ་ཚུ་ནང་ ལག་ལེན་འཐབ་མི་ ལྗིད་ཚད་མར་ཕབ་འབད་མི་ཚུ་ཨིན།

## མིང་དང་ངོས་འཛིན།- `Name`: ནུས་ཅན་ཚིག་ཡིག་ངོས་འཛིན་པ། བར་སྟོང་དཀརཔོ་དང་ ཟུར་བཞག་ཡིག་འབྲུ་ཚུ་ `@`, `#`, `$` (ཀམ་པོ་སིཊི་ཨའི་ཌི་ཚུ་ནང་ལག་ལེན་འཐབ་ཡོདཔ་) འབད་མི་ཆོག། བདེན་དཔྱད་དང་གཅིག་ཁར་ `FromStr` བརྒྱུད་དེ་བཟོ་བསྐྲུན་འབད་ཚུགས། མིང་ཚུ་ ཡུ་ནི་ཀོཌི་ཨེན་ཨེཕ་སི་ལུ་ སྤྱིར་བཏང་བཟོཝ་ཨིན་ ༼ཁྲིམས་ལུགས་འདྲ་མཉམ་གྱི་ཡིག་སྡེབ་ཚུ་ ཅོག་འཐདཔ་སྦེ་ བརྩི་འཇོག་འབད་དེ་ གསོག་འཇོག་འབད་དེ་ཡོདཔ་ཨིན།༽ དམིགས་བསལ་གྱི་མིང་ `genesis` འདི་ བཀག་བཞག་ཡོདཔ་ཨིན།
- `IdBox`: རྒྱབ་སྐྱོར་འབད་ཡོད་པའི་ཨའི་ཌི་གང་རུང་ཅིག་གི་དོན་ལུ་ བསྡོམས་རྩིས་དབྱེ་བ་ཡིག་ཤུབས་ (`DomainId`, `AccountId`, `AssetDefinitionId`, `AssetId`, Norito, Norito `TriggerId`, `RoleId`, `Permission`, `CustomParameterId`). སྤྱིར་བཏང་རྒྱུན་འབབ་དང་ Norito ཨིན་ཀོ་ཌིང་གི་དོན་ལུ་ དབྱེ་བ་རྐྱང་པ་སྦེ་ཕན་ཐོགས་ཅན་ཨིན།
- `ChainId`: ཚོང་འབྲེལ་ཚུ་ནང་ བསྐྱར་རྩེད་སྲུང་སྐྱོབ་ཀྱི་དོན་ལུ་ ལག་ལེན་འཐབ་མི་ མཐོང་མ་ཚུགས་པའི་ རིམ་སྒྲིག་ངོས་འཛིན་པ།ཨའི་ཌི་ཚུ་གི་ཡིག་རྒྱུན་འབྲི་ཤོག་ཚུ་ (`Display`/`FromStr` དང་ཅིག་ཁར་ སྐོར་ཐེངས་འགྲུལ་བསྐྱོད་འབད་བཏུབ་):
- `DomainId`: `name` (དཔེར་ན་ `wonderland`)།
- `AccountId`: ཀེ་ནོ་ནིཀ་ཌོ་མེན་མེད་པའི་རྩིས་ཐོ་ངོས་འཛིན་འབད་མི་འདི་ `AccountAddress` བརྒྱུད་དེ་ I105 རྐྱངམ་ཅིག་སྦེ་ ཨིན་ཀོ་ཌི་འབད་ཡོདཔ་ཨིན། དབྱེ་དཔྱད་འབད་མི་ཨིན་པུཊི་ཚུ་ ཀེ་ནོ་ནིཀ་ I105; མངའ་ཁོངས་རྗེས་འཇུག་ཚུ་ (`@domain`) དང་ ཀེ་ནོ་ནིཀ་ I105 ཡིག་འབྲུ་ཚུ་ མིང་གཞན་ཚུ་ ཀེ་ནོ་ནིཀ་ཧེགསི་པར་སར་ཨིན་པུཊི་ རིག་གཞུང་ `norito:` པེ་ལོཌི་ཚུ་ དང་ `uaid:`/`uaid:`/`uaid:`/`opaque:` ཚུ་ཨིན།
- `AssetDefinitionId`: ཀེ་ནོ་ནིཀ་`aid:<32-lower-hex-no-dash>` (UUID-v4 བཱའིཊིསི)།
- `AssetId`: ཀེ་ནོ་ནིཀ་ཨེན་ཀོ་ཌི་འབད་ཡོད་པའི་ཚིག་ཡིག་ `norito:<hex>` (རིག་གཞུང་ཚིག་ཡིག་འབྲི་ཤོག་ཚུ་ གསར་བཏོན་འགོ་དང་པ་ནང་ རྒྱབ་སྐྱོར་མེདཔ་ཨིན།)།
- `NftId`: `nft$domain` (དཔེར་ན་ `rose$garden`)།
- `PeerId`: `public_key` (མཉམ་རོགས་འདྲ་མཉམ་འདི་ མི་མང་ལྡེ་མིག་གིས་ཨིན།)།

## ངོ་བོ་ཚུ།

### མངའ་ཁོངས།
- `DomainId { name: Name }` – གཞན་དང་མ་འདྲ་བའི་མིང་།
- `Domain { id, logo: Option<SorafsUri>, metadata: Metadata, owned_by: AccountId }`.
- བཟོ་བསྐྲུན་པ་: `NewDomain` དང་ `with_logo`, `with_metadata`, དེ་ལས་ `Registrable::build(authority)` གིས་ `owned_by` གཞི་སྒྲིག་འབདཝ་ཨིན།### རྩིས་ཐོ།
- `AccountId` འདི་ ཚད་འཛིན་པ་གིས་ ལྡེ་མིག་བཙུགས་ཏེ་ ཀེ་ནོ་ནིཀ་ མངའ་ཁོངས་མེད་པའི་རྩིས་ཐོའི་ངོ་རྟགས་དང་ ཀེ་ནོ་ནིཀ་ I105 སྦེ་ ཨིན་ཀོ་ཌི་འབད་ཡོདཔ་ཨིན།
- `ScopedAccountId { account: AccountId, domain: DomainId }` གིས་ ཁྱབ་ཚད་མཐོང་སྣང་དགོ་ས་ལུ་རྐྱངམ་ཅིག་ གསལ་ཏོག་ཏོ་མངའ་ཁོངས་སྐབས་དོན་འབག་འོང་།
- `Account { id, metadata, label?, uaid? }` — `label` འདི་ རི་ཀི་དྲན་ཐོ་ཚུ་གིས་ལག་ལེན་འཐབ་མི་ གདམ་ཁ་ཅན་གྱི་གཏན་འཇགས་མིང་གཞན་ཅིག་ཨིན། `uaid` གིས་ གདམ་ཁ་ཅན་གྱི་ Nexus-wide [ཡོངས་ཁྱབ་རྩིས་ཐོ་ཨའི་ཌི་](Kotodama) འབག་འོང་།

- འཕྲུལ་རིག་: `iroha_data_model_derive::model` མེཀ་རོ་གིས་ `#[cfg(feature = "transparent_api")] pub` དང་ཅིག་ཁར་ མི་མང་ས་སྒོ་རེ་རེ་བཞིན་དུ་ ལོག་བྲིས་ཏེ་ སྔོན་སྒྲིག་བཟོ་བསྐྲུན་གྱི་དོན་ལུ་ སྒེར་གྱི་འདྲ་བཤུས་ཅིག་བཞགཔ་ཨིན། ཁྱད་རྣམ་འདི་ལྕོགས་ཅན་བཟོ་མི་འདི་གིས་ cfgs དེ་ཚུ་ ཕིལཔ་འབདཝ་ལས་ `Account`, `Domain`, `Asset`, ལ་སོགས་པ་ཚུ་ མེདཔ་བཏང་མི་འདི་ ཁོང་རའི་ངེས་ཚིག་ཚད་གཞི་ཚུ་གི་ཕྱི་ཁར་ ཁྲིམས་མཐུན་འགྱོཝ་ཨིན།
- ཁ་ཐོག་བརྟག་དཔྱད་: ཀེརེཊི་གིས་ `TRANSPARENT_API: bool` རྟག་བརྟན་ཅིག་ཕྱིར་འདྲེན་འབདཝ་ཨིན། མར་འབབ་ཀྱི་ཨང་རྟགས་འདི་གིས་ དར་ཆ་འདི་དང་ཡན་ལག་འདི་ དྭངས་གསལ་མེད་པའི་གྲོགས་རམ་འབད་མི་ཚུ་ལུ་ལོག་འོང་དགོཔ་ད་ བརྟག་ཞིབ་འབད་ཚུགས།
- ལྕོགས་ཅན་བཟོ་ནི་: `features = ["transparent_api"]` འདི་ `Cargo.toml` ནང་ལུ་བརྟེན་མི་ལུ་ཁ་སྐོང་བརྐྱབ། JSON པར་བརྙན་དགོ་པའི་ ལཱ་གི་ས་སྒོའི་སྒམ་ཚུ་གིས་ (དཔེར་ན་ `iroha_torii`) རང་བཞིན་གྱིས་ དར་འདི་གདོང་ཁར་གཏངམ་ཨིན་རུང་ ཕྱོགས་གསུམ་པའི་ཉོ་སྤྱོད་པ་ཚུ་གིས་ བཀྲམ་སྤེལ་ཚད་འཛིན་འབད་དེ་ API ཁ་ཐོག་རྒྱ་ཆེཝ་འདི་ངོས་ལེན་མ་འབད་ཚུན་ཚོད་ འདི་བཀག་བཞག་དགོ།
