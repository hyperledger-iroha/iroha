## Решение

- **Утвердить развертывание полезной нагрузки v1** для SDK и CLI, что позволит кошелькам Swift создавать конфиденциальные конверты без специальной настройки.
- **Условия:** 
  - Держите приборы четности под оповещениями о дрейфе CI (привязанными к `scripts/check_norito_bindings_sync.py`).
  - Задокументируйте операционную схему в `docs/source/confidential_assets.md` (уже обновлено через Swift SDK PR).
  - Запишите данные калибровки и телеметрии перед тем, как устанавливать какие-либо производственные флаги (отслеживаются под M2).
