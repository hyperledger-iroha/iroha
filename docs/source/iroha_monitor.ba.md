# Iroha Монитор

Рефакторлы Iroha мониторы еңел терминаль UI менән йәнләндерелгән парлы.
фестивалендә ASCII сәнғәте һәм традицион Этенраку темаһы.  Ул ике йүнәлеш бирә .
ябай эш ағымы:

- **Инжен-лит режимы** – тиҫтерҙәрен имитациялаусы эфемер статусы/метрик стабтарҙы башлай.
- **Беркетергә ** – мониторға ғәмәлдәге Torii HTTP ос нөктәләрендә күрһәтегеҙ.

UI һәр яңыртыуҙа өс төбәкте күрһәтә:

1. **Torii skyline башы** – анимацион тори ҡапҡаһы, Mt. Fuji, кой тулҡындары, һәм йондоҙ
   ялан, тип прокручивать синхронизация менән яңыртыу каденцияһы.
2. **Йыйынтыҡ таҫма ** – агрегацияланған блоктар/транзакциялар/газ плюс яңыртыу ваҡыты.
3. **Тиңер өҫтәле & фестивал шыбырлай** – һул яҡтағы тиҫтерҙәр рәттәре, әйләнешле ваҡиға
   уңға логин, тип иҫкәртмәләрҙе тота (тайм-аут, ҙур файҙалы йөктәр һ.б.).
4. **Газь тенденцияһы ** – `--show-gas-trend` ҡоролмаһы осҡон һыҙығын ҡушыу өсөн
   дөйөм газ ҡулланыу бөтә тиҫтерҙәре буйынса дөйөмләштереү.

Был рефакторҙа яңы:

- Япон стилендә ASCII сәхнәһе менән кой, тори һәм фонарь.
- Ябайлаштырылған команда өҫтө (`--spawn-lite`, `--attach`, `--interval`).
- Интро баннер менән өҫтәмә аудио уйнау гагаку темаһы (тышҡы MIDI
  уйынсы йәки платформа/аудио стека ярҙам иткәндә йомшаҡ синтезатор төҙөлгән).
- `--no-theme` / `--no-audio` флагтары өсөн CI йәки тиҙ төтөн йүгерә.
- Һуңғы иҫкәртмә, ваҡыт йәки эш ваҡыты ҡылған “кәйеф” бағанаһына.

## Quickstart

Монитор төҙөү һәм уны ҡаршы йөрөтөү өсөн тиҫтерҙәре ҡатыштырып:

```bash
cargo run -p iroha_monitor -- --spawn-lite --peers 3
```

Ғәмәлдәге Torii ос нөктәләренә беркетелгән:

```bash
cargo run -p iroha_monitor -- \
  --attach http://127.0.0.1:8080 http://127.0.0.1:8081 \
  --interval 500
```

CI-дуҫ саҡырыу (сиквать интро анимация һәм аудио):

```bash
cargo run -p iroha_monitor -- --spawn-lite --no-theme --no-audio
```

### CLI флагтары

```
--spawn-lite         start local status/metrics stubs (default if no --attach)
--attach <URL...>    attach to existing Torii endpoints
--interval <ms>      refresh interval (default 800ms)
--peers <N>          stub count when spawn-lite is active (default 4)
--no-theme           skip the animated intro splash
--no-audio           mute theme playback (still prints the intro frames)
--midi-player <cmd>  external MIDI player for the built-in Etenraku .mid
--midi-file <path>   custom MIDI file for --midi-player
--show-gas-trend     render the aggregate gas sparkline panel
--art-speed <1-8>    multiply the animation step rate (1 = default)
--art-theme <name>   choose between night, dawn, or sakura palettes
--headless-max-frames <N>
                     cap headless fallback to N frames (0 = unlimited)
``` X

## Тема интро

Ғәҙәттә, стартап ҡыҫҡа ASCII анимация уйнай, ә Etenraku балл
башлана.  Аудио һайлау тәртибе:

1. Әгәр `--midi-player` бирелгән, генерациялау демо MIDI (йәки ҡулланыу `--midi-file`)
   һәм бойороҡто үрсетергә.
2. Юғиһә, macOS/Windows (йәки Linux менән `--features iroha_monitor/linux-builtin-synth`)
   гагаку йомшаҡ синтезаторы менән баллды күрһәтеү (тышҡы аудио юҡ
   активтар кәрәк).
3. Әгәр аудио өҙөлгән йәки инициализация етешһеҙлектәр, интро һаман да баҫтырып сығара
   анимация һәм шунда уҡ TUI инә.

CPAL-ҡуйылған синтезатор авто-макОС һәм Windows өҫтөндә. Linux-та ул
опт-ин ҡотолоу өсөн юғалған ALSA/Int башлыҡтары эш урыны ваҡытында төҙөү; уны индереү
`--features iroha_monitor/linux-builtin-synth` менән, әгәр һеҙҙең система тәьмин итә
эшләү аудио стека.

Ҡулланыу `--no-theme` йәки `--no-audio` CI йәки башһыҙ снарядтарҙа эшләгәндә.

Йомшаҡ синтезатор хәҙер *MIDI синтезатор дизайнында тотолған ойоштороуҙан һуң 2010 йылда .
Rust.pdf*: хичирики һәм рютеки гетерофоник көй менән бүлешә, ә шо
документта һүрәтләнгән аайтак прокладкаларын тәьмин итә.  Ваҡытлы иҫкәрмә мәғлүмәттәре йәшәй
`etenraku.rs`-та; ул CPAL шылтыратыу һәм генерацияланған демо MIDI ҡөҙрәте лә.
Ҡасан аудио сығышы юҡ, монитор уйнауҙы үткәрә, әммә һаман да рендер
ASCII анимацияһы.

## UI дөйөм мәғлүмәт- **Баш сәнғәте** – һәр кадр `AsciiAnimator` менән генерацияланған; кой, тории фонарь,
  һәм тулҡындар өҙлөкһөҙ хәрәкәт итеү өсөн дрейфлай.
- **Йыйынтыҡ таҫма** – онлайн тиҫтерҙәре күрһәтә, тиҫтерҙәр һаны тураһында хәбәр, блок дөйөм,
  буш булмаған блок дөйөм, tx раҫлау/кире кире ҡағыу, газ ҡулланыу, һәм яңыртыу ставкаһы.
- **Тиңер таблицаһы** – псевдоним/ос нөктәһе өсөн бағана, блоктар, транзакциялар, сират күләме,
  газ ҡулланыу, латентлыҡ, һәм “кәйеф” кәңәше (иҫкәртмәләр, ваҡыт үтәй, эш ваҡыты).
- **Фестиваль шыбырлаусылар** – иҫкәртмәләрҙең логирование журналы (тоташтырыу хаталары, файҙалы йөк
  сикләүҙәрҙе сикләү, яй ос нөктәләре).  Хәбәрҙәр кире ҡайтарыла (өҫтөндә һуңғы).

Клавиатура ярлыҡтары:

- `n` / Уң / Аҫҡа – фокус киләһе тиңдәшкә күсергә.
- `p` / Һул / Өҫкә – фокусты элекке тиҫтерҙәренә күсерергә.
- `q` / Esc / Ctrl-C – сығыу һәм терминалды тергеҙеү.

Монитор альтернатив-экран буферы менән кросстер + рататуй ҡуллана; сығыуҙа уны
курсорҙы тергеҙә һәм экранды таҙарта.

## Төтөн һынауҙары

Йәшник суднолар интеграция һынауҙары, улар режимдарҙы ла, HTTP сиктәрен дә ҡуллана:

- `spawn_lite_smoke_renders_frames`

- `invalid_endpoint_surfaces_warning`
- `status_limit_warning_is_rendered`

Монитор һынауҙарын ғына эшләгеҙ:

```bash

монитор һынауҙары айырым әле лә тиҙ раҫлау өсөн файҙалы, ҡасан һеҙ эшләйһегеҙ
тулы люкс кәрәкмәй.
