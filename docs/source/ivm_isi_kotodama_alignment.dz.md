# IVM ⇄ ISI ⇄ གནས་སྡུད་དཔེ་སྟོན། ⇄ Kotodama — ཕྲང་སྒྲིག་བསྐྱར་ཞིབ།

ཡིག་ཆ་རྩིས་ཞིབ་འདི་གིས་ Iroha བར་ཅུ་ཡལ་འཕྲུལ་ཆས་ (IVM) བཀོད་རྒྱ་ཆ་ཚན་དང་ syscall ཁ་ཐོག་གི་སབ་ཁྲ་འདི་ Iroha དམིགས་བསལ་བཀོད་རྒྱ་ (ISI) དང་ Kotodama དང་ Kotodama དེ་ལས་ Kotodama ལུ་ ག་དེ་སྦེ་ཨིན་ན། འདི་གིས་ ད་རེས་ཀྱི་ བར་སྟོང་ཚུ་ངོས་འཛིན་འབད་དེ་ བང་རིམ་བཞི་འདི་ གཅིག་ཁར་བསྡོམས་ཏེ་ གཏན་འབེབས་བཟོ་ནི་དང་ ལཱ་འབད་ནི་གི་ཐད་ཁར་ གཅིག་ཁར་མཐུན་སྒྲིག་འབདཝ་ཨིན།

བཱའིཊི་ཀོཌི་དམིགས་གཏད་སྐོར་ལས་ དྲན་འཛིན་: Kotodama smar གན་རྒྱ་ཚུ་ Iroha ཝིར་ཊུ་ཨཱལ་འཕྲུལ་ཆས་ (IVM) བཱའིཊི་ཀོཌི་ (Kotodama) ལུ་ བསྡུ་སྒྲིག་འབད་ཡོདཔ་ཨིན། དེ་ཚུ་གིས་ “risc5”/RISC‐V འདི་ རང་དབང་གི་བཟོ་བཀོད་ཅིག་སྦེ་ དམིགས་གཏད་བསྐྱེད་མི་བཏུབ། འདི་ཁར་ RISC-V བཟུམ་གྱི་ ཨིན་ཀོ་ཌིང་ཚུ་ IVM གི་ སླ་བསྲེ་བཀོད་རྒྱ་གི་རྩ་སྒྲིག་གི་ཆ་ཤས་ཅིག་ཨིནམ་ལས་ ལག་ལེན་འཐབ་ཐངས་ཀྱི་ཁ་གསལ་སྦེ་རང་ ལུས་ཡོདཔ་ཨིན།

## ཉེན་ཁ་དང་ འགན་འཛིན་འཆར་གཞི།
- ཕྲང་སྒྲིག་ཆ་ཚན་ཅིག་ (རྒྱུ་དངོས་ + རྩིས་ཐོ་ཚུ) གི་དོན་ལུ་ གློག་ཐག་ཧོསིཊི་གིས་འགོ་བཙུགས་ཏེ་ ཆེད་དམིགས་བརྟག་དཔྱད་ཚུ་ཁ་སྐོང་འབད།
- མི་ཁུངས་ངོ་མ་ཨའི་ཨེསི་ཨའི་ ལག་ལེན་འཐབ་ནི་འདི་ དབང་ཚད་ཅན་གྱི་འགྲུལ་ལམ་སྦེ་ བདག་འཛིན་འཐབ་སྟེ་ ཡིག་བརྡའི་རིག་པ་འདི་ རྒས་ཤོས་སྦེ་བཞག་དགོ། འདྲ་མཚུངས་ཀྱི་མཐའ་མཇུག་གི་ནུས་པ་དང་བྱུང་ལས་ཚུ་བདེན་བཤད་འབད་ནི་ལུ་ བརྟག་དཔྱད་ཚུ་ནང་ “གྱིབ་མ་ཐབས་ལམ་” གི་འོག་ལུ་འགྲུལ་ལམ་གཉིས་ཆ་ར་གཡོག་བཀོལ།
- འདྲ་མཉམ་འདི་བདེན་དཔྱད་འབད་ཚར་བའི་ཤུལ་ལས་ ཐོན་སྐྱེད་ནང་ལུ་ IVM ཧོསིཊི་འདི་ ལྕོགས་ཅན་བཟོ། ཤུལ་ལས་ དུས་རྒྱུན་གྱི་ཚོང་འབྲེལ་ཚུ་ IVM བརྒྱུད་དེ་ བསམ་ཞིབ་འབད་དགོ།
