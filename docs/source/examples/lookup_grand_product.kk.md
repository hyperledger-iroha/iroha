# Үлкен өнім үлгісін іздеңіз

Бұл мысалда аталған FASTPQ рұқсатын іздеу аргументі кеңейтілген
`fastpq_plan.md`.  2-кезең құбырында провер селекторды бағалайды
Төмен дәрежелі кеңейтімдегі (LDE) (`s_perm`) және куә (`perm_hash`) бағандары
домен, `Z_i` жұмыс істеп тұрған үлкен өнімді жаңартады және соңында бүкіл
Посейдонмен бірізділік.  Хештелген аккумулятор транскриптке қосылады
`fastpq:v1:lookup:product` доменінде, соңғы `Z_i` әлі де сәйкес келеді
бекітілген рұқсат кестесінің өнімі `T`.

Біз келесі селектор мәндері бар шағын партияны қарастырамыз:

| қатар | `s_perm` | `perm_hash` |
| --- | -------- | ------------------------------------------------------- |
| 0 | 1 | `0x019a...` (грант рөлі = аудитор, рұқсат = трансфер_актив) |
| 1 | 0 | `0xabcd...` (рұқсат өзгеріссіз) |
| 2 | 1 | `0x42ff...` (қайтарып алу рөлі = аудитор, рұқсат = burn_asset) |

`gamma = 0xdead...` келесіден алынған Fiat-Shamir іздеу тапсырмасы болсын.
транскрипт.  Провер `Z_0 = 1` инициализациясын жасайды және әр жолды бүктейді:

```
Z_0 = 1
Z_1 = Z_0 * (perm_hash_0 + gamma)^(s_perm_0) = 1 * (0x019a... + gamma)
Z_2 = Z_1 * (perm_hash_1 + gamma)^(s_perm_1) = Z_1 (selector is zero)
Z_3 = Z_2 * (perm_hash_2 + gamma)^(s_perm_2)
```

`s_perm = 0` аккумуляторды өзгертпейтін жолдар.  өңдеуден кейін
Trace, провер Посейдон-транскрипт үшін `[Z_1, Z_2, ...]` тізбегін хэштейді.
сонымен қатар кестеге сәйкестендіру үшін `Z_final = Z_3` (соңғы жұмыс істейтін өнім) шығарады
шекаралық шарт.

Кесте жағында бекітілген рұқсат Merkle тармағы детерминистикалық мәнді кодтайды
ұяға арналған белсенді рұқсаттар жинағы.  Тексеруші (немесе провер
куәгер генерациясы) есептейді

```
T = product over entries: (entry.hash + gamma)
```

Протокол `Z_final / T = 1` шекаралық шектеуін жүзеге асырады.  Егер із
кестеде жоқ рұқсатты енгізді (немесе оны өткізіп жіберді
болса), негізгі өнім қатынасы 1-ден ауытқиды және тексеруші қабылдамайды.  Өйткені
екі жағы Goldilocks өрісінің ішінде `(value + gamma)` көбейтіледі, қатынас
CPU/GPU серверлерінде тұрақты болып қалады.

Арматуралар үшін мысалды Norito JSON ретінде сериялау үшін, кортежді жазыңыз
`perm_hash`, селектор және әр жолдан кейін аккумулятор, мысалы:

```json
{
  "gamma": "0xdead...",
  "rows": [
    {"s_perm": 1, "perm_hash": "0x019a...", "z_after": "0x5f10..."},
    {"s_perm": 0, "perm_hash": "0xabcd...", "z_after": "0x5f10..."},
    {"s_perm": 1, "perm_hash": "0x42ff...", "z_after": "0x9a77..."}
  ],
  "table_product": "0x9a77..."
}
```

Оналтылық толтырғыштарды (`0x...`) бетон Алтын құлыптармен ауыстыруға болады
автоматтандырылған сынақтарды жасау кезінде өріс элементтері.  Қосымша 2-кезең арматурасы
жұмыс істеп тұрған аккумулятордың Poseidon хэшін жазыңыз, бірақ бірдей JSON пішінін сақтаңыз,
сондықтан мысал болашақ сынақ векторлары үшін үлгі ретінде екі еселене алады.
