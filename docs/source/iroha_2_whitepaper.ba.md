## 3. Мәғлүмәттәр моделе

Metadata is deterministic (`Metadata` map). Numeric assets support fixed-point

- **Төҙөүсе башҡарыусы** Iroha махсус күрһәтмәләрен (ISI) актив, иҫәп, рөхсәт, рөхсәт итеү өсөн ярҙам итеүен дауам итә.
