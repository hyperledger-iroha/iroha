# གྲོས་གཞི་ཚོགས་འདུའི་གྲོས་འཆར་ལས་རིམ།(MINFO-2a)

ལམ་སྟོན་གཞི་བསྟུན་: **MINFO-2a — གྲོས་འཆར་རྩ་སྒྲིག་བདེན་དཔྱད་འབད་མི་.**

གྲོས་གཞི་ལྷན་ཚོགས་ཀྱི་ལཱ་གིས་ མི་སེར་གྱིས་ ཐོ་ཡིག་ནགཔོ་དང་ སྲིད་བྱུས་བསྒྱུར་བཅོས་ཚུ་ འཕྱེལ་འགྱོ་དོ་ཡོདཔ་ཨིན།
གཞུང་སྐྱོང་ཚོགས་ཆུང་ཚུ་གིས་ བསྐྱར་ཞིབ་མ་འབད་བའི་ཧེ་མར་ གྲོས་འཆར་ཚུ། ཡིག་ཆ་འདི་གིས་ ངེས་འཛིན་འབདཝ་ཨིན།
ཀེར་ནིཀ་པེ་ལོཌ་ལས་འཆར་དང་ སྒྲུབ་བྱེད་དགོས་མཁོ་ དེ་ལས་ འདྲ་བཤུས་བརྟག་དཔྱད་ལམ་ལུགས་ཚུ།
དེ་འབདཝ་ལས་ བདེན་དཔྱད་འབད་མི་གསརཔ་ (`cargo xtask ministry-agenda validate`) གིས་ བཀོལ་སྤྱོད་འབདཝ་ཨིན།
གྲོས་འཆར་བཀོད་མི་ཚུ་གིས་ དྲྭ་ཚིགས་ནང་ སྐྱེལ་བཙུགས་མ་འབད་བའི་ཧེ་མར་ ས་གནས་ནང་ ཇེ་ཨེསི་ཨོན་ཕུལ་མི་ཚུ་ བཀལ་ཚུགས།
