# 执行摘要

该部根据 MINFO-8 完成了第一个完全透明的发布
计划。哈希和上下文审核模型保持在 ±3% 以内
点漂移带，而视觉政策由于低而受到抑制
样本计数，现在正在排队等待重新训练。上诉案件达 412 宗，
98.3% 的问题在 SLA 内解决，中位周转时间为 41.6 小时。
拒绝名单流失人数增加 76 项（四项紧急隔离）和财务
剩下的流量 **53150XOR** 为即将到来的争议预算保留。所有文物
以 ε 计数 = 0.75 / ε 准确度 = 0.5 进行消毒，锚定于
`artifacts/ministry/transparency/2026-Q3/`，并镜像到 SoraFS CID
`7f4c2d81a6b13579ccddeeff00112233`。

## 季度回顾

### 亮点
- 哈希 + 上下文模型保持在 ±3pp 假阳性/假阴性护栏内；视觉策略仅在 52 个样本后就被抑制，并安排了专门的再训练运行。
- 上诉：提出 412 起，解决 397 起，98.3% 属于 SLA 范围内，其中 7 起违规行为已上报至治理委员会； 18 起案件重新审理以获取更多证据。
- 财政部：148750XOR 存款 vs 95600XOR 支出，将 **53150XOR** 指定为第四季度争议储备金，同时志愿者覆盖范围达到 128 份简报（EN/JA/ES 的引用覆盖率为 91.5%）。
