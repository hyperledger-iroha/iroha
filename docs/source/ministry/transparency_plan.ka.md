# გამჭვირვალობა და აუდიტის ანგარიშები (MINFO-8)

საგზაო რუქის მითითება: **MINFO-8 — გამჭვირვალობა და აუდიტის ანგარიშები** და **MINFO-8a — კონფიდენციალურობის შენარჩუნების გამოშვების პროცესი**

ინფორმაციის სამინისტრომ უნდა გამოაქვეყნოს განმსაზღვრელი გამჭვირვალობის არტეფაქტები, რათა საზოგადოებამ შეძლოს ზომიერების ეფექტურობის აუდიტი, აპელაციების დამუშავება და შავი სიის გამორიცხვა. ეს დოკუმენტი განსაზღვრავს ფარგლებს, არტეფაქტებს, კონფიდენციალურობის კონტროლს და სამუშაო პროცესს, რომელიც საჭიროა MINFO-8-ის დახურვისთვის Q32026 სამიზნემდე.

## მიზნები და მიწოდება

- შექმენით კვარტალური გამჭვირვალობის პაკეტები, რომლებიც აჯამებენ ხელოვნური ინტელექტის ზომიერების სიზუსტეს, გასაჩივრების შედეგებს, უარმყოფელებს, მოხალისეთა პანელის აქტივობას და სახაზინო მოძრაობებს, რომლებიც დაკავშირებულია MINFO ბიუჯეტებთან.
- გაგზავნეთ თანმხლები ნედლეული მონაცემების პაკეტები (Norito JSON + CSV) პლუს დაფები, რათა მოქალაქეებმა შეძლონ მეტრიკის დაჭრა სტატიკური PDF-ების მოლოდინის გარეშე.
- განახორციელეთ კონფიდენციალურობის გარანტიები (დიფერენციალური კონფიდენციალურობა + მინიმალური დათვლის წესები) და ხელმოწერილი ატესტაციები ნებისმიერი მონაცემთა ნაკრების გამოქვეყნებამდე.
- ჩაიწერეთ თითოეული პუბლიკაცია მმართველობით DAG-სა და SoraFS-ში, რათა ისტორიული არტეფაქტები დარჩეს უცვლელი და დამოუკიდებლად გადამოწმებადი.

## მეტრიკა და დაფები

Grafana დაფა (`dashboards/grafana/ministry_transparency_overview.json`) აჩვენებს შემდეგ პანელებს:

- AI მოდერაციის სიზუსტე: თითო მოდელის FP/FN სიხშირე, დრიფტი კალიბრაციის სამიზნესთან და გაფრთხილების ზღურბლები მიბმული `docs/source/sorafs/reports/ai_moderation_calibration_*.md`-თან.
- აპელაციის სასიცოცხლო ციკლი: წარდგენები, SLA შესაბამისობა, შებრუნებები, ობლიგაციების დამწვრობა, თითო დონის ჩამორჩენა.
- Denylist churn: დამატებები/ამოღება თითო ჰეშის ოჯახზე, TTL ვადის გასვლა, საგანგებო კანონის გამოძახებები.
- მოხალისეთა ბრიფინგები და პანელის მრავალფეროვნება: წარდგენები თითო ენაზე, ინტერესთა კონფლიქტის გამჟღავნება, პუბლიკაციის შეფერხება. დაბალანსებული მოკლე ველები მითითებულია `docs/source/ministry/volunteer_brief_template.md`-ში, რაც უზრუნველყოფს ფაქტების ცხრილების და მოდერაციის ტეგების მანქანით წაკითხვას.
- სახაზინო ნაშთები: დეპოზიტები, გადახდები, დავალიანება (feed MINFO-7).

გაფრთხილების წესები (კოდირებული `dashboards/alerts/ministry_transparency_rules.yml`-ში) მოიცავს:
- FP/FN გადახრა >25% კალიბრაციის საწყისთან მიმართებაში.
- გასაჩივრება SLA გამოტოვების მაჩვენებელი >5% კვარტალში.
- გადაუდებელი კანონის TTL-ები უფრო ძველია ვიდრე პოლიტიკა.
- გამოქვეყნების შეფერხება >14 დღე კვარტლის დახურვის შემდეგ.
