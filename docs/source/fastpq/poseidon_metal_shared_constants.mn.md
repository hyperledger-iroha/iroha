# Посейдон металлын дундын тогтмолууд

Металл цөм, CUDA цөм, Rust prover болон SDK бэхэлгээ бүрийг хуваалцах ёстой.
Техник хангамжийг хурдасгахын тулд яг ижил Poseidon2 параметрүүд
хэш deterministic. Энэ баримт бичигт каноник агшин зуурын агшинг хэрхэн яаж хийхийг бичсэн
үүнийг дахин үүсгэх, GPU дамжуулах хоолой нь өгөгдлийг хэрхэн залгахыг хүлээж байна.

## нөхөн сэргэлт

Манифестийг Rust эх сурвалжаас `xtask` ажиллуулж үүсгэсэн.
туслагч. Энэ тушаал нь каноник файл болон SDK толин тусгалуудыг хоёуланг нь бичдэг:

```bash
cargo xtask offline-poseidon-fixtures --tag iroha.offline.receipt.merkle.v1
```

Очих газруудыг дарахын тулд `--constants <path>`/`--vectors <path>`-г ашиглана уу.
`--no-sdk-mirror` зөвхөн каноник агшин зуурын зургийг сэргээх үед. Туслах хүн болно
тугийг орхигдуулсан үед олдворуудыг Swift болон Android мод руу тусгах,
Энэ нь хэшийг CI-д нийцүүлэн байлгадаг.

## Feeding Metal/CUDA Builds

- `crates/fastpq_prover/metal/kernels/poseidon2.metal` ба
  `crates/fastpq_prover/cuda/fastpq_cuda.cu` -аас дахин сэргээгдэх ёстой
  Хүснэгт өөрчлөгдөх бүрт илэрдэг.
- Бөөрөнхий ба MDS тогтмолуудыг залгаа `MTLBuffer`/`__constant` болгон шаталсан.
  манифестын загварт тохирох сегментүүд: `round_constants[round][state_width]`
  дараа нь 3x3 MDS матриц.
- `fastpq_prover::poseidon_manifest()` нь агшин зуурын зургийг ачаалж баталгаажуулдаг
  ажиллах хугацаа (Металл дулаацуулах үед) тул оношилгооны хэрэгсэл нь үүнийг баталж чадна
  шэйдерийн тогтмолууд нь дамжуулан нийтлэгдсэн хэштэй таарч байна
  `fastpq_prover::poseidon_manifest_sha256()`.
- SDK төхөөрөмжийн уншигч (Swift `PoseidonSnapshot`, Android `PoseidonSnapshot`) болон
  Norito офлайн хэрэгсэл нь ижил манифест дээр тулгуурладаг бөгөөд энэ нь зөвхөн GPU-ээс сэргийлдэг.
  параметрийн сэрээ.

## Баталгаажуулалт

1. Манифестыг сэргээсний дараа `cargo test -p xtask`-г ажиллуулж
   Посейдоны бэхэлгээ үүсгэх нэгжийн туршилтууд.
2. Шинэ SHA-256-г энэ баримт бичиг болон хянадаг хяналтын самбарт тэмдэглэнэ үү
   GPU олдворууд.
3. `cargo test -p fastpq_prover poseidon_manifest_consistency` задлан шинжилнэ
   `poseidon2.metal` болон `fastpq_cuda.cu` нь бүтээн байгуулалтын үед байгаа бөгөөд тэдгээрийн
   Цуваачилсэн тогтмолууд нь манифесттэй таарч, CUDA/Metal хүснэгт болон
   түгжих алхам дахь каноник агшин зуурын зураг.Манифестийг GPU бүтээх зааврын дагуу хадгалах нь Metal/CUDA-г өгдөг
ажлын урсгал нь тодорхойлогч гар барих: цөм нь санах ойгоо оновчтой болгоход чөлөөтэй байдаг
