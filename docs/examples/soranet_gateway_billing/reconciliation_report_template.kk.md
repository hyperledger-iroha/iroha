# SoraGlobal Gateway есепшоттарын салыстыру

- **Терезе:** `<from>/<to>`
- **Жалға алушы:** `<tenant-id>`
- **Каталог нұсқасы:** `<catalog-version>`
- **Қолдану суреті:** `<path or hash>`
- **Қорғаулар:** жұмсақ қалпақ `<soft-cap-xor> XOR`, қатты қалпақ `<hard-cap-xor> XOR`, ескерту шегі `<alert-threshold>%`
- **Төлеуші -> Қазынашылық:** `<payer>` -> `<treasury>` `<asset-definition>`
- **Төлемнің жалпы сомасы:** `<total-xor> XOR` (`<total-micros>` micro-XOR)

## Жол элементін тексеру
- [ ] Пайдалану жазбалары тек каталог есептегіш идентификаторларын және жарамды есеп айырысу аймақтарын қамтиды
- [ ] Саны бірліктері каталог анықтамаларына сәйкес келеді (сұраулар, GiB, мс, т.б.)
- [ ] Каталогқа сәйкес қолданылатын аймақ көбейткіштері мен жеңілдік деңгейлері
- [ ] CSV/Parquet экспорттары JSON шот-фактура жол элементтеріне сәйкес келеді

## Қорғауды бағалау
- [ ] Жұмсақ қақпақ туралы ескерту шегіне жеттіңіз бе? `<yes/no>` (иә болса, ескерту дәлелін тіркеңіз)
- [ ] Қатты қақпақ асып кетті ме? `<yes/no>` (иә болса, қайта анықтауды растауды тіркеңіз)
- [ ] Шот-фактураның ең төменгі деңгейі қанағаттандырылды

## Бухгалтерлік журналдың проекциясы
