# Форма обратной связи по docs‑preview (волна W1)

Используйте этот шаблон при сборе обратной связи от ревьюеров волны W1. Для каждого
партнёра сделайте копию, заполните метаданные и сохраните готовый файл под
`artifacts/docs_preview/W1/preview-2025-04-12/feedback/<partner-id>/`.

## Метаданные ревьюера

- **ID партнёра:** `partner-w1-XX`
- **Тикет запроса:** `DOCS-SORA-Preview-REQ-PXX`
- **Отправка инвайта (UTC):** `YYYY-MM-DD hh:mm`
- **Подтверждение checksum (UTC):** `YYYY-MM-DD hh:mm`
- **Основные зоны фокуса:** (например, _документация оркестратора SoraFS_,
  _ISO‑флоу Torii_)

## Подтверждение телеметрии и артефактов

| Пункт чек‑листа | Результат | Доказательство |
| --- | --- | --- |
| Проверка checksums | ✅ / ⚠️ | Путь к логу (например, `build/checksums.sha256`) |
| Smoke‑тест Try it‑proxy | ✅ / ⚠️ | Фрагмент лога `npm run manage:tryit-proxy …` |
| Просмотр Grafana‑дашборда | ✅ / ⚠️ | Путь(и) к скриншоту(ам) |
| Просмотр отчёта probe‑портала | ✅ / ⚠️ | `artifacts/docs_preview/.../preflight-summary.json` |

Добавьте строки для любых дополнительных SLO, которые ревьюер проверяет.

## Журнал обратной связи

| Область | Severity (info/minor/major/blocker) | Описание | Предложенный фикс или вопрос | Issue для трекинга |
| --- | --- | --- | --- | --- |
| | | | | |

Ссылайтесь в последнем столбце на issue в GitHub или внутренний тикет, чтобы
preview‑tracker мог связать элементы ремедиации с этой формой.

## Итоги опроса

1. **Насколько вы уверены в инструкциях по checksums и процессе инвайтов?** (1–5)
2. **Какие разделы документации были наиболее/наименее полезными?** (краткий ответ)
3. **Возникали ли блокеры при доступе к Try it‑proxy или дашбордам телеметрии?**
4. **Требуется ли дополнительная локализация или контент по доступности (accessibility)?**
5. **Есть ли другие комментарии до выхода GA?**

Зафиксируйте краткие ответы и приложите «сырые» выгрузки опросов, если вы пользуетесь
внешней формой.

## Проверка знаний

- Оценка: `__/10`
- Неправильные вопросы (если есть): `[#1, #4, …]`
- Follow‑up‑действия (если оценка < 9/10): назначен ли созвон по ремедиации? да/нет
