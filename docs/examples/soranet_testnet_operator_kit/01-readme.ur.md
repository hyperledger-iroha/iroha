# SoraNet ٹیسٹ نیٹ آن بورڈنگ کٹ (مرحلہ T0)

خوش آمدید! آپ کو یہ کٹ بند SoraNet ٹیسٹ نیٹ (SNNet-10 مرحلہ T0) کے حصے کے طور پر مل رہی ہے۔ اپنے relay کو جوڑنے سے پہلے ہر چیک لسٹ آئٹم مکمل کریں۔ Governance Council آپ کی جمع شدہ دستاویزات کی جانچ کرے گا اس سے پہلے کہ آپ کے descriptors کو whitelist کیا جائے۔

اس ڈائریکٹری میں موجود چیزیں:

1. `01-readme.md` - یہ جائزہ.
2. `02-checklist.md` - آپریٹر کے پری فلائٹ کام.
3. `03-config-example.toml` - relay + orchestrator کی بنیادی کنفیگ.
4. `04-telemetry.md` - مطلوبہ ڈیش بورڈز اور الرٹ وائرنگ.
5. `05-incident-playbook.md` - brownout/downgrade ردعمل ورک فلو.
6. `06-verification-report.md` - smoke ٹیسٹ کے بعد جمع کرانے کا ٹیمپلیٹ.
7. `07-metrics-sample.json` - SNNet-10 میٹرکس ویلیڈیٹر کے لئے ان پٹ مثال.

کٹ کو نجی رکھیں؛ relay کی منظوری کے بعد ایک دستخط شدہ نقل governance کے ذریعے محفوظ کی جائے گی۔
