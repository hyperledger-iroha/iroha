# SoraNet Testnet საბორტო ნაკრები (ფაზა T0)

(SNNet-10 ფაზა T0). შეავსეთ საკონტროლო სიის ყველა ელემენტი თქვენს დაკავშირებამდე
რელე. მმართველი საბჭო დაადასტურებს თქვენს წარდგენას თეთრ სიაში შესვლამდე
შენი აღწერები.

არტეფაქტები ამ დირექტორიაში:

1. `01-readme.md` — ეს მიმოხილვა.
2. `02-checklist.md` — ოპერატორის ფრენისწინა დავალებები.
3. `03-config-example.toml` — საბაზისო რელე + ორკესტრატორის კონფიგურაცია.
4. `04-telemetry.md` — საჭირო დაფები და გაფრთხილების გაყვანილობა.
5. `05-incident-playbook.md` — ბრუნაუტ/დაქვეითება რეაგირების სამუშაო პროცესი.
6. `06-verification-report.md` — შაბლონი წარსადგენი კვამლის ტესტების შემდეგ.
7. `07-metrics-sample.json` — შეყვანის მაგალითი SNNet-10 მეტრიკის ვალიდატორისთვის.
