# ערכת קליטה לרשת הבדיקה SoraNet (שלב T0)

ברוכים הבאים! אתם מקבלים ערכה זו כחלק מרשת הבדיקה הסגורה של SoraNet (SNNet-10 שלב T0). השלימו כל פריט ברשימת הבדיקה לפני חיבור ה-relay שלכם. מועצת הממשל תאמת את ההגשה לפני הכללת המזהים ברשימת ההיתרים.

הארטיפקטים בתיקיה זו:

1. `01-readme.md` — סקירה זו.
2. `02-checklist.md` — משימות קדם-הפעלה למפעיל.
3. `03-config-example.toml` — תצורת baseline ל-relay + orchestrator.
4. `04-telemetry.md` — דשבורדים נדרשים וחיווט התראות.
5. `05-incident-playbook.md` — תהליך תגובה ל-brownout/downgrade.
6. `06-verification-report.md` — תבנית להגשה לאחר בדיקות smoke.
7. `07-metrics-sample.json` — דוגמת קלט למאמת מדדים של SNNet-10.

שמרו על הערכה פרטית; עותק חתום יאוחסן על ידי הממשל לאחר אישור ה-relay.
