# Kit de onboarding del testnet SoraNet (fase T0)

Bienvenido! Recibes este kit como parte del testnet cerrado de SoraNet (SNNet-10 fase T0). Completa cada elemento de la checklist antes de conectar tu relay. El Consejo de Governance validara tu envio antes de poner tus descriptores en lista blanca.

Artefactos en este directorio:

1. `01-readme.md` - este resumen.
2. `02-checklist.md` - tareas previas del operador.
3. `03-config-example.toml` - configuracion base de relay + orchestrator.
4. `04-telemetry.md` - dashboards requeridos y conexion de alertas.
5. `05-incident-playbook.md` - flujo de respuesta a brownout/downgrade.
6. `06-verification-report.md` - plantilla a enviar despues de las pruebas smoke.
7. `07-metrics-sample.json` - ejemplo de entrada para el validador de metricas SNNet-10.

Manten el kit en privado; una copia firmada sera archivada por governance una vez que tu relay sea aprobado.
