# SoraNet Testnet ներբեռնման հավաքածու (Փուլ T0)

Բարի գալուստ Դուք ստանում եք այս փաթեթը որպես փակ SoraNet թեստային ցանցի մի մաս
(SNNet-10 փուլ T0): Լրացրեք ստուգաթերթի բոլոր կետերը նախքան ձերը միացնելը
ռելե. Կառավարման խորհուրդը կհաստատի ձեր ներկայացումը նախքան սպիտակ ցուցակում ներառելը
ձեր նկարագրիչները:

Արտեֆակտներ այս գրացուցակում.

1. `01-readme.md` — այս ակնարկ.
2. `02-checklist.md` — օպերատորի նախաթռիչքային առաջադրանքներ:
3. `03-config-example.toml` — ելակետային ռելե + նվագախմբի կոնֆիգուրացիա:
4. `04-telemetry.md` — պահանջվող վահանակներ և ազդանշանային լարեր:
5. `05-incident-playbook.md` — անսարքություն/նվազեցում արձագանքման աշխատանքային հոսք:
6. `06-verification-report.md` — կաղապար՝ ծխի թեստերից հետո ներկայացնելու համար:
7. `07-metrics-sample.json` — օրինակ մուտքագրում SNNet-10 չափումների վավերացման համար:

Կոմպլեկտը գաղտնի պահեք; ստորագրված պատճենը կարխիվացվի կառավարման կողմից, երբ ձեր կողմից
