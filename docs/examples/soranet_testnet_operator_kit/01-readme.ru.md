# Набор для онбординга тестнета SoraNet (фаза T0)

Добро пожаловать! Вы получаете этот набор как часть закрытого тестнета SoraNet (SNNet-10 фаза T0). Завершите каждый пункт чеклиста перед подключением вашего relay. Совет по управлению проверит вашу отправку перед добавлением ваших дескрипторов в белый список.

Артефакты в этом каталоге:

1. `01-readme.md` — этот обзор.
2. `02-checklist.md` — предварительные задачи оператора.
3. `03-config-example.toml` — базовая конфигурация relay + orchestrator.
4. `04-telemetry.md` — обязательные дашборды и подключение алертов.
5. `05-incident-playbook.md` — рабочий процесс реагирования на brownout/downgrade.
6. `06-verification-report.md` — шаблон для отправки после smoke тестов.
7. `07-metrics-sample.json` — пример входных данных для валидатора метрик SNNet-10.

Держите набор приватным; подписанная копия будет архивирована governance после утверждения вашего relay.
