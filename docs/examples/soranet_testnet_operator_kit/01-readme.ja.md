# SoraNet テストネット オンボーディングキット（フェーズ T0）

このディレクトリのアーティファクト:

1. `01-readme.md` — この概要。
2. `02-checklist.md` — オペレーターの事前タスク。
3. `03-config-example.toml` — relay + orchestrator の基本設定。
4. `04-telemetry.md` — 必須ダッシュボードとアラート配線。
5. `05-incident-playbook.md` — brownout/downgrade 対応フロー。
6. `06-verification-report.md` — スモークテスト後に提出するテンプレート。
7. `07-metrics-sample.json` — SNNet-10 メトリクス検証の入力例。

このキットは非公開にしてください。relay の承認後、署名済みコピーが governance によりアーカイブされます。
