# SoraNet Testnet суулгах хэрэгсэл (Phase T0)

Тавтай морил! Та энэ багцыг хаалттай SoraNet тестийн сүлжээний нэг хэсэг болгон хүлээн авч байна
(SNNet-10 фазын T0). Холбохын өмнө шалгах хуудас бүрийг бөглөнө үү
реле. Удирдлагын зөвлөл нь зөвшөөрөгдсөн жагсаалтад оруулахаас өмнө таны ирүүлсэн материалыг баталгаажуулна
таны тайлбарлагчид.

Энэ лавлах дахь олдворууд:

1. `01-readme.md` — энэ тойм.
2. `02-checklist.md` — операторын нислэгийн өмнөх даалгавар.
3. `03-config-example.toml` — үндсэн реле + найруулагчийн тохиргоо.
4. `04-telemetry.md` — шаардлагатай хяналтын самбар болон дохиоллын утас.
5. `05-incident-playbook.md` — гацах/дуулах хариу үйлдлийн урсгал.
6. `06-verification-report.md` — утааны туршилтын дараа ирүүлэх загвар.
7. `07-metrics-sample.json` — SNNet-10 хэмжигч баталгаажуулагчийн жишээ оролт.

Багцыг нууцлах; гарын үсэг зурсан хуулбарыг нэгэнт архивлах болно
реле батлагдсан.
