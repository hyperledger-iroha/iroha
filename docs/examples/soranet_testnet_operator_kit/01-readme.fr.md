# Kit d'onboarding du testnet SoraNet (phase T0)

Bienvenue! Vous recevez ce kit dans le cadre du testnet ferme de SoraNet (SNNet-10 phase T0). Terminez chaque element de la checklist avant de connecter votre relay. Le Governance Council validera votre soumission avant de mettre vos descripteurs en liste blanche.

Artefacts dans ce repertoire:

1. `01-readme.md` - cet apercu.
2. `02-checklist.md` - taches pre-vol operateur.
3. `03-config-example.toml` - configuration de base relay + orchestrator.
4. `04-telemetry.md` - dashboards requis et branchement des alertes.
5. `05-incident-playbook.md` - workflow de reponse brownout/downgrade.
6. `06-verification-report.md` - modele a soumettre apres les tests smoke.
7. `07-metrics-sample.json` - exemple d'entree pour le validateur de metriques SNNet-10.

Gardez le kit prive; une copie signee sera archivee par la governance une fois votre relay approuve.
