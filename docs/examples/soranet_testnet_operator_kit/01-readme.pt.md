# Kit de onboarding do testnet SoraNet (fase T0)

Bem-vindo! Voce esta recebendo este kit como parte do testnet fechado de SoraNet (SNNet-10 fase T0). Complete cada item do checklist antes de conectar seu relay. O Conselho de Governance validara sua submissao antes de colocar seus descritores na lista branca.

Artefatos neste diretorio:

1. `01-readme.md` - este resumo.
2. `02-checklist.md` - tarefas previas do operador.
3. `03-config-example.toml` - configuracao base de relay + orchestrator.
4. `04-telemetry.md` - dashboards requeridos e conexao de alertas.
5. `05-incident-playbook.md` - fluxo de resposta a brownout/downgrade.
6. `06-verification-report.md` - template a enviar apos os testes smoke.
7. `07-metrics-sample.json` - exemplo de entrada para o validador de metricas SNNet-10.

Mantenha o kit privado; uma copia assinada sera arquivada pela governance assim que seu relay for aprovado.
