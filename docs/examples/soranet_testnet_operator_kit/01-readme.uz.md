(SNNet-10 faza T0). Ulanishdan oldin nazorat roʻyxatining har bir bandini toʻldiring
rele. Boshqaruv Kengashi oq roʻyxatga kiritishdan oldin taqdimotingizni tasdiqlaydi
tavsiflovchilaringiz.

5. `05-incident-playbook.md` - qo'ng'iroq qilish/pastlash javobi ish jarayoni.

reley tasdiqlangan.
