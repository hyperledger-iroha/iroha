# SoraNet Testnet Onboarding Kit (Phase T0)

ကြိုဆိုပါတယ် သင်သည် ပိတ်ထားသော SoraNet testnet ၏ တစ်စိတ်တစ်ပိုင်းအနေဖြင့် ဤကိရိယာကို လက်ခံရရှိနေပါသည်။
(SNNet-10 အဆင့် T0)။ သင့်အား မချိတ်ဆက်မီ စစ်ဆေးရန်စာရင်းတစ်ခုစီကို ဖြည့်စွက်ပါ။
ထပ်ဆင့်လွှင့်။ အုပ်ချုပ်မှုကောင်စီသည် သင့်တင်ပြချက်ကို တရားဝင်ခွင့်ပြုမထားပါ။
သင်၏ဖော်ပြချက်များ။

ဤလမ်းညွှန်ရှိ ပစ္စည်းများ-

1. `01-readme.md` — ဤသုံးသပ်ချက်။
2. `02-checklist.md` — အော်ပရေတာ ပျံသန်းမှုအကြိုလုပ်ဆောင်စရာများ။
3. `03-config-example.toml` — အရင်းခံ ထပ်ဆင့်လွှင့် + သံစုံတီးဝိုင်း ဖွဲ့စည်းမှု။
4. `04-telemetry.md` — လိုအပ်သော ဒက်ရှ်ဘုတ်များနှင့် သတိပေးချက် ဝိုင်ယာကြိုးများ။
5. `05-incident-playbook.md` — အညိုရောင်ထွက်/နှိမ့်ချသည့် တုံ့ပြန်မှုလုပ်ငန်းအသွားအလာ။
6. `06-verification-report.md` — မီးခိုးစမ်းသပ်မှုများပြီးနောက် တင်ပြရန် နမူနာပုံစံ။
7. `07-metrics-sample.json` — SNNet-10 မက်ထရစ်စစ်ဆေးခြင်းအတွက် ဥပမာထည့်သွင်းခြင်း။

ပစ္စည်းအစုံကို သီးသန့်ထားပါ။ လက်မှတ်ရေးထိုးထားသောမိတ္တူကို အုပ်ချုပ်မှုစနစ်ဖြင့် သိမ်းဆည်းသွားပါမည်။
relay ကိုအတည်ပြုသည်။
