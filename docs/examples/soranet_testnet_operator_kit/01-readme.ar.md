# حزمة الانضمام لشبكة اختبار SoraNet (المرحلة T0)

مرحبا! تتلقى هذه الحزمة كجزء من شبكة اختبار SoraNet المغلقة (SNNet-10 المرحلة T0). اكمل كل بند في قائمة التحقق قبل توصيل relay الخاص بك. سيقوم مجلس الحوكمة بالتحقق من تسليمك قبل ادراج المعرّفات في القائمة البيضاء.

القطع الموجودة في هذا الدليل:

1. `01-readme.md` - هذه النظرة العامة.
2. `02-checklist.md` - مهام ما قبل التشغيل للمشغل.
3. `03-config-example.toml` - اعدادات baseline للrelay + orchestrator.
4. `04-telemetry.md` - لوحات المعلومات المطلوبة وربط التنبيهات.
5. `05-incident-playbook.md` - سير عمل الاستجابة للbrownout/downgrade.
6. `06-verification-report.md` - قالب للتسليم بعد اختبارات smoke.
7. `07-metrics-sample.json` - مثال ادخال لمدقق مقاييس SNNet-10.

احتفظ بالحزمة بشكل خاص؛ سيتم ارشفة نسخة موقعة بواسطة الحوكمة بمجرد اعتماد relay الخاص بك.
