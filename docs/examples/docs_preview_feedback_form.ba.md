1. **Һеҙ ни тиклем ышаныслы һеҙ тикшерелгән һәм саҡырыу процесын саҡыра?** (1–5)

3. **Блокерҙар бар ине, уны прокси йәки телеметрия панелдәрен һынап ҡарау?**
4. **Өҫтәмә локализация йәки ҡулланыу мөмкинлеге кәрәкме?**
