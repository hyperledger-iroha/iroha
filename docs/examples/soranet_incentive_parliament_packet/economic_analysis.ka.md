# ეკონომიკური ანალიზი - 2025-10 -> 2025-11 Shadow Run

წყაროს არტეფაქტი: `docs/examples/soranet_incentive_shadow_run.json` (ხელმოწერა +
საჯარო გასაღები იმავე დირექტორიაში). სიმულაციამ გაიმეორა 60 ეპოქა თითო რელეზე
ჯილდოს ძრავით მიმაგრებული `RewardConfig`-ში ჩაწერილი
`reward_config.json`.

## განაწილების რეზიუმე

- **სულ გადახდები:** 5160 XOR 360 დაჯილდოვებულ ეპოქაში.
- **სამართლიანობის კონვერტი:** ჯინის კოეფიციენტი 0,121; ზედა სარელეო წილი 23.26%
  (საკმაოდ ქვემოთ 30%-იანი მმართველობის ღობე).
- **ხელმისაწვდომობა:** ფლოტის საშუალოდ 96.97%, ყველა რელე დარჩა 94%-ზე ზემოთ.
- **გამტარუნარიანობა:** ფლოტი საშუალოდ 91,20%, ყველაზე დაბალი მაჩვენებლით 87,23%
  დაგეგმილი მოვლის დროს; ჯარიმები ავტომატურად იქნა გამოყენებული.
- ** შესაბამისობის ხმაური: ** დაფიქსირდა 9 გამაფრთხილებელი ეპოქა და 3 შეჩერება და
  გადაითარგმნა გადახდის შემცირებაში; არცერთი რელე არ აჭარბებდა 12-გამაფრთხილებელ თავსახურს.
- **ოპერაციული ჰიგიენა:** არცერთი მეტრიკის ანაბეჭდი გამოტოვებული არ არის გამოტოვების გამო
  კონფიგურაცია, ობლიგაციები ან დუბლიკატები; არ გამოსულა კალკულატორის შეცდომები.

## დაკვირვებები

- შეჩერებები შეესაბამება იმ ეპოქებს, სადაც რელეები შედიოდა შენარჩუნების რეჟიმში. The
  გადახდის ძრავამ გამოუშვა ნულოვანი ანაზღაურება იმ ეპოქებისთვის, ხოლო შენარჩუნებისას
  აუდიტის ბილიკი ჩრდილოვანი JSON-ში.
- გამაფრთხილებელი ჯარიმები შემცირდა 2%-ით დაზარალებულ გადახდებზე; შედეგად მიღებული
  განაწილება კვლავ თანხვედრაშია დროის/სიჩქარის წონების წყალობით (650/350
  თითო მილზე).
- გამტარუნარიანობის ვარიაცია აკონტროლებს ანონიმურ მცველთა სითბოს რუკას. ყველაზე დაბალი შემსრულებელი
  (`6666...6666`) შეინარჩუნა 620 XOR ფანჯრის გასწვრივ, 0.6x სართულის ზემოთ.
- შეყოვნებისადმი მგრძნობიარე გაფრთხილებები (`SoranetRelayLatencySpike`) დარჩა გაფრთხილების ქვემოთ
  ზღურბლები მთელ ფანჯარაში; კორელირებული დაფები აღბეჭდილია ქვეშ
  `dashboards/grafana/soranet_incentives.json`.

## რეკომენდებული მოქმედებები GA-მდე

1. გააგრძელეთ ყოველთვიური ჩრდილის გამეორებები და განაახლეთ არტეფაქტების ნაკრები და ეს
   ანალიზი, თუ იცვლება ფლოტის შემადგენლობა.
2. კარიბჭის ავტომატური გადახდები Grafana გაფრთხილების კომპლექტზე, რომელიც მითითებულია საგზაო რუკაზე
   (`dashboards/alerts/soranet_incentives_rules.yml`); დააკოპირეთ ეკრანის ანაბეჭდები მასში
   მმართველობის წუთები განახლების მოთხოვნისას.
3. ხელახლა გაუშვით ეკონომიკური სტრესის ტესტი, თუ ბაზის ჯილდო, დრო/სიჩქარის წონა, ან
   შესაბამისობის ჯარიმა იცვლება >=10%-ით.
