# ཡིག་ཆ་ཚུ་ དྲྭ་ཚིགས་སྔོན་ལྟ་འབོད་བརྡ་ (ཊེམ་པེལེན)།

བསྐྱར་ཞིབ་འབད་མི་ཚུ་ལུ་ སྔོན་ལྟའི་འཛུལ་སྤྱོད་བཀོད་རྒྱ་ཚུ་གཏང་པའི་སྐབས་ ཊེམ་པེལེཊི་འདི་ལག་ལེན་འཐབ། བརྗེས༌སོར༌འབད༌ནི
འབྲེལ་ཡོད་གནས་གོང་ཚུ་དང་གཅིག་ཁར་ ས་གནས་འཛིན་མི་ (`<...>`) འགྲེལ་བཤད་ + མཉམ་སྦྲགས་འབད།
ཡིག་མཛོད་ཀྱི་ ཅ་རྙིང་ཚུ་ བརྡ་འཕྲིན་ནང་ གཞི་བསྟུན་འབད་དེ་ མཐའ་མཇུག་གི་ཚིག་ཡིག་འདི་ ནང་ན་ནང་ གསོག་འཇོག་འབད།
འབྲེལ་མཐུན་གྱི་ རིན་བསྡུར་གྱི་ འཛུལ་ཞུགས་ཤོག་བྱང་།
