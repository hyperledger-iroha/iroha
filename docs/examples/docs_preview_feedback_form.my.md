## စစ်တမ်းအနှစ်ချုပ်

1. ** checksum လမ်းညွှန်ချက်နှင့် ဖိတ်ကြားခြင်းလုပ်ငန်းစဉ်တွင် သင် မည်မျှယုံကြည်မှုရှိသနည်း။** (1-5)
2. **မည်သည့် docs သည် အသုံးဝင်ဆုံး/အနည်းဆုံးဖြစ်သနည်း။** (အတိုချုပ် အဖြေ)
3. ** Try it proxy သို့မဟုတ် တယ်လီမီတာ ဒက်ရှ်ဘုတ်များကို ပိတ်ဆို့ထားသူများ ရှိပါသလား။**
4. ** ထပ်လောင်းဒေသခံအဖြစ်သတ်မှတ်ခြင်း သို့မဟုတ် ဝင်ရောက်နိုင်မှုအကြောင်းအရာ လိုအပ်ပါသလား။**
5. **GA မတိုင်မီ အခြားမှတ်ချက်များ ရှိပါသလား။**

အကယ်၍ သင်သည် ပြင်ပပုံစံကို အသုံးပြုပါက အဖြေတိုတိုများကို ဖမ်းယူပြီး စစ်တမ်းကြမ်းထုတ်ယူမှုများကို ပူးတွဲပါ။
