# Torii API (Swagger UI)

This page embeds the Torii OpenAPI document using `swagger-ui-react`. Use it when
you want the familiar Swagger layout, request samples, and the ability to try
out requests directly within the browser.

## Usage tips

- Provide a bearer token in the panel to pre-authorise every request. Tokens are
  never persisted.
- Try the proxy-backed “[Try it](/api/overview#try-it-sandbox-docs-3)” console
  for staged environments where CORS prevents direct Torii access.
- Keep the OpenAPI snapshot fresh with `cargo xtask openapi` (see
  `docs/portal/static/openapi/torii.json`).
