# Norito Codec Һылтанма

Norito Iroha’s канон сериализация ҡатламы. Һәр сымлы хәбәр, диск өҫтөндә
файҙалы йөк, һәм кросс-компонент API ҡуллана Norito шулай төйөндәр менән килешә бер үк байт
хатта улар төрлө аппаратта йүгергәндә лә. Был биттә хәрәкәт итеүсе өлөштәре дөйөмләштерелә .
һәм `norito.md`-та тулы спецификацияға күрһәтә.

## Ядро планировкаһы

| Компонент | Маҡсат | Сығанаҡ |
| --- | --- | --- |
| **Башыраҡ** | Һөйләшеүҙәр үҙенсәлектәр (төйөлгән структур/эҙмә-эҙлеклелек, компакт оҙонлоҡ, ҡыҫыу флагтары) һәм CRC64 тикшерелгән сумманы индерә, шуға күрә файҙалы йөк бөтөнлөгө тикшерелгәнгә тиклем тикшерелә. | `norito::header` — ҡарағыҙ `norito.md` (“Баш һәм флагтар”, һаҡлағыс тамыр) |
| **Яланғас файҙалы йөк** | Детерминистик ҡиммәт кодлау өсөн ҡулланылған хеширование/сағыштырыу. Шул уҡ планировка башы менән транспортҡа төрөлгән. | `norito::codec::{Encode, Decode}` |
| **Ҡыҫыу** | Zstd (һәм эксперименталь GPU тиҙләнеше) `compression` флагы аша әүҙемләштерелгән. | `norito.md`, “Ҡыҫыу тураһында һөйләшеүҙәр” |

Тулы флаг реестры (пакетлы-структура, ҡапланған-seq, компакт оҙонлоҡтар, ҡыҫыу)
йәшәй `norito::header::flags`. `norito::header::Flags` уңайлыҡтарҙы фашлай
эшләү ваҡыты тикшерелеүен тикшерә; запас макет биттары декодерҙар тарафынан кире ҡағыла.

## Ярҙам алыу

`norito_derive` суднолары `Encode`, `Decode`, `IntoSchema`, һәм JSON ярҙамсы сыға.
Төп конвенциялар:

- Структуралар/анулалар `packed-struct` функцияһы булғанда ҡапланған макеттар сыға.
  өҫтөндә эшләй (дефолт). Ғәмәлгә ашырыу `crates/norito_derive/src/derive_struct.rs`-та йәшәй
  ә был ҡылыҡ `norito.md` (“Пакле макеттар”) документлаштырылған.
- Ҡапланған коллекциялар v1-ҙә нығытылған киңлектәге эҙмә-эҙлеклелек башлыҡтарын һәм офсеттарҙы ҡуллана; генә
  Ҡиммәт оҙонлоғо буйынса префикстар `COMPACT_LEN` менән йоғонто яһай.
- JSON ярҙамсылары (`norito::json`) детерминистик Norito JSON өсөн детерминистик тәьмин итә.
  асыҡ API-лар. Ҡулланыу `norito::json::{to_json_pretty, from_json}` — бер ҡасан да `serde_json`.

## Мультодек һәм идентификатор таблицалары

Norito үҙенең күп кодекслы заданиеларын һаҡлай `norito::multicodec`. Һылтанма
таблица (хаштар, төп төрҙәре, файҙалы йөк дескрипторҙары) `multicodec.md`-та һаҡлана.
һаҡлағыс тамырында урынлашҡан. Яңы идентификатор өҫтәлгәндә:

1. Яңыртыу `norito::multicodec::registry`.
2. `multicodec.md`-тағы таблицаны оҙайтырға.
3. Регенерациялау нижестоящий бәйләү (Python/Java) әгәр улар карта ҡулланыу.
