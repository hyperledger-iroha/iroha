Энэ хэсэг нь Iroha-д зориулсан "үүнийг техникийн үзүүлэлт болгон унших" материалыг нэгтгэдэг. Эдгээр хуудсууд нь тогтвортой хэвээр байна
гарын авлага, зааварчилгаанууд хөгжиж байна.

## Өнөөдөр бэлэн байна

- **Norito кодлогчийн тойм** – `reference/norito-codec.md` эрх бүхий байгууллага руу шууд холбогддог.
  Портал хүснэгтийг бөглөж байх үед `norito.md` тодорхойлолт.
- **Torii OpenAPI** – `/reference/torii-openapi` нь хамгийн сүүлийн үеийн Torii REST үзүүлэлтийг ашиглан харуулав.
  Redoc. Тодорхойлолтыг `npm run sync-openapi` ашиглан сэргээнэ үү.
- **Тохиргооны хүснэгтүүд** – Параметрийн бүрэн каталогийг хадгална
  `docs/source/references/configuration.md`. Портал автомат импортыг илгээх хүртэл үүнийг лавлана уу
  Тодорхой өгөгдмөл болон орчны хүчингүйд тооцогдох Markdown файл.

## Удахгүй

- **Torii АМРАХ лавлагаа** – OpenAPI тодорхойлолтуудыг энэ хэсэгт синк хийх болно
  Дамжуулах хоолойг идэвхжүүлсний дараа `docs/portal/scripts/sync-openapi.mjs`.
- **CLI командын индекс** – Үүсгэсэн тушаалын матриц (`crates/iroha_cli/src/commands` толин тусгал)
  каноник жишээнүүдийн хажууд энд буух болно.
- **IVM ABI хүснэгтүүд** – Заагч төрлийн болон системийн дуудлагын матрицууд (`crates/ivm/docs` дагуу хадгалагдана)
  Док үүсгэх ажлыг залгасны дараа портал руу оруулна.

## Энэ индексийг одоогийн байлгах

Шинэ лавлагаа материалыг нэмэхэд - үүсгэсэн API баримтууд, кодлогчийн үзүүлэлтүүд, тохиргооны матрицуудыг байрлуулна
`docs/portal/docs/reference/` доор байгаа хуудсуудыг холбож, дээр нь холбоно уу. Хэрэв хуудас автоматаар үүсгэгдсэн бол үүнийг анхаарна уу
синхрончлолын скрипт, ингэснээр хувь нэмэр оруулагчид үүнийг хэрхэн сэргээхээ мэддэг. Энэ нь лавлагаа модыг ашиглах хүртэл ашигтай байлгадаг
бүрэн автоматаар хийгдсэн навигацийн газар.
