## དེ་རིང་ཐོབ་ཐུབ།

- **Norito གསང་གྲངས་སྤྱི་ཁྱབ་མཐོང་སྣང་** – `reference/norito-codec.md` ཚུ་ དབང་ཚད་ཅན་ལུ་ཐད་ཀར་དུ་འབྲེལ་མཐུད་འབདཝ་ཨིན།
  `norito.md` གསལ་བཀོད་འབདཝ་ད་ དྲྭ་ཐོག་ཐིག་ཁྲམ་འདི་ མི་རློབས་བཙུགས་དོ་ཡོདཔ་ཨིན།
- **Torii OpenAPI*** – `/reference/torii-openapi` ལག་ལེན་འཐབ་སྟེ་ གསར་ཤོས་ Torii REST ཁྱད་ཚད་འདི་སྟོནམ་ཨིན།
  Redoc. `npm run sync-openapi` དང་མཉམ་དུ་ ཁྱད་ཚད་བསྐྱར་བཟོ་འབད།
- **རིམ་སྒྲིག་ཐིག་ཁྲམ་ཚུ་** ཚད་བཟུང་ཐོ་གཞུང་ཆ་ཚང་ནང་བཞག་ཡོདཔ་ཨིན།
  `docs/source/references/configuration.md`. དྲྭ་ཚིགས་འདི་གིས་ རང་བཞིན་ནང་འདྲེན་འབད་མ་ཚུགས་ཚུན་ཚོད་ གཞི་བསྟུན་འབད།
  སྔོན་སྒྲིག་དང་མཐའ་འཁོར་གནས་སོར་ཚུ་གི་དོན་ལུ་ མཱརཀ་ཌའོན་ཡིག་སྣོད་།

## མགྱོགས༌པ༌རང༌འོང༌དོ

- **Torii REST གཞི་བསྟུན་* – OpenAPI ངེས་ཚིག་ཚུ་ དབྱེ་ཚན་འདི་ནང་ མཉམ་འབྱུང་འབད་འོང་།
  མདོང་ལམ་འདི་ལྕོགས་ཅན་བཟོ་ཚར་བའི་ཤུལ་ལས་ `docs/portal/scripts/sync-openapi.mjs` ཨིན།
- **CLI བརྡ་བཀོད་ཟུར་ཐོ་** བཟོ་བཏོན་བརྡ་བཀོད་མེ་ཊིགསི་ (མེ་ལོང་ `crates/iroha_cli/src/commands`)
  འདི་ཁར་ ཁྲིམས་ལུགས་ཀྱི་དཔེ་ཚུ་དང་གཅིག་ཁར་ ཆགས་འོང་།
- **IVM ABI ཐིག་ཁྲམ་ཚུ་** – དཔག་བྱེད་དབྱེ་བ་དང་ སི་ཀཱལ་མེ་ཊིགསི་ (`crates/ivm/docs` གི་འོག་ལུ་ རྒྱུན་སྐྱོང་འཐབ་ཡོདཔ་)།
  ཡིག་ཆ་བཟོ་ནིའི་ལཱ་འདི་ གློག་ཐག་བཏང་ཚར་བའི་ཤུལ་ལས་ དྲྭ་ཐོག་ནང་ལུ་ བཀོད་བཞག་འོང་།

## ཟུར་ཐོ་འདི་ད་ལྟོ་བཞག་དོ།

གཞི་བསྟུན་དངོས་པོ་གསརཔ་ཁ་སྐོང་བརྐྱབ་པའི་སྐབས་—ཨེ་པི་ཨའི་ཡིག་ཆ་དང་ཀོམ་ཀོཌི་སི་ཁྱད་ཚད་ རིམ་སྒྲིག་མེ་ཊིསི་—ས་སྒོ།
ཤོག་ལེབ་ `docs/portal/docs/reference/` འོག་ལུ་ཡོད་པའི་ཤོག་ལེབ་འདི་ གོང་ལུ་འབྲེལ་མཐུད་འབད། ཤོག་ལེབ་ཅིག་རང་བཞིན་བཟོ་བཏོན་འབད་བ་ཅིན་ དྲན་འཛིན་འབད།
མཉམ་མཐུད་ཡིག་ཚུགས་ དེ་འབདཝ་ལས་ ཕན་འདེབས་འབད་མི་ཚུ་གིས་ གསར་བསྐྲུན་འབད་ཐངས་ཤེས། འདི་གིས་ གཞི་བསྟུན་ཤིང་འདི་ ལས་ ལུ་ཕན་ཐོགས་ཅན་སྦེ་བཞགཔ་ཨིན།
ཆ་ཚང་སྦེ་བཟོ་མི་ འགྲུལ་བཞུད་ཀྱི་ས་ཆ་ཚུ།
