# Norito Կոդեկ տեղեկանք

Norito-ը Iroha-ի կանոնական սերիալացման շերտն է: Ամեն մի հաղորդագրություն, սկավառակի վրա
օգտակար բեռնվածություն, և խաչաձեւ բաղադրիչ API-ն օգտագործում է Norito, այնպես որ հանգույցները համաձայնվում են նույնական բայթերի վրա
նույնիսկ երբ դրանք աշխատում են տարբեր սարքավորումների վրա: Այս էջում ամփոփված են շարժվող մասերը
և մատնանշում է ամբողջական բնութագրերը `norito.md`-ում:

## Հիմնական դասավորությունը

| Բաղադրիչ | Նպատակը | Աղբյուր |
| --- | --- | --- |
| **Վերնագիր** | Բանակցում է առանձնահատկությունները (փաթեթավորված կառուցվածքներ/հաջորդականություններ, կոմպակտ երկարություններ, սեղմման դրոշակներ) և տեղադրում է CRC64 ստուգիչ գումարը, որպեսզի բեռնվածքի ամբողջականությունը ստուգվի նախքան ապակոդավորումը: | `norito::header` — տես `norito.md` («Header & Flags», պահեստի արմատ) |
| **Մերկ օգտակար բեռ ** | Դետերմինիստական ​​արժեքի կոդավորում, որն օգտագործվում է հեշինգի/համեմատման համար: Նույն դասավորությունը փաթաթված է վերնագրի կողմից տրանսպորտի համար: | `norito::codec::{Encode, Decode}` |
| **Սեղմում** | Ընտրովի Zstd (և փորձարարական GPU արագացում) ակտիվացված է `compression` դրոշի բայթի միջոցով: | `norito.md`, «Կոմպրեսիոն բանակցություններ» |

Դրոշի ամբողջական ռեգիստրը (փաթեթավորված կառուցվածք, փաթեթավորված հաջորդականություն, կոմպակտ երկարություններ, սեղմում)
ապրում է `norito::header::flags`-ում: `norito::header::Flags` բացահայտում է հարմարավետությունը
ստուգումներ գործարկման ժամանակի ստուգման համար; վերապահված դասավորության բիթերը մերժվում են ապակոդավորիչների կողմից:

## Ստացեք աջակցություն

`norito_derive` առաքվում է `Encode`, `Decode`, `IntoSchema` և JSON օգնականը ստացվում է:
Հիմնական կոնվենցիաներ.

- Structs/enums-ը ստանում են փաթեթավորված դասավորություններ, երբ `packed-struct` հատկանիշն է
  միացված է (կանխադրված): Իրականացումն ապրում է `crates/norito_derive/src/derive_struct.rs`-ում
  և վարքագիծը փաստաթղթավորված է `norito.md`-ում («Փաթեթավորված դասավորություններ»):
- Փաթեթավորված հավաքածուները օգտագործում են ֆիքսված լայնության հաջորդականության վերնագրեր և օֆսեթներ v1-ում; միայն
  մեկ արժեքի երկարության նախածանցների վրա ազդում է `COMPACT_LEN`-ը:
- JSON օգնականները (`norito::json`) ապահովում են դետերմինիստական Norito-ով ապահովված JSON
  բացել API-ները: Օգտագործեք `norito::json::{to_json_pretty, from_json}` — երբեք `serde_json`:
