Бұл бөлім Iroha үшін «оны спецификация ретінде оқу» материалын біріктіреді. Бұл беттер сияқты тұрақты болып қалады
нұсқаулықтар мен оқулықтар дамып келеді.

## Бүгін қол жетімді

- **Norito кодектеріне шолу** – `reference/norito-codec.md` беделдіге тікелей сілтеме жасайды
  Портал кестесі толтырылып жатқан кезде `norito.md` сипаттамасы.
- **Torii OpenAPI** – `/reference/torii-openapi` соңғы Torii REST спецификациясын пайдаланады.
  Redoc. `npm run sync-openapi` көмегімен ерекшелікті қалпына келтіріңіз.
- **Конфигурация кестелері** – Толық параметрлер каталогы сақталады
  `docs/source/references/configuration.md`. Портал автоимпортты жібермейінше, оған сілтеме жасаңыз
  Нақты әдепкі мәндерге және ортаны қайта анықтауға арналған Markdown файлы.

## Жақында

- **Torii REST сілтемесі** – OpenAPI анықтамалары осы бөлімге арқылы синхрондалады
  Құбыр қосылғаннан кейін `docs/portal/scripts/sync-openapi.mjs`.
- **CLI пәрмен индексі** – Жасалған пәрмен матрицасы (`crates/iroha_cli/src/commands` шағылыстыру)
  мұнда канондық мысалдармен бірге түседі.
- **IVM ABI кестелері** – Көрсеткіш типі және жүйелік матрицалар (`crates/ivm/docs` астында сақталады)
  құжатты жасау жұмысы қосылғаннан кейін порталға көрсетіледі.

## Бұл индексті ағымдағы сақтау

Жаңа анықтамалық материал қосылғанда — жасалған API құжаттары, кодек сипаттамалары, конфигурация матрицалары—орын
`docs/portal/docs/reference/` астындағы бетті ашып, оны жоғарыда байланыстырыңыз. Егер бет автоматты түрде жасалса, мынаны ескеріңіз
синхрондау сценарийін қосушылар оны қалай жаңарту керектігін біледі. Бұл сілтеме ағашын келесіге дейін пайдалы сақтайды
толығымен автогенерацияланған навигациялық жерлер.
