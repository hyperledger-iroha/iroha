## རིམ་སྒྲིག་འབད་ནི།

སྲིད་བྱུས་འདི་ `torii.da_ingest.replication_policy` གི་འོག་ལུ་སྡོད་ཡོདཔ་དང་ 1 འདི་ གསལ་སྟོན་འབདཝ་ཨིན།
*སྔོན་སྒྲིག་* ཊེམ་པེལེཊི་དང་ དབྱེ་རིམ་རེ་རེ་གི་ཟུར་རིམ་གྱི་ཨེ་རེ་ཅིག། འཛིན་གྲྭ་ངོས་འཛིན་ཚུ་ནི།
གནས་སྟངས་-ཚོར་ཤུགས་དང་ `taikai_segment`, `nexus_lane_sidecar`, དང་ལེན་འབད།
གཞུང་སྐྱོང་གིས་ཆ་འཇོག་འབད་མི་ རྒྱ་བསྐྱེད་ཀྱི་དོན་ལུ་ `governance_artifact`, ཡང་ན་ `custom:<u16>` ཨིན།
གསོག་འཇོག་སློབ་ཚན་ཚུ་གིས་ `hot`, I18NI00000000050X, ཡང་ན་ I18NI0000000051X ངོས་ལེན་འབདཝ་ཨིན།

I18NF0000009X

གོང་ལུ་ཐོ་བཀོད་འབད་ཡོད་པའི་སྔོན་སྒྲིག་ཚུ་དང་གཅིག་ཁར་ ལག་ཆ་མ་ལེན་པར་བཞག་མི་ སྡེབ་ཚན་འདི་བཞག། བསྡམས་པའི་དོན་ལུ།
class, མཐུན་སྒྲིག་བཀག་ཆ་འདི་དུས་མཐུན་བཟོ་ནི།; དབྱེ་རིམ་གསརཔ་གི་དོན་ལུ་གཞི་རྟེན་བསྒྱུར་བཅོས་འབད་ནི་ལུ་;
ཞུན་དག་ `default_retention`.

Taikai འཐོབ་ཚུགས་པའི་སློབ་ཚན་ཚུ་ རང་དབང་ཅན་སྦེ་ བརྒྱུད་དེ་ བཀག་ཚུགས།
I18NI0000003X:

```toml

cold_retention_secs = 15552000     # 180 d
required_replicas = 3
storage_class = "cold"
governance_tag = "da.taikai.archive"
```

  བསྐོར་བའི་སྐབས་ མཐུན་སྒྲིག་མེད་པའི་ འབོད་བརྡ་ཚུ་ ཁ་ཐོག་ལུ་བཟོ་ནི།
