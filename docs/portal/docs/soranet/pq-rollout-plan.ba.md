## Конфигурация һылтанмаһы

### оркестрсы (I18NI000000022X)

```toml

# Optional: force a specific stage independent of the phase
```

### Rust клиент / I18NI000000028X.


Ике `cargo xtask` ярҙамсылары график генерациялау һәм артефакт тотоу автоматлаштырыла.

## Канар планлаштырыу тикшерелгән исемлеге

1. **Префлайт (Т минус 2 аҙна)**

- стадия раҫлау ставкаһы <1% алдағы ике аҙна һәм PQ ҡаплау >=70% бер төбәк (`sorafs_orchestrator_pq_candidate_ratio`).
   - Канар тәҙрәһен раҫлаған идара итеүҙе тикшергән слот планлаштырыу.
   - Яңыртыу `sorafs.gateway.rollout_phase = "ramp"` стажировка (оркестратор JSON һәм үҙгәртеп ҡороу редакторы) һәм ҡоро идара итеү торба үткәргес.

2. **Делай канар (Т көн)**

   - Бер үк ваҡытта бер үк ваҡытта `rollout_phase = "ramp"` оркестрҙа һәм ҡатнашыусы эстафета манифестарын ҡуйып пропагандалау.
   - Монитор "Сәйәсәт ваҡиғалары бер тапҡыр" һәм "Ҡырҡыу ставкаһы" PQ Ratchet приборҙар таҡтаһында (хәҙер унда ролдель панелдәре ролл-аут) ике тапҡыр һаҡсы кэш TTL.
   - Cut I18NI000000082X снимоктар алдынан һәм аудит һаҡлау өсөн эшләгәндән һуң.

3. **Клиент/СДК канары (Т плюс 1 аҙна)**

   - Клиент конфигурацияларында йәки I18NI000000084X тапшырыуында Flip I18NI0000083X билдәләнгән SDK когорталары өсөн өҫтөнлөк бирә.
   - Капиталь телеметрия диффтары (I18NI0000085X `client_id` һәм I18NI000000087X менән төркөмләнгән) һәм уларҙы таратыу инцидент журналына беркетелгән.

4. **Промятка (Т плюс 3 аҙна)**

   - Бер тапҡыр идара итеү ҡул ҡуйыла, оркестр һәм клиент I18NI000000088X тиклем конфигурацияларын һүндерергә һәм ҡул ҡуйылған әҙерлек тикшерелгән исемлеген сығарыу артефакттарына әйләндерергә.

## Rollback ҡармаҡтар

### Ғәҙәттәгесә → пандустар (С этап → В этап)

1. I18NI000000119X менән оркестрҙы төшөрөү (һәм SDK конфигурациялары буйынса бер үк фазаны көҙгө) шуға күрә В этап резюмелары автопарк-киң.
2. Клиенттар көсләп хәүефһеҙ транспорт профиле аша I18NI000000120X, стенограмманы тотоп, шулай итеп, I18NI0000121X remediafication эш ағымы аудит ҡала.
3. Run `cargo xtask soranet-rollout-capture --label rollback-default` архив һаҡсы-каталогы диффтарына, промтуль сығышына һәм `artifacts/soranet_pq_rollout/` буйынса приборҙар таҡтаһы скриншоттарына тиклем.

### рампа → Канар (В этап → А этабы)

1. Импорт һаҡсы-каталог снимок әсирлеккә эләккәнсе I18NI000000124X һәм перерасписание I18NI000000125X шулай итеп, демоция пакеты үҙ эсенә хеш.
2. I18NI0000126X (йәки `anonymity_policy stage-a` менән өҫтөнлөк) оркестр һәм клиент конфигурацияларында, һуңынан [PQ трещотка runbook](./pq-ratchet-runbook.md) PQ трещотка реплейы торба үткәргесенең нисбәтен иҫбатлау өсөн.
3. Яңыртылған PQ Ratchet һәм SN16 телеметрия скриншоттарын беркетергә плюс иҫкәртмә һөҙөмтәләре инцидент журналы идара итеү тураһында хәбәр итеү алдынан.
