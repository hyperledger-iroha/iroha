# SORA Nexus әзірлеуші порталына қош келдіңіз

SORA Nexus әзірлеуші порталы интерактивті құжаттама, SDK жинақтарын береді
оқулықтар және Nexus операторлары мен Hyperledger Iroha үшін API сілтемелері
салымшылар. Ол практикалық нұсқаулықтарды көрсету арқылы негізгі құжаттар сайтын толықтырады
және спецификацияларды тікелей осы репозиторийден жасады. Қондырғыш бет енді тасымалданады
тақырыптық Norito/SoraFS кіру нүктелері, қол қойылған OpenAPI суреттері және арнайы
Norito ағынды сілтемесі, осылайша салымшылар ағынды басқару жазықтығын таба алады.
түбірлік спецификацияны зерттемей-ақ келісімшарт жасаңыз.

## Мұнда не істеуге болады

- **Norito үйреніңіз** – шолудан бастаңыз және түсіну үшін жылдам бастаңыз.
  сериялау үлгісі және байт-код құралдары.
- **Bootstrap SDKs** – бүгін JavaScript және Rust үшін жылдам бастауларды орындаңыз; Python,
  Swift және Android гидтері рецепттер тасымалданған кезде оларға қосылады.
- **API сілтемелерін шолу** – Torii OpenAPI беті соңғы REST нұсқасын көрсетеді
  спецификация және конфигурация кестелері канондық Markdown бағдарламасына қайта сілтеме жасайды
  көздері.
- **Орналастыруларды дайындаңыз** – операциялық жұмыс кітаптары (телеметрия, есеп айырысу, Nexus
  қабаттасуы) `docs/source/` құрылғысынан тасымалдануда және осы сайтта келесідей болады
  көші-қоны жүріп жатыр.

## Ағымдағы күй

- ✅ Жаңартылған типографиясы бар тақырыптық Docusaurus v3 қону, градиентке негізделген
  қаһарман/карталар және Norito Ағынды жиынтықты қамтитын ресурс тақталары.
- ✅ Torii OpenAPI плагині `npm run sync-openapi` желісіне қосылған, қол қойылған суреті бар
  `buildSecurityHeaders` арқылы орындалған тексерулер мен CSP қорғаушылары.
- ✅ CI (`docs-portal-preview.yml` +) жүйесінде қамтуды алдын ала қарау және тексеру
  `scripts/portal-probe.mjs`), қазір ағындық құжатты ашады, SoraFS жылдам іске қосу,
  және артефактілер жарияланғанға дейін анықтамалық тексеру парақтары.
- ✅ Norito, SoraFS және SDK жылдам бастаулары, сонымен қатар анықтамалық бөлімдер мына жерде көрсетіледі.
  бүйірлік тақта; `docs/source/` жаңа импорттары (ағын, оркестрлеу, runbooks)
  олар автор ретінде осында жер.

## Қатысу

- Жергілікті әзірлеу командалары үшін `docs/portal/README.md` қараңыз (`npm install`,
  `npm run start`, `npm run build`).
- Мазмұнды тасымалдау тапсырмалары `DOCS-*` жол картасы элементтерімен қатар бақыланады.
  Жарналар қабылданады — `docs/source/` порт бөлімдері және бетті қосыңыз
  бүйірлік тақтаға.
- Жасалған артефактты (спецификациялар, конфигурация кестелері) қоссаңыз, құрастыруды құжаттаңыз
  пәрмені болашақ салымшылар оны оңай жаңарта алады.
