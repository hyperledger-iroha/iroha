## Նպատակներ

1. Սահմանել համայնքի ընդունման չափանիշները և ստուգման աշխատանքների ընթացքը:
2. Ստացեք կառավարման հաստատում առաջարկվող ցուցակի և ընդունելի օգտագործման հավելվածի համար:
3. Թարմացրեք ստուգիչ գումարով հաստատված նախադիտման արտեֆակտը և հեռաչափության փաթեթը նոր պատուհանի համար:
4. Հրավեր ուղարկելուց առաջ բեմադրեք «Փորձեք այն վստահված անձը» + վահանակները:
