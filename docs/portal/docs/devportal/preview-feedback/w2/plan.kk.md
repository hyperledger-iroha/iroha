## Мақсаттар

1. Қауымдастықты қабылдау критерийлері мен тексеру жұмыс процесін анықтаңыз.
2. Ұсынылған тізімге және қолайлы пайдалану қосымшасына басқарудың мақұлдауын алу.
3. Жаңа терезе үшін бақылау сомасы тексерілген алдын ала қарау артефакті мен телеметрия бумасын жаңартыңыз.
4. Шақыру жіберілмес бұрын проксиді көріңіз + бақылау тақталарын қойыңыз.
