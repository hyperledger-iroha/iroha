## ပေါ်လွင်ချက်များ

1. **အုပ်ချုပ်မှုနှင့် ကိရိယာတန်ဆာပလာ** — 2025-05-20 တွင် ရပ်ရွာစားသုံးမှုမူဝါဒကို တညီတညွတ်တည်း အတည်ပြုခဲ့သည်။ လှုံ့ဆော်မှု/အချိန်ဇုန်အကွက်များ `docs/examples/docs_preview_request_template.md` အောက်တွင် မွမ်းမံထားသော တောင်းဆိုချက်ပုံစံပုံစံ။
2. **Preflight အထောက်အထား** — ၎င်းကိုစမ်းကြည့်ပါ ပရောက်စီပြောင်းလဲမှု `OPS-TRYIT-188` သည် 2025-06-09၊ Grafana ဒက်ရှ်ဘုတ်များကို ဖမ်းယူပြီး၊ နှင့် `preview-2025-06-15` ဖော်ပြချက်/ checksum/probe outputs 009 I အောက်တွင် သိမ်းဆည်းထားသည်။
3. **Invite wave** — ကွန်မြူနတီသုံးသပ်သူ ရှစ်ဦးကို 2025-06-15 တွင် ဖိတ်ကြားထားပြီး tracker ဖိတ်ကြားမှုဇယားတွင် မှတ်တမ်းဝင်ထားသည့် အသိအမှတ်ပြုမှုများ၊ browsing မလုပ်မီ checksum အားလုံးကို ပြီးမြောက်အောင် စစ်ဆေးပါ။
4. **တုံ့ပြန်ချက်** — `docs-preview/w2 #1` (tooltip wording) နှင့် `#2` (localization sidebar order) ကို 2025-06-18 တွင် တင်သွင်းခဲ့ပြီး 2025-06-21 (Docs-core-04/05) တွင် ဖြေရှင်းခဲ့သည်။ လှိုင်းလုံးကြီးအတွင်း အဖြစ်အပျက်များ မရှိပါ။

- နောက်ဆုံး တယ်လီမီတာ ရိုက်ချက်များ (`docs.preview.integrity`၊ `TryItProxyErrors`၊ `DocsPortal/GatewayRefusals`) သည် စိမ်းနေခဲ့သည်။ မှတ်တမ်းများအပြင် `DOCS-SORA-Preview-W2` နှင့် ပူးတွဲပါရှိသော ပရောက်စီမှတ်တမ်းများကို စမ်းကြည့်ပါ။

- Tracker W2 စစ်ဆေးရေးဂိတ်မှတ်တမ်းကို ထွက်ပေါက်မှတစ်ဆင့် အပ်ဒိတ်လုပ်ထားပြီး၊ W3 အစီအစဉ်မစတင်မီ လမ်းပြမြေပုံသည် စာရင်းစစ်မှတ်တမ်းတစ်ခုကို သိမ်းဆည်းထားသည်။
