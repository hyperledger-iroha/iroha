## Зорилтууд

1. Олон нийтийг хүлээн авах шалгуур болон шалгалтын ажлын явцыг тодорхойл.
2. Санал болгож буй нэрсийн жагсаалт болон зөвшөөрөгдөхүйц ашиглах нэмэлтийн засаглалын зөвшөөрлийг авах.
3. Шинэ цонхонд шалгах нийлбэрээр баталгаажсан урьдчилан харах олдвор болон телеметрийн багцыг сэргээнэ үү.
4. Урилгыг илгээхийн өмнө Try it proxy + хяналтын самбарыг байрлуул.
