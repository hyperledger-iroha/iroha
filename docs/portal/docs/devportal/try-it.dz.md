## ས་གནས་ནང་ ངོ་ཚབ་འགོ་བཙུགས།

ཁྱོད་ཀྱིས་ དྲྭ་ཚིགས་འདི་གཞི་སྒྲིག་འབད་བའི་སྐབས་ བརྟེན་པ་གཞི་བཙུགས་འབད་ནི།

```bash
cd docs/portal
npm install
```

