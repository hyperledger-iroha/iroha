# Тайкай якорь күҙәтеүсәнлеге йүнәтмә

Был портал күсермәһе 2012 йылда канонлы runbook-ты көҙгөләй.
[`docs/source/taikai_anchor_monitoring.md`] (https://github.com/hyperledger-iroha/iroha/blob/master/docs/source/taikai_anchor_monitoring.md).
Ҡулланыу, уны репетиция SN13-C маршрутлаштырыу-манифест (ТРМ) якорь шулай I18NT0000000008X/SoraNet .
операторҙар катушка артефакттары, I18NT000000000000000000000-гә бәйле була ала.
дәлилдәр порталы алдан ҡарау төҙөүҙән сыҡмайынса.

## Скап & Хужалар

- **Программа:** SN13-C — Тайкай тайпылыштар & СорНС якорь.
- **Хужалар:** Медиа платформаһында WG, DA программаһы, Селтәрҙәр ТЛ, Док/ДевРел.
- ** Маҡса:** Sev1/Sev2 иҫкәртмәләре өсөн детерминистик пьесалар китабын тәьмин итеү, телеметрия
  раҫлау, һәм дәлилдәрҙе тотоу, ә Тайкай маршрутлаштырыу манифестары алға ролл
  псевдонимдар аша.

## Quickstar (Sev1/Sev2)

1. **Тылсымлы катушка артефакттары** — һуңғыһын күсерергә
   `taikai-anchor-request-*.json`, I18NI000000019X, һәм
   I18NI000000020X файлдары
   `config.da_ingest.manifest_store_dir/taikai/` эшселәрҙе яңынан башлау алдынан.
2. **Думп I18NI0000000022Х телеметрия** — яҙып алыу
   I18NI0000000023X массивын иҫбатлау өсөн ниндәй асыҡ тәҙрә
   иткер:
   ```bash
   curl -sSf "$TORII/status" | jq '.telemetry.taikai_alias_rotations'
   ```
3. **Тикшерергә приборҙар таҡталары һәм иҫкәртмәләр** — йөк
   I18NI000000024X (кластер + ағым фильтрҙары) һәм иҫкәрмә
   ниндәй ҙә булһа ҡағиҙәләр, 2012 йылда.
   I18NI000000025X атыу (I18NI000000026X,
   `TaikaiIngestFailure`, `TaikaiCekRotationLag`, I18NT000000009X һаулыҡ һаҡлау саралары).
4. **Иҫ киткес I18NT0000000001X** — §“Метрик һылтанма” тип раҫлау өсөн һорауҙарҙы раҫлау өсөн
   ингест латентлыҡ/дрейф һәм псевдоним-ротация иҫәпләүселәр үҙҙәрен көткәнсә тота. Эскалит
   әгәр I18NI0000000029X ларектар өсөн бер нисә тәҙрә йәки әгәр
   хаталар иҫәпләүселәре арта.

## метрик белешмә

| Метрика | Маҡсат |
| --- | --- |
| I18NI000000030X | CMAF ингест латентлығы гистограммаһы кластер/ағым (маҡсат: p95<750мс, p99<900мс). |
| `taikai_ingest_live_edge_drift_ms` | Тора-киләһе дрейф араһында кодер һәм якорь эшселәре (биттәр p99>1.5s өсөн 10мин). |
| `taikai_ingest_segment_errors_total{reason}` | Хата ҡаршылары сәбәп буйынса (`decode`, `manifest_mismatch`, I18NI000000035X, ...). Теләһә ниндәй арттырыу триггерҙары I18NI000000036X. |
| `taikai_trm_alias_rotations_total{alias_namespace,alias_name}` | I18NI000000038X инкремменттары псевдоним өсөн яңы ТРМ ҡабул итә; ҡулланыу I18NI000000039X ротация каденцияһын раҫлау өсөн. |
| `/status → telemetry.taikai_alias_rotations[]` | JSON снимок менән I18NI000000041X, `window_end_sequence`, `manifest_digest_hex`, `rotations_total`, һәм ваҡыт маркалары өсөн дәлилдәр өйөмдәре. |
| I18NI000000045X (ребуфер, CEK ротация йәше, PQ һаулыҡ, иҫкәртмәләр) | Тамаша яғында KPIs тәьмин итеү өсөн CEK әйләнеш + PQ схемалары ведущий ваҡытында һау-сәләмәт булып ҡала. |

## Приборҙар таҡталары & иҫкәртмәләр

- **I18NT0000000003X тамашасы таҡтаһы:** I18NI0000000046X — p95/p99
  латентлыҡ, йәшәү рәүеше дрейф, сегмент хаталары, CEK әйләнеш йәше, тамашасы иҫкәртмәләр.
- **I18NT00000000004X кэш таҡтаһы:** I18NI000000047X — эҫе/йылы/һалҡын
  акция һәм QoS инҡар итә, ҡасан псевдоним тәҙрәләре әйләнә.
- **Иҫкәртмәсе ҡағиҙәләре:** I18NI000000048X — дрейф
  пейджинг, ингест етешһеҙлеге тураһында иҫкәртмәләр, CEK әйләнеше лаг, һәм I18NT0000000010X корпус-һаулыҡ
  штрафтар/һыуытҡыстар. Ҡабул итеүселәрҙең һәр етештереү кластеры өсөн барлығын тәьмин итеү.

## Дәлилдәр өйөмө тикшерелгән исемлек

- Спул артефакттар (I18NI000000049X, `taikai-trm-state-*`,
  `taikai-lineage-*`).
- Run I18NI000000052X ҡул ҡуйылған JSON инвентаризацияһын сығарыу өсөн көтөлгән/тапшырылған конверттар һәм күсермәһе запрос/SSM/TRM/нәҫел файлдарын бурау өйөмөнә сығарыу. Ғәҙәттәгесә катушка юлында I18NI0000053X `torii.toml`-тан.
- Grafana снимокты ҡаплаған I18NI0000000056X.
- I18NT000000002X экспорты (JSON/CSV) өсөн өҫтәге метрика өҫтөндә инцидент тәҙрә.
- Grafana скриншоттар менән фильтрҙар күренеп тора.
- Иҫкәртмән идентификаторҙары тейешле ҡағиҙә янғындарына һылтанма яһай.
- Һылтанма I18NI0000000057X һүрәтләнгән
  канонлы дәлилдәр пакеты.

## Приборҙар таҡтаһы көҙгөһө & бурау каденцияһы

SN13-C юл картаһы талаптарын ҡәнәғәтләндереүҙең тигәнде аңлата, тип иҫбатлау, тип тайкай .
тамашасы/кэш приборҙар таҡталары порталы эсендә сағыла ** һәм**, тип якорь
дәлилдәр быраулау күҙаллана торған каденцияла эшләй.

1. **Порталь көҙгө.** Ҡасан I18NI000000058X йәки .
   I18NI000000059X үҙгәрештәр, дельталарҙы дөйөмләштереп 2018 йылда дөйөмләштереп.
   I18NI000000060X (был портал) һәм JSON-ға иғтибар итегеҙ
   порталь пиар тасуирламаһында чемпионерҙар суммаһы. Яңы панелдәрҙе айырып күрһәтегеҙ/сиктәрен шулай
   рецензенттар идара итеү I18NT0000000006X папкаһы менән корреляциялана ала.
2. **Айлы бура.**
   - Һәр айҙағы тәүге шишәмбе көндө 15:00UTC шулай дәлилдәр менән йүгерергә
     SN13 идара итеү синхронизацияһына тиклемге ерҙәр.
   - Ҡаплау катушкаһы артефактары, I18NI000000061X телеметрияһы, һәм I18NT0000000007X скриншоттары эсендә
     `artifacts/sorafs_taikai/drills/<YYYYMMDD>/`.
   - 2012 йылда атҡарыуҙы теркәгеҙ.
     `scripts/telemetry/log_sorafs_drill.sh --scenario taikai-anchor`.
3. **Тикшерергә & баҫтырып сығарыу.** 48hours эсендә, тикшерергә иҫкәртмәләр/ялған ыңғай менән .
   DA программаһы + NetOps, быраулау журналында эҙмә-эҙлекле әйберҙәр яҙып алыу, һәм һылтанма

## Ярҙамлы командалар

```bash
# Snapshot alias rotation telemetry to an artefact directory
curl -sSf "$TORII/status" \
  | jq '{timestamp: now | todate, aliases: .telemetry.taikai_alias_rotations}' \
  > artifacts/taikai/status_snapshots/$(date -u +%Y%m%dT%H%M%SZ).json

# List spool entries for a specific alias/event
find "$MANIFEST_DIR/taikai" -maxdepth 1 -type f -name 'taikai-*.json' | sort

# Inspect TRM mismatch reasons from the spool log
jq '.error_context | select(.reason == "lineage_replay")' \
  "$MANIFEST_DIR/taikai/taikai-ssm-20260405T153000Z.norito"
```
