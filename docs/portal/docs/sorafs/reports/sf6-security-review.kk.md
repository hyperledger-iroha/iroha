# SF-6 қауіпсіздік шолуы

**Бағалау терезесі:** 2026-02-10 → 2026-02-18  
**Шолу жетекшілері:** Қауіпсіздік инженерлік гильдиясы (`@sec-eng`), Құрал-саймандар жөніндегі жұмыс тобы (`@tooling-wg`)  
**Қолдану аясы:** SoraFS CLI/SDK (`sorafs_cli`, `sorafs_car`, `sorafs_manifest`), дәлелді ағынды API интерфейстері, Torii манифестті өңдеу, Sigstore/OIDC интеграциясы, CI босату ілмектері.  
**Артефактілер:**  
- CLI көзі және сынақтары (`crates/sorafs_car/src/bin/sorafs_cli.rs`)  
- Torii манифест/дәлелдеу өңдегіштері (`crates/iroha_torii/src/sorafs/api.rs`)  
- Шығарылымды автоматтандыру (`ci/check_sorafs_cli_release.sh`, `scripts/release_sorafs_cli.sh`)  
- Детерминистік тепе-теңдік жиегі (`crates/sorafs_car/tests/sorafs_cli.rs`, [SoraFS Orchestrator GA Parity Report](./orchestrator-ga-parity.md))

## Әдістеме

1. **Қауіпті модельдеу шеберханалары** әзірлеушінің жұмыс станциялары, CI жүйелері және Torii түйіндері үшін шабуылдаушы мүмкіндіктерін салыстырды.  
2. **Кодты шолу** тіркелгі деректеріне (OIDC таңбалауышпен алмасу, кілтсіз қол қою), Norito манифестті тексеруге және ағынды кері қысымды дәлелдеуге бағытталған.  
3. **Динамикалық тестілеу** қайта ойнатылатын арматура манифесттері мен модельденген сәтсіздік режимдері (таңбалауышты қайта ойнату, манифестті бұрмалау, қысқартылған дәлелдеу ағындары) паритет белгішесін және тапсырыс берілген айқын дискілерді пайдаланады.  
4. **Конфигурацияны тексеру** тексерілген `iroha_config` әдепкі мәндерін, CLI жалауын өңдеуді және детерминирленген, тексерілетін іске қосуларды қамтамасыз ету үшін шығарылым сценарийлерін.  
5. **Процесс сұхбаты** түзету ағынын, күшейту жолдарын және Tooling WG шығарылымының иелерімен аудиторлық дәлелдемелерді алуды растады.

## Бақылауды тексеру

- **Тіркелгі деректерінің ауқымы:** Әдепкі CI үлгілері енді анық аудитория мен эмитент бекітулерін міндеттейді; `--identity-token-audience` `--identity-token-provider` сүйемелдемейінше, CLI және босату көмекшісі екеуі де жылдам істемейді.  
- **Детерминистік қайталау:** Жаңартылған сынақтар оң/теріс манифест жіберу ағындарын қамтиды, сәйкес келмейтін дайджесттер детерминирленген емес ақаулар болып қала береді және желіге тигенге дейін пайда болады.  
- **Дәлелдеу ағынының кері қысымы:** Torii енді PoR/PoTR элементтерін шектелген арналар арқылы ағынмен жібереді және CLI тек қысқартылған кідіріс үлгілерін + бес сәтсіздік үлгісін сақтайды, детерминирленген қорытындыларды сақтай отырып, жазылушының шектеусіз өсуіне жол бермейді.  
- **Байқау мүмкіндігі:** Дәлелдеу ағынының есептегіштері (`torii_sorafs_proof_stream_*`) және CLI қорытындылары операторларға аудит үзінділерімен қамтамасыз ете отырып, тоқтату себептерін жазады.  
- **Құжаттама:** Әзірлеуші ​​нұсқаулығы ([әзірлеуші ​​индексі](../developer-index.md), [CLI анықтамасы](../developer-cli.md)) қауіпсіздікке сезімтал жалаушалар мен эскалация жұмыс процестерін шақырады.

## Бақылау тізімінің қосымшаларын шығарыңыз

Шығарылым менеджерлері GA кандидатын алға жылжытқанда **міндетті түрде келесі дәлелдерді қоса беруі керек:

1. Қауіпсіздікті тексеру туралы соңғы жазбаның хэші (осы құжат).  
2. Бақыланатын түзету билетіне сілтеме (мысалы, `governance/tickets/SF6-SR-2026.md`).  
3. Ашық аудитория/эмитент аргументтерін көрсететін `scripts/release_sorafs_cli.sh --manifest ... --bundle-out ... --signature-out ...` шығысы.  
4. Паритеттік қосқыштан алынған журналдар (`cargo test -p sorafs_car -- --nocapture sorafs_cli::proof_stream::bounded_channels`).  
5. Torii шығарылым ескертпелерінде шектелген ағынды телеметрия есептегіштері бар екенін растау.

Жоғарыдағы артефактілерді жинамау GA кіруді блоктайды.

## Керемет бақылаулар

- **Қауіпті үлгіні жаңарту:** Осы шолуды тоқсан сайын немесе негізгі CLI жалаушасын қосу алдында қайталаңыз.  
- **Бұлыңғыр қамту:** Дәлелді ағынды тасымалдау кодтаулары `fuzz/proof_stream_transport` арқылы сәйкестендіру, gzip, deflate және zstd пайдалы жүктемелерін қамтиды.  
- **Оқиғаға репетиция:** Токендердің компромиссі мен манифестті кері қайтаруды модельдейтін оператор жаттығуын жоспарлаңыз, құжаттамада тәжірибелік процедураларды көрсететініне көз жеткізіңіз.
