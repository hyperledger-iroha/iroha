### དབུགས་འགག་དང་བཙོན་ཁང་གི་ལས་རིམ།མོས་མཐུན་གྱིས་ བདེན་དཔྱད་འབད་མི་ཅིག་གིས་ མཐུན་གྲོས་ལས་འགལ་བའི་སྐབས་ Norito-encoded `Evidence` འདི་ བཏོནམ་ཨིན། དྲན་ཚད་ནང་ `EvidenceStore` ནང་ པེ་ལོཌ་གི་ས་ཆ་རེ་རེ་དང་ གལ་སྲིད་ མ་མཐོང་པ་ཅིན་ ཌབ་ལུ་ཨེསི་ཝི་-རྒྱབ་སྐྱོར་ཡོད་པའི་ `consensus_evidence` སབ་ཁྲ་ནང་ དངོས་གཞི་བཟོཝ་ཨིན། `sumeragi.npos.reconfig.evidence_horizon_blocks` ལས་རྙིངམ་ཐོ་བཀོད་ཚུ་ (སྔོན་སྒྲིག་ `7 200` སྡེབ་ཚན་ཚུ་) ཆ་མེད་བཏང་ཡོདཔ་ལས་ གཏན་མཛོད་འདི་ མཚམས་འཇོག་འབད་དེ་ཡོད་རུང་ ངོས་ལེན་འདི་ བཀོལ་སྤྱོད་པ་ཚུ་གི་དོན་ལུ་ ནང་བསྐྱོད་འབད་ཡོདཔ་ཨིན། གནམ་སྟོང་ནང་འཁོད་ཀྱི་སྒྲུབ་བྱེད་ཚུ་གིས་ མཉམ་འབྲེལ་མཐུན་གྲོས་སྒྲིག་གཞི་ (`mode_activation_height requires next_mode to be set in the same block`) དང་ ཤུགས་ལྡན་བཟོ་ནི་གི་ཕྱིར་འགྱངས་ (`sumeragi.npos.reconfig.activation_lag_blocks`) དེ་ལས་ སྔོན་སྒྲིག་ `1`) དང་ བརྡབ་འགྱོ་ནི་ཕྱིར་འགྱངས་ (`sumeragi.npos.reconfig.slashing_delay_blocks`, སྔོན་སྒྲིག་ `259200`) སྔོན་སྒྲིག་ `259200`) དེ་གི་ཧེ་མ་ ཉེས་ཆད་བཀལ་ཚུགས། བཙུགས་ནི།

ངོས་འཛིན་ཅན་གྱི་ཉེས་འཛུགས་ཚུ་གིས་ གཅིག་ལས་གཅིག་ལུ་ `EvidenceKind`; ཉེ་རིང་ཕྱེ་མི་ཚུ་ གནས་སྡུད་དཔེ་ཚད་ཀྱིས་ བརྟན་ཏོག་ཏོ་དང་ བསྟར་སྤྱོད་འབདཝ་ཨིན།

```rust
use iroha_data_model::block::consensus::EvidenceKind;

let offences = [
    EvidenceKind::DoublePrepare,
    EvidenceKind::DoubleCommit,
    EvidenceKind::InvalidQc,
    EvidenceKind::InvalidProposal,
    EvidenceKind::Censorship,
];

for (expected, kind) in offences.iter().enumerate() {
    assert_eq!(*kind as u16, expected as u16);
}
```

- **ཌུ་བཱལ་པྲེ་པར་/ཌུ་བཱལ་ཁཱམ་མིཊ་** — བདེན་དཔྱད་འབད་མི་གིས་ མིང་རྟགས་བཀོད་མི་ འགལ་བ་ཡོད་པའི་ ཧ་ཤེ་ཚུ་གིས་ `(phase,height,view,epoch)` གི་ ཊུཔ་ལཱ།
- **InvalidQc** — མཉམ་བསྡོམས་འབད་མི་ཅིག་གིས་ ཁས་བླངས་ལག་ཁྱེར་ཅིག་ དབྱིབས་བཟོ་མ་ཚུགས་པའི་ བརྟག་ཞིབ་ཚུ་ འཐུས་ཤོར་བྱུང་མི་ (དཔེར་ན་ ནང་བསྐྱོད་འབད་མི་ བིཊི་མེཔ་)།
- **Invalid Proposal** — འགོ་ཁྲིདཔ་ཅིག་གིས་ བཟོ་བཀོད་བདེན་དཔྱད་འབད་མ་ཚུགས་པའི་ བཀག་ཆ་ཅིག་ གྲོས་འཆར་བཀོད་ཡོདཔ་ད་ (དཔེར་ན་ བསྡམ་བཞག་ཡོད་པའི་རྒྱུན་རིམ་གྱི་ལམ་ལུགས་འདི་ བརྡལ་བཤིག་གཏངམ་ཨིན།)
- **Sensorship** — མཚན་རྟགས་བཀོད་མི་ ཞུ་ཡིག་ཐོབ་མི་ཚུ་གིས་ གྲོས་འཆར་མེད་པའི་ ཚོང་འབྲེལ་ཅིག་ སྟོནམ་ཨིན།

ཝི་ཨར་ཨེཕ་ཉེས་ཆད་ཚུ་ `activation_lag_blocks` གི་ཤུལ་ལས་ རང་བཞིན་གྱིས་ བསྟར་སྤྱོད་འབདཝ་ཨིན། གཞུང་སྐྱོང་གིས་ ཉེས་ཆད་འདི་ ཆ་མེད་མ་བཏང་ཚུན་ཚོད་ མོས་མཐུན་གྱི་ བརྡབ་གསིག་འདི་ `slashing_delay_blocks` སྒོ་སྒྲིག་གི་ཤུལ་ལས་རྐྱངམ་ཅིག་ ལག་ལེན་འཐབ་ཨིན།

བཀོལ་སྤྱོད་པ་དང་ ལག་ཆས་ཚུ་གིས་ བརྟག་ཞིབ་དང་ ལོག་སྟེ་ རྒྱང་བསྒྲགས་འབད་ཚུགས།

༡ **ལོ་ན་མ་འགྱོ་བའི་ཧེ་མ་ འབབ་ཁུངས་** བསྡུ་ལེན་འབད། མཐོ་ཚད་/མཐོང་སྣང་མེ་ཊ་ཌེ་ཊ་དང་གཅིག་ཁར་ Norito བཱའིཊིསི་འདི་ དཔར་བསྐྲུན་འབད།
2. **དགོས་ངེས་ཡོད་ན་** `CancelConsensusEvidencePenalty` འདི་ `slashing_delay_blocks` མ་འགྱོ་བའི་ཧེ་མ་ སྒྲུབ་བྱེད་ཀྱི་ འབབ་ཁུངས་ཚུ་དང་གཅིག་ཁར་ ཕུལ་དགོ། ཐོ་བཀོད་འདི་ `penalty_cancelled` དང་ `penalty_cancelled_at_height` རྟགས་བཀོད་དེ་ཡོདཔ་ལས་ བརྡབ་འགྱོ་མི་འདི་ འཇུག་སྤྱོད་འབདཝ་ཨིན།
༣ **ཉེས་ཆད་འདི་ འོས་འདེམས་ཡང་ན་ སུ་ཌོ་བཀོད་རྒྱ་ནང་ བཀལ་ཐོག་ལས་ ཉེས་ཆད་བཀལ་དགོཔ་ཨིན། (དཔེར་ན་ `Unregister::peer`) ལག་ལེན་འཐབ་མི་འདི་གིས་ པེ་ལོཌ་འདི་ ལོག་བདེན་དཔྱད་འབདཝ་ཨིན། ནོར་འཁྲུལ་ཅན་གྱི་སྒྲུབ་བྱེད་དང་ སྒྲུབ་བྱེད་ཚུ་ ངོས་ལེན་མ་འབད་བར་ ངོས་ལེན་འབདཝ་ཨིན།
༤. ** རྗེས་འཇུག་ཊོ་པོ་ལོ་ཇི་** འདི་ རིམ་སྒྲིག་འབདཝ་ལས་ ཉེས་ཅན་གྱི་བདེན་དཔྱད་པ་དེ་ དེ་འཕྲོ་ལས་ ལོག་སྟེ་འཛུལ་མི་ཚུགས། སྤྱིར་བཏང་གི་བཞུར་ཚད་གྱལ་རིམ་ `SetParameter(Sumeragi::NextMode)` དང་ `SetParameter(Sumeragi::ModeActivationHeight)` གིས་ དུས་མཐུན་བཟོ་ཡོད་པའི་ཐོ་ཡིག་དང་གཅིག་ཁར་ཨིན།

### མཉམ་རུབ་གྲོས་འཆར།

མཉམ་འབྲེལ་གྱི་མོས་མཐུན་འདི་གིས་ ཕྱིར་འཐོན་བདེན་དཔྱད་པ་དེ་གིས་ ཆ་ཚན་གསརཔ་འདི་ གྲོས་འཆར་འགོ་མ་བཙུགས་པའི་ཧེ་མ་ ས་མཚམས་སྡེབ་ཚན་འདི་ མཐའ་དཔྱད་འབདཝ་ཨིནམ་སྦེ་ ངེས་གཏན་བཟོཝ་ཨིན། རན་ཊའིམ་གྱིས་ ཆ་སྒྲིག་འབད་ཡོད་པའི་ཚད་བཟུང་ཚུ་བརྒྱུད་དེ་ ལམ་ལུགས་འདི་ བསྟར་སྤྱོད་འབདཝ་ཨིན།- `SumeragiParameter::NextMode` དང་ `SumeragiParameter::ModeActivationHeight` འདི་ **same block* ནང་ལུ་ ཁས་བླངས་འབད་དགོཔ་ཨིན། `mode_activation_height` འདི་ དུས་མཐུན་བཟོ་མི་ སྡེབ་ཚན་མཐོ་ཚད་ལས་ དམ་དམ་སྦེ་ མཐོ་དྲགས་དགོཔ་ཨིན་ དེ་གིས་ ཉུང་མཐའ་གཅིག་གི་ བཀག་ཆ་ཅན་གྱི་ ལག་ཆ་བྱིནམ་ཨིན།
- `sumeragi.npos.reconfig.activation_lag_blocks` (སྔོན་སྒྲིག་ `1`) འདི་ ཀླད་ཀོར་ལྷག་པའི་ལག་ཐོག་བཀག་བཞག་མི་ རིམ་སྒྲིག་སྲུང་སྐྱོབ་འདི་ཨིན།
- `sumeragi.npos.reconfig.slashing_delay_blocks` (སྔོན་སྒྲིག་ `259200`) ཕྱིར་འགྱངས་ཀྱི་ཕྱིར་འགྱངས་འབདཝ་ལས་ གཞུང་སྐྱོང་གིས་ ཞུ་ཡིག་མ་བཙུགས་པའི་ཧེ་མ་ ཉེས་ཆད་ཚུ་ ཆ་མེད་བཏང་ཚུགས།
