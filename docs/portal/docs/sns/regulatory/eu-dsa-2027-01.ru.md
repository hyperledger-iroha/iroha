## 1. Резюме intake

- **Бюллетень:** Еврокомиссия “Trusted flaggers & rapid takedown obligations for DNS/hosting providers” (Ref. DSA/2026/1219).
- **Ключевые требования:** опубликовать workflow onboarding trusted flagger, гарантировать <4 ч время реакции для высоких уровней severity и публиковать контакты abuse по юрисдикциям для всех публичных суффиксов.
- **Дата подачи:** 2026‑12‑20.
- **Срок:** 2027‑01‑31 (декларация соответствия для DG CONNECT).

Согласно SNS charter Section 11, находка внесена в governance‑трекер 2026‑12‑20 (тикет `SNS-REG-2026-48`).
