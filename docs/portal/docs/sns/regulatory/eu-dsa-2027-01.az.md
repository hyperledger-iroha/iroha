## 1. Qəbul Xülasəsi

- **Bülleten:** Avropa Komissiyası “DNS/hostinq provayderləri üçün etibarlı bayraqçılar və sürətli ləğv öhdəlikləri” (İst. DSA/2026/1219).
- **Əsas tələblər:** Etibarlı bayraqçı işə qəbul iş prosesini dərc edin, yüksək ciddilik bildirişləri üçün <4 saat cavab vaxtına zəmanət verin və bütün ictimai şəkilçilər arasında yurisdiksiyaya aid sui-istifadə kontaktlarını ifşa edin.
- **Təqdim etmə tarixi:** 2026-12-20.
- **Son tarix:** 2027‑01‑31 (DG CONNECT-ə görə uyğunluq bəyannaməsi).

SNS nizamnaməsinin 11-ci bölməsinə uyğun olaraq, tapıntı 2026‑12‑20 (bilet `SNS-REG-2026-48`) tarixində idarəetmə izləyicisinə daxil oldu.
