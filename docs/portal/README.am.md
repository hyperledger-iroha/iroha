### ተኪ ውቅር ይሞክሩት።

የ"ሞክሩት" ማጠሪያ መንገዶች በ`scripts/tryit-proxy.mjs` በኩል ይጠይቃሉ። አዋቅር
ከመጀመሩ በፊት ፕሮክሲው ከአካባቢ ተለዋዋጮች ጋር፡-

```bash
export TRYIT_PROXY_TARGET="https://torii.staging.sora"
export TRYIT_PROXY_ALLOWED_ORIGINS="http://localhost:3000"
export TRYIT_PROXY_BEARER="sora-dev-token"          # optional
export TRYIT_PROXY_PUBLIC_URL="http://localhost:8787"
npm run tryit-proxy
```

- `TRYIT_PROXY_LISTEN` (ነባሪ I18NI0000094X) የማሰሪያ አድራሻውን ይቆጣጠራል።
- `TRYIT_PROXY_RATE_LIMIT` / `TRYIT_PROXY_RATE_WINDOW_MS` ውስጠ-ማህደረ ትውስታን ያዋቅራል
  ተመን ገዳይ (ነባሪ ወደ 60 ጥያቄዎች በ60 ሰከንድ)።
- የደዋዩን `Authorization` ለማስተላለፍ `TRYIT_PROXY_ALLOW_CLIENT_AUTH=1` ያዘጋጁ
  በነባሪው ተሸካሚ ቶከን ላይ ከመታመን ይልቅ ራስጌ።
- `static/openapi/torii.json` ከተንሳፈፈ ፕሮክሲው ለመጀመር ፈቃደኛ አይሆንም
  በ `static/openapi/manifest.json` ውስጥ የተፈረመ አንጸባራቂ። ሩጡ
  ዝርዝሩን ለማደስ `npm run sync-openapi -- --latest`; ወደ ውጭ መላክ
  `TRYIT_PROXY_ALLOW_STALE_SPEC=1` ለአደጋ ጊዜ መሻር ብቻ (ማስጠንቀቂያ ነው።
  ገብቷል እና ተኪው ለማንኛውም ይጀምራል)።
- ለማጨስ `npm run probe:tryit-proxy` ን ያሂዱ - የዝግጅት ፕሮክሲውን ይሞክሩ እና ሽቦውን ያሽጉ
  የክትትል ስራዎችን ማዘዝ; `npm run manage:tryit-proxy -- update` ያቃልላል
  የ Torii የመጨረሻ ነጥቦችን ማሽከርከር የ`.env` ምትኬን ለመልሰህ ጠብቅ።
- `TRYIT_PROXY_PROBE_METRICS_FILE` እና `TRYIT_PROXY_PROBE_LABELS` ፍተሻውን ይፍቀዱለት
  `probe_success`/`probe_duration_seconds` በ I18NT00000000 የጽሑፍ ፋይል ቅርጸት
  የጤና ፍተሻውን ወደ node_exporter ወይም ማንኛውም ባች ሰብሳቢ ማያያዝ ይችላሉ።
- Prometheus ቧጨራዎች `tryit_proxy_request_duration_ms_bucket`/`_count`/`_sum` ያጋልጣሉ
  ሂስቶግራም ተከታታይ; የሰነዶች ፖርታል I18NT0000015X ዳሽቦርድ ለp95/p99 መዘግየት ይበላቸዋል።
  SLO መከታተያ (`dashboards/grafana/docs_portal.json`)።

መመሪያውን ያከብራል `X-TryIt-Auth` ይሽራል፣ ስለዚህ የOAuth ተለዋዋጮችን ማስወገድ ይችላሉ።
የማስታወቂያ ቶከኖችን ለመለጠፍ በመረጡ ቁጥር።

### የደህንነት ራስጌዎች

`docusaurus.config.js` አሁን ቆራጥ የደህንነት ራስጌዎችን ያወጣል-የይዘት ደህንነት
ፖሊሲ፣ የታመኑ ዓይነቶች፣ ፈቃዶች-መመሪያ፣ እና አጣቃሽ-ፖሊሲ—ስለዚህ የማይለዋወጥ አስተናጋጆች
ከዴቭ አገልጋዩ ጋር ተመሳሳይ የጥበቃ መስመሮችን ይውረሱ። ነባሪዎቹ ስክሪፕቶችን ብቻ ይፈቅዳሉ
ከፖርታል መነሻው አገልግሏል እና መነሻውን `connect-src` ትራፊክን ይገድባል
የተዋቀረው የትንታኔ የመጨረሻ ነጥብ እና ፕሮክሲ ይሞክሩት። የምርት ግንባታዎች ውድቅ ናቸው
የኤችቲቲፒኤስ ያልሆኑ ትንታኔዎች/የመጨረሻ ነጥቦችን ይሞክሩ; ለአካባቢያዊ ቅድመ-እይታዎች ከ `http://`
ዒላማዎች፣ `DOCS_SECURITY_ALLOW_INSECURE=1` በግልፅ አስቀምጠው የጥበቃ መንገዱ
ማሽቆልቆሉን አምኗል።

### አካባቢያዊነት የስራ ፍሰት

መግቢያው እንግሊዝኛ እንደ ምንጭ ቋንቋ እና ጃፓንኛ፣ ዕብራይስጥ፣
