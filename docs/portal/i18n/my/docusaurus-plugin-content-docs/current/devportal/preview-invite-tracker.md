ဤခြေရာခံကိရိယာသည် docs portal အစမ်းကြည့်လှိုင်းတိုင်းကို မှတ်တမ်းတင်ထားသောကြောင့် DOCS-SORA ပိုင်ရှင်များနှင့်
အုပ်ချုပ်မှု ပြန်လည်သုံးသပ်သူများသည် မည်သည့်အဖွဲ့ကို တက်ကြွစွာ လုပ်ဆောင်နေကြောင်း၊ ဖိတ်ကြားချက်များကို အတည်ပြုခဲ့သူ၊
မည်သည့်အရာများ သည် သတိထားရန် လိုအပ်ပါသနည်း။ ဖိတ်ကြားချက်များကို ပေးပို့သည့်အခါတိုင်း ၎င်းကို အပ်ဒိတ်လုပ်ပါ။
ရုတ်သိမ်းသည် သို့မဟုတ် ရွှေ့ဆိုင်းထားသောကြောင့် စာရင်းစစ်လမ်းကြောင်းသည် သိုလှောင်ခန်းအတွင်း၌ ရှိနေမည်ဖြစ်သည်။
