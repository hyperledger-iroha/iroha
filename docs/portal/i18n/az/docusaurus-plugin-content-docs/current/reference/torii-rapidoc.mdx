# Torii API (RapiDoc)

`RapiDoc` eyni Torii-in yığcam, sənədləşdirmə-ilk renderini təklif edir
OpenAPI sənədi. Daha sıx icmal və ya yalnız oxumaq üçün nəzərdən keçirməyə ehtiyacınız olduqda istifadə edin
sorğu/cavab modelləri.

## İstifadə məsləhətləri

- İnteraktiv zəngləri aktivləşdirmək üçün daşıyıcı nişanı daxil edin. Dəyər bir kimi ötürülür
  `X-TryIt-Auth` başlığı və proxy tərəfindən normallaşdırılıb.
- Portala interaktiv xidmət göstərərkən `TRYIT_PROXY_PUBLIC_URL` konfiqurasiya edin
  sorğular staging proxy vasitəsilə yönləndirilir. Widget proksi olduqda xəbərdarlıq edir
  URL yoxdur.
- tərəfindən dərc edilmiş tarixi xüsusiyyətləri yükləmək üçün "OpenAPI snapshot" açılan menyusundan istifadə edin
  `npm run sync-openapi`; `Latest (tracked)` seçimi həmişə həll edir
  `/openapi/torii.json`.
- “[Try it](/api/overview#try-it-sandbox-docs-3)” konsolu ilə birləşdirin
  proxy tələb edən mərhələli şlüzlərə qarşı işləyir.
