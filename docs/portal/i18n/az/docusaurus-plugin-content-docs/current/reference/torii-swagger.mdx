# Torii API (Swagger UI)

Bu səhifə `swagger-ui-react` istifadə edərək Torii OpenAPI sənədini daxil edir. Zaman istifadə edin
tanış Swagger layout, sorğu nümunələri və cəhd etmək imkanı istəyirsiniz
sorğuları birbaşa brauzer daxilində həyata keçirin.

## İstifadə məsləhətləri

- Hər sorğuya əvvəlcədən icazə vermək üçün paneldə daşıyıcı nişanı təqdim edin. Tokenlərdir
  `X-TryIt-Auth` başlığından istifadə edərək proksiyə yönləndirilir və heç vaxt saxlanılmır.
- Portala xidmət edərkən `TRYIT_PROXY_PUBLIC_URL`-i konfiqurasiya edin ki, widget olsun
  istəkləri yığılmış proksi vasitəsilə yönləndirir. Xəbərdarlıq banneri görünür
  proksi söndürülübsə.
- Ən son xüsusiyyətlər arasında keçid etmək üçün açılan menyudan OpenAPI şəklini seçin
  və `npm run sync-openapi` tərəfindən yaradılan arxivləşdirilmiş versiyalar.
- Proksi ilə dəstəklənən “[Try it](/api/overview#try-it-sandbox-docs-3)” konsolunu sınayın
  CORS-in birbaşa Torii girişinə mane olduğu mərhələli mühitlər üçün.
- OpenAPI şəklini `cargo xtask openapi` ilə təzə saxlayın (bax.
  `docs/portal/static/openapi/torii.json`).
