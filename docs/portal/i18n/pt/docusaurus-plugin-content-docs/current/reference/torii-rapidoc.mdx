# API Torii (RapiDoc)

`RapiDoc` oferece uma renderização compacta e com documentação inicial do mesmo Torii
Documento OpenAPI. Use-o quando precisar de uma visão geral mais densa ou de uma revisão somente leitura de
modelos de solicitação/resposta.

## Dicas de uso

- Insira um token de portador para ativar chamadas interativas. O valor é encaminhado como um
  Cabeçalho `X-TryIt-Auth` e normalizado pelo proxy.
- Configurar `TRYIT_PROXY_PUBLIC_URL` ao atender o portal de forma interativa
  as solicitações são roteadas por meio do proxy de teste. O widget avisa quando o proxy
  O URL está faltando.
- Use o menu suspenso “OpenAPI snapshot” para carregar especificações históricas publicadas por
  `npm run sync-openapi`; a opção `Latest (tracked)` sempre resolve para
  `/openapi/torii.json`.
- Combine com o console “[Try it](/api/overview#try-it-sandbox-docs-3)” quando
  trabalhando em gateways preparados que exigem o proxy.
