Nexus (Iroha 3) Iroha 2 はマルチレーンをサポートし、マルチレーンをサポートします。ガバナンスと SDK の統合。 Эта страница отражает новый обзор `docs/source/nexus_overview.md` в монорепозитории, чтобы читатели портала быстро поняли, как Ѝлементы архитектуры сочетаются。

## Линейки релизов

- **Iroha 2** - 問題が解決されました。
- **Iroha 3 / Sora Nexus** - 複数のレーンを通過する必要があります。ガバナンス、決済、観察可能性を備えています。
- ワークスペース (IVM + ツールチェーン Kotodama)、SDK、ABI のサポートфикстуры Norito остаются переносимыми. Операторы скачивают пакет `iroha3-<version>-<os>.tar.zst`, чтобы присоединиться к Nexus; обратитесь к `docs/source/sora_nexus_operator_onboarding.md` за полноэкранным чек-листом.

| N1 - Публичный запуск | Добавляет суффиксы `.nexus`、аукционы、セルフサービス レジストラ、XOR 決済。 |リゾルバー/ゲートウェイを使用すると、問題が解決されます。 |
| N2 - Расвирение | Вводит `.dao`、リセラー API、аналитику、портал споров、スコアカードとスチュワード。 |コンプライアンス ツール、オンライン ポリシー陪審ツールキット、オンライン ポリシー ツールキットを備えています。 |

## Ответственность операторов1. **Гигиена конфигурации** - держите `config/config.toml` синхронизированным с опубликованным каталогом レーンとデータスペース。 `--trace-config` が表示されます。
2. **Отслеживание манифестов** - сверяйте записи каталога с последним пакетом Space Directory перед вступлением или обновлениемうそ。
3. **Покрытие телеметрией** - `nexus_lanes.json`、`nexus_settlement.json` および SDK のバージョン。 PagerDuty と修復を実行します。
4. **Отчетность об инцидентах** - [Nexus 操作](./nexus-operations) および RCA の操作течение пяти рабочих дней.
5. **ガバナンス** - участвуйте в голосованиях совета Nexus, влияющих на вази lan, и раз в кварталロールバック (отслеживается через `docs/source/project_tracker/nexus_config_deltas/`)。
