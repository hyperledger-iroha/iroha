## レポートのフック- 報告書は、実際の活動状況を追跡し、問題の報告書を作成します (招待活動、活動の見直し、事件)。
- Cuando una ola cierre、フィードバックの再開 (por ejemplo、`docs/portal/docs/devportal/preview-feedback/w0/summary.md`) および enlazala desde `status.md`。
- [プレビュー招待フロー](./preview-invite-flow.md) の一時停止基準をアクティブにし、招待状を回復するための措置を講じます。
