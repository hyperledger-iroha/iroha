# Журнал миграции SoraFS

Этот журнал отражает лог изменений миграции, зафиксированный в RFC архитектуры
SoraFS. Записи сгруппированы по вехам и показывают окно действия, затронутые команды
и требуемые действия. Обновления плана миграции ДОЛЖНЫ менять эту страницу ו-RFC
(`docs/source/sorafs_architecture_rfc.md`), чтобы держать downstream-потребителей
в согласовании.

| Веха | Окно действия | Сводка изменений | Затронутые команды | Действия | Статус |
|------|--------------|----------------|--------------------|--------|--------|
| M1 | Недели 7–12 | גופי CI принуждает детерминированные; כינוי הוכחות доступны в בימוי; כלים показывает явные דגלי ציפיות. | מסמכים, אחסון, ממשל | Убедиться, что fixtures остаются подписанными, зарегистрировать aliases в staging registry, обновить checklists releases с требованием I100000X. | ⏳ Ожидается |

Протоколы контрольного плана ממשל, ссылающиеся на эти вехи, находятся в
`docs/source/sorafs/`. Команды должны добавлять датированные пункты под каждой строкой
при возникновении заметных событий (например, новые регистрации כינוי, ретроспективы
инцидентов registry), чтобы предоставить аудируемый след.
