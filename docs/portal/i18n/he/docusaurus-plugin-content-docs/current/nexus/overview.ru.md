Nexus (Iroha 3) расширяет Iroha 2 счет multi-lane исполнения, пространств даннных полних общих инструментов для каждого SDK. Эта страница отражает новый обзор `docs/source/nexus_overview.md` в монорепозитории, чтобы читатели портала быстро полкали, архитектуры сочетаются.

## Линейки релизов

- **Iroha 2** - самостоятельные развертывания для консорциумов или частных сетей.
- **Iroha 3 / Sora Nexus** - публичная multi-lane сеть, где операторы регистрируют пространства данних (тоюнства данних общие инструменты ממשל, התיישבות и צפייה.
- Обе линейки собираются из одного סביבת עבודה (IVM + כלי כלים Kotodama), поэтому исправления SDK, обикновия Norito остаются переносимыми. Операторы скачивают пакет `iroha3-<version>-<os>.tar.zst`, чтобы присоединиться к Nexus; обратитесь к `docs/source/sora_nexus_operator_onboarding.md` за полноэкранным чек-листом.

|-------|-------|-------------|
| N0 - Закрытая бета | הרשם под управлением совета (`.sora`), ручной onboarding операторов, статический каталог ליין. | Подписанные DS манифесты + отрепетированные передачи ממשל. |

| N2 - Расширение | Вводит `.dao`, ממשק API, аналитику, PORTAL споров, כרטיסי ניקוד לדיילים. | אמנות תאימות כלליות, ערכת כלים מקוונת של חבר מושבעים למדיניות, отчеты прозрачности казначейства. |

## Ответственность операторов

1. **Гигиена конфигурации** - держите `config/config.toml` синхронизированным с опубликованным каталогом ליין ומרחבי נתונים; архивируйте вывод `--trace-config` с каждым релизным тикетом.
2. **Отслеживание манифестов** - сверяйте записи каталога с последним пакетом Space Directory перед вступлением илиобниов.
3. **Покрытие телеметрией** - публикуйте `nexus_lanes.json`, `nexus_settlement.json` и связанные SDK дашборды; התקן את התיקון של PagerDuty ו-Prоводите квартальные обзоры по плану שיקום.
4. **Отчетность об инцидентах** - следуйте матрице серьезности в [Nexus פעולות](./nexus-operations) иветиветече пяти рабочих дней.
5. **Готовность к governance** - участвуйте в голосованиях совета Nexus, влияющих на ваши ליין, и развет варе инструкции החזרה לאחור (отслеживается через `docs/source/project_tracker/nexus_config_deltas/`).
