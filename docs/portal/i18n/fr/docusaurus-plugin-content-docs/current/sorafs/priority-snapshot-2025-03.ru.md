## Обзор

L'instantané de Martov est disponible pour les initiatives docs/content-network dans les logiciels
доставочными треками SoraFS (SF-3, SF-6b, SF-9). Как только все mène подтвердят
instantané dans le canal de direction Nexus, sélectionnez « Beta » pour vous.

### Thèmes ciblés

1. **Assurer les priorités des instantanés** — enregistrer les accusés de réception et
   зафиксировать их в minutes совета от 2025-03-05.
2. **Ouvrir la passerelle de lancement/DNS** — utiliser un nouveau kit de facilitation (page 6
   dans le runbook) du 03/03/2025.
3. **Les runbooks des opérateurs** — le portail `Runbook Index` est en direct ; ouvrir la version bêta
   URL d'aperçu après l'intégration de la signature pour les évaluateurs.
4. **Треки доставки SoraFS** — согласовать оставшуюся работу SF-3/6b/9 avec plan/feuille de route :
   - PoR d'ingestion de travailleur + point de terminaison d'état dans `sorafs-node`.
   - Synchronisation des liaisons CLI/SDK dans l'orchestrateur intégré Rust/JS/Swift.
   - Temps d'exécution du câblage pour le coordination de PoR et pour GovernanceLog.

См. Il s'agit d'un fichier pour les tableaux généraux, la liste de contrôle de distribution et les entrées de journal.
