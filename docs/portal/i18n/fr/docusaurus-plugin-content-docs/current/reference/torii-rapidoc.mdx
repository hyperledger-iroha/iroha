# API Torii (RapiDoc)

`RapiDoc` offre un rendu compact et axé sur la documentation du même Torii.
Document OpenAPI. Utilisez-le lorsque vous avez besoin d'un aperçu plus dense ou d'un examen en lecture seule de
modèles de requête/réponse.

## Conseils d'utilisation

- Entrez un jeton au porteur pour activer les appels interactifs. La valeur est transmise sous forme de
  En-tête `X-TryIt-Auth` et normalisé par le proxy.
- Configurer `TRYIT_PROXY_PUBLIC_URL` lors de la desserte du portail de manière interactive
  les demandes sont acheminées via le proxy intermédiaire. Le widget avertit lorsque le proxy
  L'URL est manquante.
- Utilisez la liste déroulante « Instantané OpenAPI » pour charger les spécifications historiques publiées par
  `npm run sync-openapi` ; l'option `Latest (tracked)` résout toujours à
  `/openapi/torii.json`.
- Combinez avec la console « [Try it](/api/overview#try-it-sandbox-docs-3) » lorsque
  travailler contre des passerelles par étapes nécessitant le proxy.
