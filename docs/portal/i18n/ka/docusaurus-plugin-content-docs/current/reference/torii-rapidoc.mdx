# Torii API (RapiDoc)

`RapiDoc` გთავაზობთ კომპაქტურ, დოკუმენტაციას, იგივე Torii-ის პირველ რენდერს
OpenAPI დოკუმენტი. გამოიყენეთ ის, როდესაც გჭირდებათ უფრო მჭიდრო მიმოხილვა ან მხოლოდ წაკითხვადი მიმოხილვა
მოთხოვნის/პასუხის მოდელები.

## გამოყენების რჩევები

- შეიყვანეთ გადამტანი ჟეტონი ინტერაქტიული ზარების გასააქტიურებლად. ღირებულება გადაგზავნილია როგორც
  `X-TryIt-Auth` სათაური და ნორმალიზებულია პროქსის მიერ.
- დააკონფიგურირეთ `TRYIT_PROXY_PUBLIC_URL`, როდესაც პორტალს ასე ინტერაქტიულად ემსახურება
  მოთხოვნების მარშრუტი ხდება დადგმის პროქსის მეშვეობით. ვიჯეტი აფრთხილებს პროქსის დროს
  URL აკლია.
- გამოიყენეთ ჩამოსაშლელი მენიუ „OpenAPI Snapshot“ ისტორიული სპეციფიკაციების ჩასატვირთად, რომელიც გამოქვეყნებულია
  `npm run sync-openapi`; `Latest (tracked)` ვარიანტი ყოველთვის გადაწყვეტს
  `/openapi/torii.json`.
- შეუთავსეთ „[სცადეთ](/api/overview#try-it-sandbox-docs-3)“ კონსოლთან, როდესაც
  მუშაობს ეტაპობრივი კარიბჭეების წინააღმდეგ, რომლებიც საჭიროებენ პროქსის.
