- پیش نظارہ ٹیگ `preview-2025-03-24` کے ساتھ [`docs/examples/docs_preview_invite_template.md`] (../../../examples/docs_preview_invite_template.md) کا استعمال کرتے ہوئے آؤٹ ریچ ٹیکسٹ پھنس گیا۔
- پہلے پانچ دیکھ بھال کرنے والوں (ٹکٹ `DOCS-SORA-Preview-REQ-01` ... `-05`) کے لئے رجسٹرڈ ان باؤنڈ درخواستیں۔
-پہلے پانچ دعوت نامے 2025-03-25 10: 00-10: 20 UTC کو لگاتار سات دن گرین ٹیلی میٹری کے بعد بھیجے گئے۔ `DOCS-SORA-Preview-W0` میں چارجز بچ گئے۔
-ٹیلی میٹری مانیٹرنگ + میزبان آفس اوقات (2025-03-31 تک روزانہ چیک ان ؛ چیک پوائنٹ نیچے نیچے)۔
- مڈویو فیڈ بیک/ایشوز جمع اور ٹیگ کردہ `docs-preview/w0` (دیکھیں [W0 ڈائجسٹ] (./preview-feedback/w0/summary.md))۔
