# Torii API (ریپڈوک)

`RapiDoc` ایک کمپیکٹ ، دستاویزات کی پہلی پیش کش کرتا ہے اسی Torii کی
OpenAPI دستاویز۔ جب آپ کو ڈینسر جائزہ یا صرف پڑھنے کے صرف جائزہ لینے کی ضرورت ہو تو اس کا استعمال کریں
درخواست/رسپانس ماڈل۔

## استعمال کے نکات

- انٹرایکٹو کالز کو قابل بنانے کے لئے بیئرر ٹوکن درج کریں۔ قیمت کو بطور AND بھیج دیا گیا ہے
  `X-TryIt-Auth` ہیڈر اور پراکسی کے ذریعہ معمول بن گیا۔
- پورٹل اتنے انٹرایکٹو کی خدمت کرتے وقت `TRYIT_PROXY_PUBLIC_URL` تشکیل دیں
  درخواستوں کو اسٹیجنگ پراکسی کے ذریعے روٹ کیا جاتا ہے۔ پراکسی جب ویجیٹ نے متنبہ کیا ہے
  یو آر ایل غائب ہے۔
- شائع کردہ تاریخی چشمیوں کو لوڈ کرنے کے لئے "OpenAPI اسنیپ شاٹ" ڈراپ ڈاؤن کا استعمال کریں
  `npm run sync-openapi` ؛ `Latest (tracked)` آپشن ہمیشہ حل ہوتا ہے
  `/openapi/torii.json`۔
- جب "[کوشش کریں] (/api/overview#try-it-sandbox-docs-3)" کنسول کے ساتھ جوڑیں
  پراکسی کی ضرورت ہوتی ہے اسٹیج گیٹ ویز کے خلاف کام کرنا۔
