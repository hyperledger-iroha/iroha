جب بھی آپ اپنے ڈویلپر پورٹل کو اپ ڈیٹ کرتے ہیں تو اس چیک لسٹ کا استعمال کریں۔ وہ اس کی ضمانت دیتا ہے
سی آئی بلڈ ، گٹ ہب صفحات کی تعیناتی اور دستی تمباکو نوشی کے ٹیسٹ رہائی سے پہلے تمام حصوں کا احاطہ کرتے ہیں
یا روڈ میپ سنگ میل۔

## 1. مقامی چیک

- `npm run sync-openapi -- --version=current --latest` (ایک یا زیادہ شامل کریں
  جھنڈے `--mirror=<label>` ، اگر Torii OpenAPI ایک منجمد اسنیپ شاٹ کے لئے تبدیل ہوتا ہے)۔
- `npm run build` - یقینی بنائیں کہ "اعتماد کے ساتھ Iroha پر تعمیر کریں" اب بھی ہے
  `build/index.html` میں ظاہر ہوتا ہے۔
- `./docs/portal/scripts/preview_verify.sh --build-dir build` - مینی فیسٹ چیک کریں
  چیکسم (ڈاؤن لوڈ شدہ CI نمونے کی جانچ کرتے وقت `--descriptor`/`--archive` شامل کریں)۔
- `npm run serve` - چیکنگ چیکز کے ساتھ مددگار موڈ لانچ کرتا ہے ، جو توثیق کرتا ہے
  `docusaurus serve` کو فون کرنے سے پہلے ظاہر کریں تاکہ جائزہ لینے والے کبھی نہیں دیکھیں
  دستخط شدہ اسنیپ شاٹ (عرف `serve:verified` واضح کالوں کے لئے باقی ہے)۔
- براہ راست دوبارہ لوڈ کے ساتھ `npm run start` اور DEV سرور کے ذریعے ترمیم شدہ مارک ڈاون فائلوں کو دیکھیں۔

## 2۔ درخواست کی جانچ پڑتال کریں

- اس بات کو یقینی بنائیں کہ نوکری `docs-portal-build` میں کامیاب رہا
  `.github/workflows/check-docs.yml`۔
- چیک کریں کہ `ci/check_docs_portal.sh` لانچ کیا گیا تھا (ہیرو چیک CI لاگز میں ظاہر ہوتا ہے)۔
- اس بات کو یقینی بنائیں کہ ورک فلو کا پیش نظارہ مینی فیسٹ (`build/checksums.sha256`) بھری ہوئی ہے اور وہ
  پیش نظارہ اسکرپٹ کامیابی کے ساتھ مکمل ہوا (CI لاگز میں آؤٹ پٹ ہوتا ہے
  `scripts/preview_verify.sh`)۔
- GITHUB صفحات کے ماحول سے PR کی تفصیل میں شائع شدہ پیش نظارہ URL شامل کریں۔

## 3. حصوں کے ذریعہ دستخط کرنا

| سیکشن | مالک | چیک لسٹ |
| -------- | ---------- | ------------ |
| ہوم | ڈیوریل | ہیرو کاپی رائٹ ظاہر ہوتا ہے۔ کوئیک اسٹارٹ کارڈ درست راستوں کا باعث بنتے ہیں۔ سی ٹی اے بٹن کام کرتے ہیں۔ |
| Norito | Norito wg | جائزہ اور شروع کرنے والے ہدایت نامہ موجودہ CLI جھنڈوں اور Norito-SCHEMA-DOCS کا حوالہ دیتے ہیں۔ |
| SoraFS | اسٹوریج ٹیم | کوئیک اسٹارٹ تکمیل تک چلتا ہے ، مینی فیسٹ رپورٹ فیلڈز دستاویزی ہیں ، بازیافت نقلی ہدایات کی جانچ کی جاتی ہے۔ |
| SDK گائڈز | SDK لیڈز | مورچا/ازگر/جے ایس گائیڈ موجودہ مثالوں کو جمع کرتے ہیں اور زندہ ذخیروں سے لنک کرتے ہیں۔ |
| حوالہ | دستاویزات/ڈیوریل | انڈیکس میں تازہ ترین وضاحتیں ہیں۔ Norito - کوڈیک ڈائرکٹری `norito.md` سے مساوی ہے۔ |
| پیش نظارہ-اتیفیکٹ | دستاویزات/ڈیوریل | نمونہ `docs-portal-preview` PR کے ساتھ منسلک ہے ، دھواں چیکوں کو منظور کیا گیا ہے ، جائزہ لینے والوں کے لئے لنک شائع کیا گیا ہے۔ |
| سیکیورٹی اور کوشش کریں سینڈ باکس | دستاویزات/ڈیوریل · سیکیورٹی | OAuth ڈیوائس کوڈ لاگ ان (`DOCS_OAUTH_*`) تشکیل دیا گیا ہے ، چیک لسٹ `security-hardening.md` مکمل ہے ، CSP/قابل اعتماد قسم کے ہیڈر `npm run build` یا `npm run probe:portal` کے ذریعے چیک کیے جاتے ہیں۔ |

PR کے جائزے کے دوران ہر لائن کو نشان زد کریں یا فالو اپ کے کاموں کو ریکارڈ کریں تاکہ حیثیت کی حیثیت
درست رہا۔

## 4. جاری نوٹ

- ریلیز نوٹ میں `https://docs.iroha.tech/` (یا تعیناتی کی نوکری سے ماحولیاتی URL) شامل کریں
  اور حیثیت کی تازہ کاری۔
- واضح طور پر نئے یا تبدیل شدہ حصوں کی فہرست بنائیں تاکہ بہاو ٹیمیں سمجھ جائیں
  کون سا دھواں ٹیسٹ انہیں دوبارہ شروع کرنے کی ضرورت ہے۔
