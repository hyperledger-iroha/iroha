- کنٹرول ڈٹرمینسٹک برابر ہے (`crates/sorafs_car/tests/sorafs_cli.rs` ، [آپریٹر SoraFS کے لئے GA پیریٹی رپورٹ] (./orchestrator-ga-parity.md))

## لانچ چیک لسٹ میں اضافے

** جی اے فلٹر کو اپ گریڈ کرتے وقت لانچ مینیجرز کو درج ذیل شواہد شامل کرنا ہوں گے:

1۔ تازہ ترین سیکیورٹی جائزہ نوٹ (اس دستاویز) کا ہیش۔  
2. فالو اپ ریمیڈیشن ٹکٹ (جیسے `governance/tickets/SF6-SR-2026.md`) سے لنک کریں۔  
3. `scripts/release_sorafs_cli.sh --manifest ... --bundle-out ... --signature-out ...` کا آؤٹ پٹ واضح سامعین/جاری کرنے والے پیرامیٹرز کو دکھا رہا ہے۔  
4. برابری کنٹرول لاگ ان پر قبضہ (`cargo test -p sorafs_car -- --nocapture sorafs_cli::proof_stream::bounded_channels`)۔
