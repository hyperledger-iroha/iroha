# Torii API（Swagger UI）

此頁面使用 `swagger-ui-react` 嵌入 Torii OpenAPI 文檔。當
您想要熟悉的 Swagger 佈局、請求示例以及嘗試的能力
直接在瀏覽器中輸出請求。

## 使用技巧

- 在面板中提供不記名令牌以預授權每個請求。代幣是
  使用 `X-TryIt-Auth` 標頭轉發到代理，並且永遠不會保留。
- 在為門戶提供服務時配置 `TRYIT_PROXY_PUBLIC_URL`，以便小部件
  通過捆綁的暫存代理轉發請求。出現警告橫幅
  如果代理被禁用。
- 從下拉列表中選擇 OpenAPI 快照以在最新規格之間切換
  以及 `npm run sync-openapi` 生成的存檔版本。
- 嘗試代理支持的“[Try it](/api/overview#try-it-sandbox-docs-3)”控制台
  適用於 CORS 阻止直接 Torii 訪問的分階段環境。
- 使用 `cargo xtask openapi` 保持 OpenAPI 快照最新（請參閱
  `docs/portal/static/openapi/torii.json`）。
