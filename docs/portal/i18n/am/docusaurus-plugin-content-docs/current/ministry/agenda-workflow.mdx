ሞኖ-ሪፖን ሳይገልጹ የማረጋገጫ ዝርዝር።

   ቆራጥነትን ሳያጡ የቀደሙ ጉዳዮችን ይፈልጉ ።
3. የመደርደር አጋዥን ያሂዱ፣ የመርክል ማረጋገጫ ቅርቅቡን ያዙ እና ገጹን ላይ ያድርጉት

## 2. የተባዛ-የመዝገብ አውድ ያትሙ

እያንዳንዱ ተቀባይነት ያለው ሀሳብ ከዚህ በፊት ምክክር የተደረገውን የመመዝገቢያ ምግብን ያካትታል
አስገባ። ለዚያ መዝገብ የSoraFS CID ወይም Torii አንጸባራቂ መንገድን ያንሱ እና ያክሉ
የምክር ቤት ገምጋሚዎች ተመሳሳዩን JSON እንዲያረጋግጡ ወደ እርስዎ ያስገቡት ሜታዳታ።


ኦፕሬተሮች እንደገና ግቤትን በቆራጥነት ማዋሃድ ወይም ውድቅ ማድረግ ይችላሉ።
