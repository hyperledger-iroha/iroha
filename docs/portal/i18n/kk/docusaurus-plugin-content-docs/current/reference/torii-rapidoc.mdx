# Torii API (RapiDoc)

`RapiDoc` бірдей Torii ықшам, құжаттаманы бірінші рет көрсетуді ұсынады.
OpenAPI құжаты. Оны көбірек шолу немесе тек оқуға арналған шолу қажет болғанда пайдаланыңыз
сұрау/жауап үлгілері.

## Пайдалану бойынша кеңестер

- Интерактивті қоңырауларды қосу үшін жеткізуші таңбалауышын енгізіңіз. Мән ретінде жіберіледі
  `X-TryIt-Auth` тақырыбы және прокси арқылы қалыпқа келтірілген.
- Порталға интерактивті қызмет көрсету кезінде `TRYIT_PROXY_PUBLIC_URL` теңшеңіз
  сұраулар кезеңдік прокси арқылы бағытталады. Виджет прокси болғанда ескертеді
  URL мекенжайы жоқ.
- Жариялаған тарихи сипаттамаларды жүктеу үшін "OpenAPI суреті" ашылмалы тізімін пайдаланыңыз.
  `npm run sync-openapi`; `Latest (tracked)` опциясы әрқашан шешіледі
  `/openapi/torii.json`.
- «[Көріп көріңіз](/api/overview#try-it-sandbox-docs-3)» консольімен біріктіріңіз:
  проксиді қажет ететін кезеңдік шлюздерге қарсы жұмыс.
