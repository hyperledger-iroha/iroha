# Torii API (RapiDoc)

`RapiDoc` 提供了相同 Torii 的紧凑、文档优先的渲染
OpenAPI 文档。当您需要更深入的概述或只读审查时使用它
请求/响应模型。

## 使用技巧

- 输入不记名令牌以启用交互式呼叫。该值被转发为
  `X-TryIt-Auth` 标头并由代理标准化。
- 在为门户提供交互式服务时配置 `TRYIT_PROXY_PUBLIC_URL`
  请求通过暂存代理进行路由。当代理出现时，小部件会发出警告
  网址缺失。
- 使用“OpenAPI 快照”下拉列表加载发布的历史规格
  `npm run sync-openapi`； `Latest (tracked)` 选项始终解析为
  `/openapi/torii.json`。
- 与“[Try it](/api/overview#try-it-sandbox-docs-3)”控制台结合使用
  针对需要代理的分阶段网关。
