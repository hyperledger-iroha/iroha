Estos son los primeros registros de SDK de inicio rápido y tutoriales. Una lista de verificación de fragmentos de código abierto para restaurar y combinar programas con Rust, Python y JavaScript, cuáles son las mejores aplicaciones para estos сценарий от начала до конца.

- **[Каркас входной точки Hajimari](./hajimari-entrypoint)** — Минимальный каркас контракта Kotodama с одной публичной точкой входа и хендлом состояния.

- **[Перевести актив между аккаунтами](./transfer-asset)** — Un nuevo SDK de inicio rápido y tutoriales detallados реестра.
- **[Выпустить, перевести and сжечь NFT](./nft-flow)** — Проводит по жизненному циклу NFT от начала до конца: выпуск владельцу, перевод, добавление метаданных и сжигание.
