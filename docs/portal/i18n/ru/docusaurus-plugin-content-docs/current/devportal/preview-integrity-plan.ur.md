# контрольная сумма سے مشروط پریویو پلان

یہ منصوبہ وہ باقی کام بیان کرتا ہے جو ہر پورٹل پریویو آرٹیفیکٹ کو اشاعت سے پہلے قابلِ تصدیق بنانے کے لئے درکار ہے۔ CI میں بنائی گئی بالکل وہی اسنیپ ڈاؤن لوڈ Контрольная сумма может быть изменена с помощью SoraFS کے ذریعے Norito میٹا ڈیٹا کے ساتھ قابلِ دریافت ہو۔

## اہداف

- **Детерминистические сборки:** یقینی بنائیں کہ `npm run build` قابلِ اعادہ آؤٹ پٹ بنائے اور ہمیشہ `build/checksums.sha256` خارج کرے۔
- **Зарегистрируйтесь:** ہریویو آرٹیفیکٹ کے ساتھ checksum مینی فیسٹ لازمی ہو اور تصدیق ناکام ہونے پر اشاعت روک دی جائے۔
- **Norito может быть создано:** پریویو ڈسکرپٹرز (commit ڈیٹا، контрольной суммы, Если SoraFS CID) или Norito JSON, вы можете получить доступ к данным в формате SoraFS. کو آڈٹ کر سکیں۔
- ** آپریٹر ٹولنگ:** ایک مرحلہ وار تصدیقی اسکرپٹ فراہم کریں جو صارفین لوکل Установите флажок (`./docs/portal/scripts/preview_verify.sh --build-dir build --descriptor <path> --archive <path>`); Контрольная сумма + дескриптор, который может быть использован в качестве контрольной суммы и дескриптора. معیاری پریویو کمانڈ (`npm run serve`) اب `docusaurus serve` سے پہلے یہ ہیلپر خودکار طور Для того, чтобы проверить контрольную сумму, необходимо выполнить проверку (на `npm run serve:verified`). واضح عرف برقرار رہے).

## توثیقی اسکرپٹ

`docs/portal/scripts/preview_verify.sh` может быть использован в качестве источника питания `sha256sum`. بغیر کرتا ہے۔ Для получения дополнительной информации используйте `npm run serve` (например, `npm run serve:verified`). تاکہ اسکرپٹ چل کر `docusaurus serve` ایک ہی قدم میں شروع ہو جائے۔ Ответ:

1. Выберите SHA (`sha256sum` или `shasum -a 256`) или `build/checksums.sha256`, чтобы установить его.
2. اختیاری طور پر پریویو ڈسکرپٹر `checksums_manifest` کے дайджест/فائل نام اور، اگر فراہم ہو، پریویو آرکائیو کے дайджест/فائل نام کا موازنہ کرتا ہے۔
3. کسی بھی عدم مطابقت پر نان زیرو کوڈ کے ختم ہوتا ہے تاکہ ریویورز چھیڑے گئے پریویوز کو بلاک کر سکیں۔

Ответ на вопрос (CI آرٹیفیکٹس نکالنے کے بعد):

```bash
./docs/portal/scripts/preview_verify.sh \
  --build-dir build \
  --descriptor artifacts/preview-descriptor.json \
  --archive artifacts/preview-site.tar.gz
```

CI может быть использован для решения проблем, связанных с CI. Если вам нужна помощь, вы можете использовать ее, чтобы получить больше информации.
