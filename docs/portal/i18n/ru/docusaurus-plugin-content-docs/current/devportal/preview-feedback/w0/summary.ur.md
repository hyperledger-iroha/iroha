1. **Контрольная сумма ورک فلو** - تمام рецензенты نے تصدیق کی کہ `scripts/preview_verify.sh`
   مشترکہ дескриптор/архив جوڑی کے خلاف کامیاب رہا۔ کسی دستی переопределить کی
   ضرورت نہیں ہوئی۔
2. **Нажмите здесь** — боковая панель для отображения информации о текущем состоянии.
   (`docs-preview/w0 #1-#2`). Используйте Docs/DevRel, чтобы узнать больше о том, как это сделать.
   نہیں کرتے۔
3. **SoraFS Runbook برابری** - sorafs-ops-01 نے `sorafs/orchestrator-ops` اور
   `sorafs/multi-source-rollout` کے درمیان زیادہ и перекрестные ссылки, которые можно использовать
   следующий выпуск بنایا گیا؛ W1 سے پہلے حل کرنا ہے۔
4. **ٹیلیمیٹری ریویو** - наблюдаемость-01 نے تصدیق کی کہ `docs.preview.integrity`,
   `TryItProxyErrors` اور Журналы прокси-сервера Try-it Зеленый цвет رہے؛ Оповещение о тревоге نہیں ہوا۔
