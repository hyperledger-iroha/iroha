# Трекер приглашений для предварительного просмотра

Этот трекер регистрируется каждый раз для предварительного просмотра портала документов для владельцев DOCS-SORA и правительственных ревизоров, когда группа активна, что вызывает приглашения и те артефакты, которые ожидаются. Актуализировалось каждый раз, когда вы были приглашены, отозваны или изменены приглашения, чтобы аудитория находилась в хранилище.
