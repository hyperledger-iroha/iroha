# Предварительный просмотр трекера приглашений

Этот трекер зарегистрирует небольшой предварительный просмотр портальных документов до тех пор, пока владельцы DOCS-SORA и управляющие устройствами будут активны, которые одобряют приглашения и артефакты, остающиеся предателями. Обратите внимание на то, что приглашения посланников, отозванных или репортеров для проведения аудита на складе.
