Эти примеры отражают краткое руководство по использованию SDK и парковку регистрации. Затем необходимо перегруппировать список проверок регистрации и отозвать руководства по Rust, Python и JavaScript, чтобы вы могли вернуться к сценарию боя в бою.

- **[Squelette du point d'entrée Hajimari](./hajimari-entrypoint)** — Минимальная структура контракта Kotodama с одной публичной точкой входа и государственным руководством.
- **[Enregistrer un domaine et frapper des actifs](./register-and-mint)** — Демонтируйте создание доменов с авторизацией, регистрацией действий и детерминизмом.
- **[Invoquer le Transfert Hôte depuis Kotodama](./call-transfer-asset)** — Прокомментируйте точку входа Kotodama, чтобы вызвать инструкцию Hôte `transfer_asset` с проверкой метаданных на линии.
- **[Transférer un actif entre Comptes](./transfer-asset)** — поток передачи действий простой, который отражает быстрые запуски SDK и парки регистрации.
- **[Frapper, transférer et brûler un NFT](./nft-flow)** — Parcourt le Cycle de vie d'un NFT de bout en bout: frappe au proprietaire, Transfert, ajout de métadonnées et Destruction.
