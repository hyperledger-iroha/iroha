В этом разделе вы найдете материал «подробно как специальный» для Iroha. Эти страницы содержат множество материалов, включая те, которые развиваются в соответствии с инструкциями и обучающими материалами.

## Диспонибл хой

- **Возобновление кодека Norito** - `reference/norito-codec.md` сразу попадает в авторитарную спецификацию `norito.md`, пока не будет завершена таблица портала.
- **Torii OpenAPI** - `/reference/torii-openapi` отображает конкретную спецификацию REST, полученную из Torii, используя Redoc. Обновите спецификацию с `npm run sync-openapi -- --version=current --latest` (агрегат `--mirror=<label>` для копирования снимка в дополнительные исторические версии).
- **Таблицы конфигурации** - Полный каталог параметров находится в `docs/source/references/configuration.md`. Прежде чем на публичном портале произойдет автоматический импорт, проконсультируйтесь с этим архивом Markdown для значений, связанных с дефектами, и лас-анулаций де-энторно.
- **Версия документов** - Удаленная версия на панели навигации демонстрирует замороженные снимки, созданные с `npm run docs:version -- <label>`, чтобы облегчить сравнение между версиями.
