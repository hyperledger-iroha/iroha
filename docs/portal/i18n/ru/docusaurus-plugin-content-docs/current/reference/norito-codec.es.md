# Ссылка на кодек Norito

Norito — это каноническая сериализация Iroha. Сообщения по сети, полезная нагрузка на диске и API между компонентами США Norito для того, чтобы узлы обрабатывались в байтах с идентификаторами, включая различные аппаратные средства. На этой странице можно найти резюме движущихся объектов и апунта с полной спецификацией в `norito.md`.

## База Дисено

| Компонент | Предложение | Фуэнте |
| --- | --- | --- |
| **Энкабесадо** | Инкапсуляция полезных данных с хешем магии/версии/схемы, CRC64, долготой и этикеткой сжатия; v1 требуется `VERSION_MINOR = 0x00` и действительны флаги инкапсуляции против поддерживаемой туши (из-за дефекта `0x00`). | `norito::header` - версия `norito.md` («Заголовок и флаги», выбор репозитория) |
| **Полезная нагрузка без инкабеза** | Кодификация определенных значений, используемых для хеширования/сравнения. El Transporte on-wire siempre USA encabezado; лос байты грех encabezado сын соло внутринос. | `norito::codec::{Encode, Decode}` |
| **Сжатие** | Дополнительный ZSTD (и экспериментальное ускорение графического процессора) выбирается с помощью байта сжатия инкабезадо. | `norito.md`, «Согласование сжатия» |

Регистр флагов макета (packed-struct, packaged-seq, битовый набор полей, компактная длина) жив в `norito::header::flags`. Флаги США V1 `0x00` из-за дефекта, но мы приняли флаги, явные для поддерживаемой туши; Лос-биты исчезли, и они были заменены. `norito::header::Flags` сохраняется для проверки внутренних и будущих версий.

## Поддержка извлечения

`norito_derive` ofrece является производным от `Encode`, `Decode`, `IntoSchema` и вспомогательных JSON. Клавиатура соглашений:

- Лос генераны руты AoS упаковываются; v1 USA el макет AoS из-за дефекта (флаги `0x00`) залп, который лос-флаги инкапсуляции могут быть упакованы по вариантам. Реализация вживую в `crates/norito_derive/src/derive_struct.rs`.
- Функции, которые влияют на макет (`packed-struct`, `packed-seq`, `compact-len`), включаются с помощью флагов инкабеза и кодифицируются/декодируются в форме, согласованной между узлами.
- Помощники JSON (`norito::json`) доказали, что JSON соответствует Norito для работы API. США `norito::json::{to_json_pretty, from_json}` - nunca `serde_json`.

## Мультикодек и таблицы идентификаторов

Norito содержит назначенные мультикодеки в `norito::multicodec`. Таблица ссылок (хеши, тип клавиатуры, дескрипторы полезной нагрузки) находится в `multicodec.md` в хранилище репозитория. Когда появится новый идентификатор:

1. Актуализация `norito::multicodec::registry`.
2. Расширьте таблицу в `multicodec.md`.
3. Обновите последующие привязки (Python/Java) при использовании карты.
