`dashboards/grafana/ministry_moderation_overview.json` и правила оповещения

тесты были найдены в `dashboards/alerts/tests/ministry_moderation_rules.test.yml`).
Эти артефакты появляются из-за предупреждений о блокировках при проглатывании, фотографии дрейфа
и проверка карантина файлов, удовлетворение требований мониторинга
упоминается в [Спецификация средства модерации AI] (../../ministry/ai-moderation-runner.md).
