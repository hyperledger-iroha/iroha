## Дашборды и оповещения

SRE по модерации должен импортировать панель управления Grafana из
`dashboards/grafana/ministry_moderation_overview.json` и правила оповещений Prometheus
из `dashboards/alerts/ministry_moderation_rules.yml` (покрытие тестов находится в
`dashboards/alerts/tests/ministry_moderation_rules.test.yml`). Эти материалы
генерируют оповещения при блокировках приема внутрь, всплесках, дрифте и росте очереди
карантин, выполнение требований, защита в Диптихах
[Спецификация средства модерации AI] (../../ministry/ai-moderation-runner.md).
