## تدفق موافقة светильники

1. **Вечеринка**
   - Инструмент Tooling WG для изготовления приспособлений `manifest_blake3.json`.
     В сети есть `sorafs.fixtureProposal`.
   - Дайджест новостей от BLAKE3 в журнале «Вечеринка».
2. **Вечеринка**
   - تتلقى لجنة البنية التحتية التكليف عبر طابور مهام البرلمان.
   - يفحص اعضاء اللجنة artefacts الخاصة بـ CI ويجرون اختبارات التماثل ويصوتون
     в сети
3. **Убийство**
   - Вы можете просмотреть дайджест канонического манифеста во время выполнения.
     Нападающий Меркл сыграл в матче.
   - Выполняется в соответствии с SoraFS, когда используется манифест манифеста. البرلمان.
4. **Старт**
   - В CLI (`cargo xtask sorafs-fetch-fixture`) отображается манифест вызова Nexus RPC.
     Поддержка JSON/TS/Go для встроенного программного обеспечения `export_vectors`
     Вы можете получить дайджест новостей онлайн в сети.

## سير عمل المطورين

- Календарь матчей «Турнира»:

```bash
cargo run -p sorafs_chunker --bin export_vectors
```

- استخدم اداة جلب البرلمان لتنزيل конверт المعتمد والتحقق من التواقيع وتحديث светильники
  المحلية. اشِر `--signatures` в конверте المنشور من البرلمان؛ تقوم الاداة بحل манифест
  Создан дайджест BLAKE3 на сайте `sorafs.sf1@1.0.0`.

```bash
cargo xtask sorafs-fetch-fixture \
  --signatures https://nexus.example/api/sorafs/manifest_signatures.json \
  --out fixtures/sorafs_chunker
```

Создан `--manifest` в манифесте в Сан-Франциско. يتم رفض конверты غير الموقعة الا اذا
تم ضبط `--allow-unsigned` لاختبارات Smoke المحلية.

- В манифесте промежуточного хранения и Torii отображаются полезные нагрузки:

```bash
sorafs-fetch \
  --plan=fixtures/chunk_fetch_specs.json \
  --gateway-provider=name=staging,provider-id=<hex>,base-url=https://gw-stage.example/,stream-token=<base64> \
  --gateway-manifest-id=<manifest_id_hex> \
  --gateway-chunker-handle=sorafs.sf1@1.0.0 \
  --json-out=reports/staging_gateway.json
```

- В составе CI المحلي يتطلب باسم `signer.json`.
  `ci/check_sorafs_fixtures.sh` يقارن حالة المستودع باحدث التزام on-chain ويفشل
  عندما تختلف.
