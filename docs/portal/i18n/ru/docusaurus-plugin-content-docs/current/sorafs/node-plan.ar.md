В ящике SF-3 есть ящик `sorafs-node`, установленный Iroha/Torii. Имя пользователя SoraFS. استخدم هذه الخطة بجانب [دليل تخزين العقدة](node-storage.md) и [سياسة قبول] الموفّرين](provider-admission-policy.md) и [خارطة طريق سوق سعة التخزين](storage-capacity-marketplace.md) عند ترتيب التسليمات.

## Защитное устройство (экран M1)

1. **Отключено.** Установите `sorafs_car::ChunkStore`. Он манифестирует PoR в Национальном банке.
2. **Отключите соединение.** Установите контакт HTTP с Norito для вывода PIN-кода. Он был создан PoR и создан для Torii.
3. **Отключено.** Установите флажок `SoraFsStorage` (отключено от сети). Для этого необходимо установить `iroha_config` и `iroha_core` и `iroha_torii`.
4. **Установка/закрытие.** Для получения дополнительной информации/запроса используйте В Сан-Франциско существует противодавление.
5. **Застежка-молния.** Закрепите булавку на крючке/отверстии для крепления к стене. Это сообщение PoR.
