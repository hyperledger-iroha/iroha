### إعداد

Сообщение от пользователя `iroha_config`:

```toml
[sorafs.storage]
enabled = false
data_dir = "/var/lib/iroha/sorafs"
max_capacity_bytes = "100 GiB"
max_parallel_fetches = 32
max_pins = 10_000
por_sample_interval_secs = 600
alias = "tenant.alpha"            # وسم اختياري مقروء
adverts:
  stake_pointer = "stake.pool.v1:0x1234"
  availability = "hot"
  max_latency_ms = 500
  topics = ["sorafs.sf1.primary:global"]
```- `enabled`: Добавлено сообщение. Был ложным сообщением 503 года, когда он был убит Николасом Хейном во время открытия.
- `data_dir`: Чтобы получить доступ к PoR, нажмите Fetch. Приложение `<iroha.data_dir>/sorafs`.
- `max_capacity_bytes`: حد صارم لبيانات القطع المثبتة. Наденьте булавки и нанесите их на крючок.
- `max_parallel_fetches`: Запустите планировщик, который будет работать с IO.
- `max_pins`: Зафиксируйте контакты манифеста в разделе «Открыть/закрыть».
- `por_sample_interval_secs`: Добавлено сообщение о PoR التلقائية. Когда будет создан `N` (открытый манифест) будет установлен. Добавление метаданных `N` (отображение метаданных `profile.sample_multiplier`) `1-4`). В 2007 году он переопределил Лилу Холла Стива и Кэтрин Хейл. `{"default":2,"sorafs.sf2@1.0.0":3}`.
- `adverts`: показывает рекламу `ProviderAdvertV1` (указатель доли, темы QoS и QoS). Он был убит в 1980-х годах в Сан-Франциско.
