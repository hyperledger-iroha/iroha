# Добро пожаловать на портал разработчиков SORA Nexus

Портал разработчиков SORA Nexus включает интерактивную документацию, экологические материалы SDK и справочники API для операторов Nexus и контрибьюторов Hyperledger Iroha. Он заполняет документацию основного сайта, выводы по первому плану, применимые к гайдам и характеристикам, которые можно сгенерировать непосредственно из этого репозитория. Кредитование теперь включает тематические точки входа Norito/SoraFS, подписанные снимки OpenAPI и соответствующий справочник Norito Streaming, чтобы контрибьюторы могли найти контракт контрольной плоскости стриминга, не пролистывая корневую спецификацию.

## Что можно сделать здесь

- **Изучить Norito** - начните с обзора и краткого руководства, чтобы понять модель сериализации и байт-код инструментов.
- **Запустить SDK** — следуйте краткому руководству по JavaScript и Rust уже сегодня; Руководства для Python, Swift и Android оказываются в наименьшей степени соблюдения рецептов.
- **Просмотреть справочники API** - страница Torii OpenAPI отображает актуальную спецификацию REST, а таблицы конфигураций ссылаются на канонические источники Markdown.
- **Подготовить развертывание** - оперативные runbook-и (телеметрия, расчет, оверлеи Nexus) переносятся из `docs/source/` и появляются здесь в меньшей степени.

## Текущий статус

- ✅ Тематический лендинг Docusaurus v3 с обновленной типографикой, героями/картами на градиентах и тайлами ресурсов, итоговую Norito Streaming.
- ✅ Плагин Torii OpenAPI, подключенный к `npm run sync-openapi`, проверкой подписанных фрагментов и CSP-защитой, примененными `buildSecurityHeaders`.
- ✅ Предварительный просмотр и пробное покрытие выполняются в CI (`docs-portal-preview.yml` + `scripts/portal-probe.mjs`), теперь они загружают документацию в потоковом режиме, краткие руководства SoraFS и справочные контрольные списки перед публикацией документов.
- ✅ Quickstarts Norito, SoraFS и SDK плюс раздел справочников уже в боковой панели; новые импорты из `docs/source/` (потоковая передача, оркестровка, Runbook), созданные здесь по мере их написания.

## Как принять участие

- См. `docs/portal/README.md` для локальных команд разработки (`npm install`, `npm run start`, `npm run build`).
- Задачи содержимого контента отслеживаются вместе с пунктами дорожной карты `DOCS-*`. Вклад приветствуется - перенесите раздел из `docs/source/` и добавьте страницу в боковую панель.
- Если вы добавляете генерируемый документ (спецификации, таблицы конфигурации), задокументируйте команду сборки, чтобы будущие участники могли легко его обновить.
