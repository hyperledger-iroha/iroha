## 1. Резюме intake

- **Бюллетень:** Еврокомиссия “DNS-based hosting services – transparency briefing” (Ref. DSA/2026/0311).
- **Ключевые требования:** хранить контактные метаданные резолвера 12 месяцев, публиковать addendum прозрачности обработки злоупотреблений и обеспечивать EU‑резидентов среди ревьюеров споров по жалобам потребителей.
- **Дата подачи:** 2026‑03‑12.
- **Срок:** 2026‑04‑15 (декларация соответствия для DG CONNECT).

Согласно SNS charter Section 11, находка внесена в governance‑трекер 2026‑03‑12 (тикет `SNS-REG-2026-11`).
