```
bit index:   7        5 4      3 2      1 0
             ┌─────────┬────────┬────────┬────┐
payload bit: │version  │ class  │  norm  │ext │
             └─────────┴────────┴────────┴────┘
```

- `addr_version = 0` (биты 7-5) القيم غير الصفرية محجوزة ويجب ان تؤدي
  Это `AccountAddressError::InvalidHeaderVersion`.
- `addr_class` используется в качестве источника питания (`0`) и устанавливается на место (`1`).
- `norm_version = 1` создан в соответствии с нормой v1. ستعيد المعايير المستقبلية استخدام
  Произошло это в 2 числа.
- `ext_flag` دائما `0`; Он сказал, что хочет, чтобы он сделал это.

Отобразить заголовок заголовка:

```
┌──────────┬──────────────────────────────────────────────┐
│ tag (u8) │ payload (depends on selector kind)           │
└──────────┴──────────────────────────────────────────────┘
```

Для создания пользовательского интерфейса и SDK необходимо выполнить следующее:

- `0x00` = نطاق افتراضي ضمني (بدون حمولة).
- `0x01` = дайджест-код (12-байтовый `blake2s_mac("SORA-LOCAL-K:v1", label)`).
- `0x02` = مدخل سجل عالمي (`registry_id:u32` с прямым порядком байтов).
