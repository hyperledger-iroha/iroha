# Линейная база конфиденциальной калибровки газа

В этом реестре содержатся достоверные результаты эталонов калибровки газа, которые являются конфиденциальными. Собранный документ в сочетании с лекарственными средствами, подтверждающими освобождение, с описанием процедуры в [Конфиденциальные активы и передача ZK] (./confidential-assets#calibration-baselines--acceptance-gates).

`ns/op` объединяет медиану времени по сравнению с инструкциями по критерию; `gas/op` — это медиа-арифметика расходов по расписанию, соответствующих `iroha_core::gas::meter_instruction`; `ns/gas` разделите наносекундные суммы между газовыми суммами и новыми инструкциями.

*Примечание.* Фактически хост Arm64 не возобновляет работу `raw.csv` по критерию неисправности; внесите изменения в код `CRITERION_OUTPUT_TO=csv` или внесите исправления перед этикетным выпуском для тех артефактов, которые необходимы для контрольного списка принятия этих дополнений. Если `target/criterion/` произойдет сбой после `--save-baseline`, вспомните корриду на хосте Linux или сериализуйте консоль в пакете релиза как временный временной промежуток. Как ссылка, журнал консоли Arm64 последней корриды в живых на `docs/source/confidential_assets_calibration_neon_20251018.log`.
