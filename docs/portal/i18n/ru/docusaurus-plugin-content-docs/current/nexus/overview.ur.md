Nexus (Iroha 3) Iroha 2 для многополосного подключения к сети Использование SDK для создания дополнительных приложений یہ صفحہ مونو ریپو میں نئے خلاصے `docs/source/nexus_overview.md` کی عکاسی کرتا ہے تاکہ پورٹل Если вы хотите, чтобы это было так, как вам нужно, вы можете сделать это прямо сейчас.

## ریلیز لائنز

- **Iroha 2** - Если вы хотите, чтобы это произошло, вы можете использовать
- **Iroha 3 / Sora Nexus** - Многополосная связь с автомагистралью (DS) رجسٹر کرتے ہیں اور مشترکہ گورننس, سیٹلمنٹ, اور آبزرویبیلٹی ٹولنگ وراثت میں لیتے ہیں۔
- Доступ к рабочей области (IVM + Kotodama) с использованием SDK فکسز, ABI اپڈیٹس اور Norito فکسچرز قابلِ نقل رہتے ہیں۔ آپریٹرز Nexus Для того, чтобы узнать больше о `iroha3-<version>-<os>.tar.zst`. Если вам нужна помощь с `docs/source/sora_nexus_operator_onboarding.md` دیکھیں۔

## آپریٹر کی ذمہ داریاں

1. **Добавление данных** - `config/config.toml` для создания пространства данных в пространстве данных. رکھیں؛ ہر ریلیز ٹکٹ کے ساتھ `--trace-config` آؤٹ پٹ محفوظ کریں۔
2. ** مینفسٹ ٹریکنگ** - شامل ہونے یا نوڈ اپ گریڈ سے پہلے کیٹلاگ انٹریز کو Доступ к космическому каталогу
3. **Подробнее** — `nexus_lanes.json`, `nexus_settlement.json` для SDK, который можно открыть. کریں؛ Используйте PagerDuty, чтобы узнать, как исправить ситуацию, если вы хотите использовать PagerDuty. چلائیں۔
4. **حادثہ رپورٹنگ** - [Nexus операции](./nexus-operations) Как использовать RCA или RCA?
5. **گورننس تیاری** - اپنی Lanes پر اثر انداز ہونے والی Nexus کونسل ووٹس میں حصہ لیں اور سہ ماہی откат ہدایات کی مشق کریں (ٹریکنگ `docs/source/project_tracker/nexus_config_deltas/` کے ذریعے ہوتی ہے)۔
