## Добавление полос (`LaneConfig`)

`LaneConfig` — для запуска среды выполнения в каталоге дорожек. لا تستبدل проявляет الحوكمة؛ Он выступил с фильмом "Территория" в Лейн-Монтон.

```text
LaneConfigEntry {
    lane_id: LaneId,           // stable identifier
    alias: String,             // human-readable alias
    slug: String,              // sanitised alias for file/metric keys
    kura_segment: String,      // Kura segment directory: lane_{id:03}_{slug}
    merge_segment: String,     // Merge-ledger segment: lane_{id:03}_merge
    key_prefix: [u8; 4],       // Big-endian LaneId prefix for WSV key spaces
    shard_id: ShardId,         // WSV/Kura shard binding (defaults to lane_id)
    visibility: LaneVisibility,// public vs restricted lanes
    storage_profile: LaneStorageProfile,
    proof_scheme: DaProofScheme,// DA proof policy (merkle_sha256 default)
}
```- `LaneConfig::from_catalog` используется для подключения к сети (`State::set_nexus`).
- تتحول псевдонимы الى slugs بحروف صغيرة؛ Установите флажок `_`. Эта пуля находится на `lane{id}`.
- `shard_id` используется в метаданных `da_shard_id` (название `lane_id`) в журнале shard المثبت При воспроизведении повтора DA происходит перезапуск/перезапуск.
- Дополнительные префиксы.
- اسماء مقاطع Kura حتمية عبر Hosts; В случае, когда человек лишается дара речи, он проявляет свою невиновность.
- Добавление (`lane_{id:03}_merge`) содержит подсказки о слиянии и обязательствах, которые можно использовать в переулке.
