# Базовые уровни конфиденциальной калибровки газа

Этот реестр сопровождает проверенные результаты эталонов калибровки газа, которые являются конфиденциальными. Это прямая документация в сочетании с медицинскими документами, подтверждающими освобождение, в виде процедуры описания [Конфиденциальных активов и передачи ZK] (./confidential-assets#calibration-baselines--acceptance-gates).

`ns/op` объединяет средние настенные часы для обучения среднему критерию кожи; `gas/op` и медиа-арифметика для корреспондентов расписания `iroha_core::gas::meter_instruction`; `ns/gas` Разделите наносекундные газы на несколько частей без дополнительных инструкций.

*Примечание.* Хост Arm64 на самом деле не выдает резюме `raw.csv` do Criterion por Padrao; ехал novamente com `CRITERION_OUTPUT_TO=csv` или вы исправляли выше по течению до этикета и выпускали пункты, которые os artefatos exigidos pela контрольный список de aceitacao sejam anexados. Если `target/criterion/` будет добавлен после `--save-baseline`, выполните его на хосте Linux или сериализуйте указанную консоль без пакета выпуска в качестве временного временного промежутка. Для справки запишите в журнал консоли Arm64 последнее выполнение, выполненное под `docs/source/confidential_assets_calibration_neon_20251018.log`.
