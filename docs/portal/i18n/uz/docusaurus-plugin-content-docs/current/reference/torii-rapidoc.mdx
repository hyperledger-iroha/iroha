# Torii API (RapiDoc)

`RapiDoc` bir xil Torii ning ixcham, hujjatlarni birinchi marta ko'rsatishini taklif qiladi
OpenAPI hujjati. Agar sizga batafsilroq ko'rib chiqish yoki faqat o'qish uchun mo'ljallangan ko'rib chiqish kerak bo'lganda foydalaning
so'rov/javob modellari.

## Foydalanish bo'yicha maslahatlar

- Interaktiv qo'ng'iroqlarni yoqish uchun tashuvchi tokenini kiriting. Qiymat a sifatida uzatiladi
  `X-TryIt-Auth` sarlavhasi va proksi-server tomonidan normallashtirilgan.
- Portalga interaktiv xizmat ko'rsatishda `TRYIT_PROXY_PUBLIC_URL` ni sozlang
  so'rovlar proksi-server orqali yuboriladi. Vidjet proksi kelganda ogohlantiradi
  URL manzili yo‘q.
- tomonidan nashr etilgan tarixiy xususiyatlarni yuklash uchun “OpenAPI snapshot” ochiladigan menyusidan foydalaning.
  `npm run sync-openapi`; `Latest (tracked)` varianti har doim hal qiladi
  `/openapi/torii.json`.
- “[Try it](/api/overview#try-it-sandbox-docs-3)” konsoli bilan birlashtiring.
  proksi-serverni talab qiladigan bosqichli shlyuzlarga qarshi ishlash.
