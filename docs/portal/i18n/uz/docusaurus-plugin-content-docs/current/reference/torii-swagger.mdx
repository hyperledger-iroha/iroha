# Torii API (Swagger UI)

Bu sahifa `swagger-ui-react` yordamida Torii OpenAPI hujjatini joylashtiradi. Qachon foydalaning
sizga tanish Swagger tartibi, so'rov namunalari va sinab ko'rish qobiliyatini xohlaysiz
so'rovlarni to'g'ridan-to'g'ri brauzer ichida amalga oshirish.

## Foydalanish bo'yicha maslahatlar

- Har bir so'rovni oldindan avtorizatsiya qilish uchun panelda tashuvchi tokenini taqdim eting. Tokenlar
  `X-TryIt-Auth` sarlavhasi yordamida proksi-serverga uzatiladi va hech qachon davom ettirilmaydi.
- Portalga xizmat ko'rsatishda `TRYIT_PROXY_PUBLIC_URL` ni vidjet bo'lishi uchun sozlang
  so'rovlarni yig'ilgan proksi-server orqali yuboradi. Ogohlantirish banneri paydo bo'ladi
  proksi o'chirilgan bo'lsa.
- Eng so'nggi spetsifikatsiyalar o'rtasida almashish uchun ochiladigan menyudan OpenAPI suratini tanlang.
  va `npm run sync-openapi` tomonidan yaratilgan arxivlangan versiyalar.
- Proksi-server bilan quvvatlangan “[Try it](/api/overview#try-it-sandbox-docs-3)” konsolini sinab ko'ring
  CORS to'g'ridan-to'g'ri Torii kirishiga to'sqinlik qiladigan bosqichli muhitlar uchun.
- OpenAPI suratini `cargo xtask openapi` bilan yangi saqlang (qarang.
  `docs/portal/static/openapi/torii.json`).
