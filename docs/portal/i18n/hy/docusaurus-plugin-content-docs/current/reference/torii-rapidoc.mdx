# Torii API (RapiDoc)

`RapiDoc`-ն առաջարկում է միևնույն Torii-ի կոմպակտ, փաստաթղթային ներկայացում
OpenAPI փաստաթուղթ: Օգտագործեք այն, երբ ձեզ անհրաժեշտ է ավելի խիտ ակնարկ կամ միայն կարդալու վերանայում
հարցում/պատասխան մոդելներ.

## Օգտագործման խորհուրդներ

- Մուտքագրեք կրող նշան՝ ինտերակտիվ զանգերը միացնելու համար: Արժեքը փոխանցվում է որպես ան
  `X-TryIt-Auth` վերնագիր և նորմալացված վստահված անձի կողմից:
- Կարգավորեք `TRYIT_PROXY_PUBLIC_URL` պորտալն այդքան ինտերակտիվ սպասարկելիս
  հարցումները ուղղորդվում են բեմադրող վստահված անձի միջոցով: Վիդջեթը զգուշացնում է, երբ վստահված անձը
  URL-ը բացակայում է:
- Օգտագործեք «OpenAPI snapshot» բացվող ցանկը՝ բեռնելու պատմական բնութագրերը հրապարակված
  `npm run sync-openapi`; `Latest (tracked)` տարբերակը միշտ լուծում է
  `/openapi/torii.json`.
- Միավորել «[Փորձիր այն](/api/overview#try-it-sandbox-docs-3)» վահանակի հետ, երբ
  աշխատում է բեմադրված դարպասների դեմ, որոնք պահանջում են վստահված անձ:
