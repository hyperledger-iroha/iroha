- **Տեղեկագիր․

- **Ներկայացման ամսաթիվը` ** 2026-12-20:
- **Վերջնաժամկետ՝** 2027-01-31 (համապատասխանության հայտարարություն DG CONNECT-ի պատճառով):

## 4. Ակտիվացում և ապացույցներ (օրեր 22–28)

- Registry/Ops-ն ավելացրել է `trusted_flagger_contact` + `trusted_flagger_sla_hours` GAR ձևանմուշ `v2026.12.eu-dsa`-ին և հրապարակել փոփոխությունը 2026-12-29 թվականներին:
