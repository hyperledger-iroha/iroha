Энэ трекер нь баримт бичгийн порталыг урьдчилан харах долгион бүрийг бүртгэдэг тул DOCS-SORA эзэмшигчид болон

хүчингүй болгосон, эсвэл хойшлуулсан тул аудитын мөр нь агуулах дотор үлдэх болно.
