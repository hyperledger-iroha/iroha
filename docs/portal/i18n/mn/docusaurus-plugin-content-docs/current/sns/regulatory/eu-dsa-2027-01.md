## 1. Хэрэглээний хураангуй

- **Мэдэгдэл:** Европын Комисс “DNS/хостинг үйлчилгээ үзүүлэгчийн итгэмжлэгдсэн дарцагчид ба шуурхай татан буулгах үүрэг” (Иш. DSA/2026/1219).
- **Гол шаардлагууд:** итгэмжлэгдсэн дарцаглагчийн ажлын урсгалыг нийтлэх, өндөр ноцтой мэдэгдэлд <4 цагийн хариу өгөх хугацааг баталгаажуулж, бүх нийтийн дагаварууд дээр харъяаллын тусгайлсан хүчирхийллийн харилцагчдыг илчлэх.
- **Иргүүлсэн огноо:** 2026-12-20.
- ** Эцсийн хугацаа:** 2027‑01‑31 (DG CONNECT-ийн дагуу дагаж мөрдөх тухай мэдэгдэл).

SNS дүрмийн 11-р хэсэгт зааснаар 2026-12-20-нд (тасалбар `SNS-REG-2026-48`) засаглалын мөрдөгч рүү орсон.

## 3. Дүрэм ба KPI шинэчлэлтүүд (14-21 дэх өдөр)

Зөвлөлийн түр нэмэлт `SN-CH-2026-12-A2`:

- Хэсэг 7 KPI оруулга `compliance.eu-dsa.trusted_flagger_sla` (Grafana самбар `trusted flagger SLA p95`-тай харьцуулан сар бүр шалгана уу).
- Итгэмжлэгдсэн дарцаглагчийн баталгаажуулалтын ажлын урсгалын 10-р хэсгийн лавлагаа, нярав + бүртгэгчийн runbook-тэй холбосон.

Удирдах ажилтны хүлээн зөвшөөрөлт: `docu-sign envelope f91c0f2` (2026-12-27). Prometheus дүрмийг ачаалсны дараа KPI хяналтын самбар `compliance.eu-dsa.trusted_flagger_sla=ok` харуулах болно.

## 4. Идэвхжүүлэлт ба нотлох баримт (22–28 дахь өдөр)

- Бүртгэл/Үйл ажиллагаа нь GAR загвар `v2026.12.eu-dsa` дээр `trusted_flagger_contact` + `trusted_flagger_sla_hours` нэмсэн бөгөөд өөрчлөлтийг 2026-12-29-нд нийтэлсэн.
- Хүчирхийллийн ширээний эргэлтийг одоо SNS KPI хавсралт (`.sora` 2027‑01 тайланг үзнэ үү) болон няравын тоглоомын номын хавсралтад оруулсан болно.
- `docs/portal/docs/sns/trusted-flagger-onboarding.md`-ийн дагуу итгэмжлэгдсэн дарцаглагчийг нээх маягт + гарын үсэг зурсан ажлын урсгалыг OAuth прокси порталаар дамжуулан хэрэгжүүлэх хандалтын хяналттай.
