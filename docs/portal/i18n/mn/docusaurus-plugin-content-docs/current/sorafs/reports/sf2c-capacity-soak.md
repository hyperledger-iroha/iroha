# SF-2c Хүчин чадлын аккруэл шингээлтийн тайлан

Огноо: 2026-03-21

## Хамрах хүрээ

Энэхүү тайланд SoraFS хүчин чадлын хуримтлал болон төлбөрийн шингээлтийн тодорхойлогчийг тэмдэглэсэн болно.
SF-2c замын зураглалын дагуу хүссэн туршилтууд.

- **30 хоногийн олон үйлчилгээ үзүүлэгч нэвт норгох:** Дасгал хийсэн
  `capacity_fee_ledger_30_day_soak_deterministic` инч
  `crates/iroha_core/src/smartcontracts/isi/sorafs.rs`.
  Морь нь таван үйлчилгээ үзүүлэгчийг бий болгож, 30 тооцооны цонхыг хамардаг
  Бүртгэлийн нийт дүн нь бие даасан тооцоолсон лавлагаатай таарч байгааг баталгаажуулдаг
  проекц. Туршилт нь Blake3 дижест (`capacity_soak_digest=...`) ялгаруулдаг тул
  CI нь каноник агшин зуурын зургийг авч, ялгаж чаддаг.
- **Дутуу хүргэлтийн торгууль:** Хэрэгжүүлсэн
  `record_capacity_telemetry_penalises_persistent_under_delivery`
  (ижил файл). Туршилт нь ажил хаялтын босго, хөргөлтийн хугацаа, барьцааны ташуу зураас,
  болон дэвтэр тоолуур нь детерминист хэвээр байна.

## Гүйцэтгэл

Дараахын тусламжтайгаар нэвт норгох баталгаажуулалтыг орон нутагт ажиллуулна уу:

```bash
cargo test -p iroha_core -- record_capacity_telemetry_penalises_persistent_under_delivery
cargo test -p iroha_core -- capacity_fee_ledger_30_day_soak_deterministic
```

Туршилтыг стандарт зөөврийн компьютер дээр нэг секундын дотор хийж дуусгах бөгөөд шаардлагагүй
гадаад бэхэлгээ.

## Ажиглалт

Torii нь одоо үйлчилгээ үзүүлэгчийн зээлийн агшин зуурын мэдээллийг хураамжийн дэвтэртэй хамт харуулах тул хяналтын самбарт
бага үлдэгдэл болон торгуулийн цохилтонд орох боломжтой:

- REST: `GET /v2/sorafs/capacity/state` нь `credit_ledger[*]` оруулгуудыг буцаана.
  нэвт норгох тестээр баталгаажуулсан дэвтэрийн талбаруудыг толин тусгал. Харна уу
  `crates/iroha_torii/src/sorafs/registry.rs`.
- Grafana импорт: `dashboards/grafana/sorafs_capacity_penalties.json` графикууд
  strike counters, торгуулийн нийт дүн, барьцаалагдсан барьцаа зэргийг дуудлагаар экспортолсон
  Ажилтнууд суурь үзүүлэлтүүдийг амьд орчинтой харьцуулж болно.

## Дагаж мөрдөх

- CI дахь долоо хоног тутмын гүйлтийн хуваарийг нэвт норгох туршилтыг (утааны түвшин) давтан хийнэ үү.
- Үйлдвэрлэлийн телеметрийн дараа Grafana самбарыг Torii хусах зорилттойгоор сунгана.
  экспорт идэвхтэй явагдана.
