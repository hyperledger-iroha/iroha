# Нийтлэлийн хяналтын хуудас

Хөгжүүлэгчийн порталыг шинэчлэх бүртээ энэ хяналтын хуудсыг ашиглаарай. Энэ нь баталгаажуулдаг
CI бүтээх, GitHub Pages байршуулалт, гар аргаар утааны тестүүд хэсэг бүрийг хамарна
хувилбар эсвэл замын газрын зураг буухаас өмнө.

## 1. Орон нутгийн баталгаажуулалт

- `npm run sync-openapi -- --version=current --latest` (нэг буюу хэд хэдэн нэмнэ
  `--mirror=<label>` Torii OpenAPI нь царцаасан агшин зуурын агшинд өөрчлөлт оруулах үед дарцаглана).
- `npm run build` - `Build on Iroha with confidence` баатрын хуулбарыг баталгаажуулна уу
  `build/index.html` дээр гарч ирнэ.
- `./docs/portal/scripts/preview_verify.sh --build-dir build` – баталгаажуулна уу
  шалгах нийлбэр манифест (татаж авсан CI-г туршихдаа `--descriptor`/`--archive` нэмнэ үү.
  олдворууд).
- `npm run serve` – шалгах нийлбэртэй урьдчилан харах туслахыг ажиллуулдаг.
  `docusaurus serve` руу залгахаас өмнөх манифест байдаг тул тоймчид хэзээ ч
  гарын үсэг зураагүй хормын хувилбар (`serve:verified` нь тодорхой дуудлагад үлддэг).
- `npm run start`-ээр дамжуулан таны хүрсэн тэмдэглэгээ болон шууд дахин ачааллыг газар дээр нь шалгана уу.
  сервер.

## 2. Хүсэлтийг шалгах

- `docs-portal-build` ажил `.github/workflows/check-docs.yml` дээр амжилттай хийгдсэн эсэхийг шалгана уу.
- `ci/check_docs_portal.sh` гүйж байгааг баталгаажуулна уу (CI бүртгэлүүд нь баатар утааны шалгалтыг харуулдаг).
- Манифест (`build/checksums.sha256`) байршуулсан урьдчилан харах ажлын урсгалыг баталгаажуулах ба
  Урьдчилан харах баталгаажуулалтын скрипт амжилттай болсон (CI бүртгэлүүд нь
  `scripts/preview_verify.sh` гаралт).
- GitHub Pages орчноос нийтлэгдсэн урьдчилан үзэх URL-г PR-д нэмнэ үү
  тайлбар.

## 3. Хэсэгт гарын үсэг зурах

| хэсэг | Эзэмшигч | Хяналтын хуудас |
|---------|-------|-----------|
| Нүүр хуудас | DevRel | Баатарын хуулбар дүрслэл, хурдан эхлүүлэх картууд нь хүчинтэй маршруттай холбогдож, CTA товчлуурууд шийдэгддэг. |
| Norito | Norito WG | Тойм болон эхлүүлэх гарын авлага нь хамгийн сүүлийн үеийн CLI тугууд болон Norito схемийн баримтуудыг иш татдаг. |
| SoraFS | Хадгалах баг | Хурдан эхлүүлэх ажиллагаа дуустал гүйж, манифест тайлангийн талбаруудыг баримтжуулж, дуурайлган дуудах зааварчилгааг баталгаажуулсан. |
| SDK гарын авлага | SDK тэргүүлэгч | Rust/Python/JS гарын авлага нь одоогийн жишээнүүдийг эмхэтгэж, шууд репо руу холбоно. |
| Лавлагаа | Docs/DevRel | Индекс нь хамгийн сүүлийн үеийн үзүүлэлтүүдийг жагсаасан бөгөөд Norito кодлогчийн лавлагаа `norito.md`-тай таарч байна. |
| Олдворыг урьдчилан үзэх | Docs/DevRel | PR-д хавсаргасан `docs-portal-preview` олдвор, утааны шалгалтын хуудас, тоймчидтой хуваалцсан холбоос. |
| Аюулгүй байдал & Үүнийг туршаад үзээрэй хамгаалагдсан хязгаарлагдмал орчин | Docs/DevRel · Аюулгүй байдал | OAuth төхөөрөмжийн кодын нэвтрэлтийг тохируулсан (`DOCS_OAUTH_*`), `security-hardening.md` шалгах хуудсыг гүйцэтгэсэн, CSP/Итгэмжлэгдсэн төрлүүдийн толгой хэсгийг `npm run build` эсвэл `npm run probe:portal`-ээр баталгаажуулсан. |

Мөр бүрийг өөрийн PR тоймны нэг хэсэг болгон тэмдэглэж эсвэл дараагийн даалгавруудыг тэмдэглэ
хянах нь үнэн зөв хэвээр байна.

## 4. Хувилбарын тэмдэглэл

- `https://docs.iroha.tech/` (эсвэл орчны URL
  байршуулалтын ажлаас) хувилбарын тэмдэглэл болон статусын шинэчлэлтүүд.
- Доод урсгалын багууд хаана байгааг мэдэхийн тулд шинэ эсвэл өөрчлөгдсөн хэсгийг шууд дуудаарай
  өөрсдийн утааны шинжилгээг дахин явуулах.
