# Torii API (Swagger UI)

Энэ хуудсанд Torii OpenAPI баримтыг `swagger-ui-react` ашиглан оруулсан болно. Хэзээ ашигла
Та танил Swagger загвар, дээж хүсэх, оролдох чадварыг хүсч байна
хүсэлтийг хөтөч дотор шууд гаргах.

## Хэрэглэх зөвлөмж

- Хүсэлт бүрийг урьдчилан зөвшөөрөхийн тулд самбарт эзэмшигчийн жетон бичнэ үү. Токенууд нь
  `X-TryIt-Auth` толгойг ашиглан прокси руу дамжуулсан бөгөөд хэзээ ч хадгалагдахгүй.
- Виджет болохын тулд порталд үйлчлэхдээ `TRYIT_PROXY_PUBLIC_URL`-г тохируулна уу
  хүсэлтийг багцалсан проксигаар дамжуулдаг. Анхааруулах баннер гарч ирнэ
  хэрэв прокси идэвхгүй бол.
- Хамгийн сүүлийн үеийн үзүүлэлтүүдийн хооронд шилжихийн тулд унадаг цэснээс OpenAPI агшин зуурын зургийг сонгоно уу.
  болон `npm run sync-openapi` үүсгэсэн архивлагдсан хувилбарууд.
- Прокси дэмждэг “[Оролдоод үз](/api/overview#try-it-sandbox-docs-3)” консолыг туршаад үзээрэй.
  CORS нь Torii шууд хандалтыг хориглодог үе шаттай орчинд зориулагдсан.
- OpenAPI агшин зуурын зургийг `cargo xtask openapi` ашиглан шинэ байлгаарай (харна уу).
  `docs/portal/static/openapi/torii.json`).
