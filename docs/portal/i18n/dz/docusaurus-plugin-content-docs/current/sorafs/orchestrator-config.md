# འབྱུང་ཁུངས་ཕེཆ་ཨོར་ཀེཊ་ཊར་ལམ་སྟོན།

```json
}
```

### ༥.༢ བཀོལ་སྤྱོད་པ་ཐོ་ཡིག (ཐོན་ལས་ཀྱི་རླབས་རིམ།)1. **རིམ་སྒྲིག་འདི་ `iroha_config`.** གི་ཐོག་ལས་ གཏན་གཏན་སྦེ་ལག་ལེན་འཐབ་མི་ JSON དཔར་བསྐྲུན་འབད།
