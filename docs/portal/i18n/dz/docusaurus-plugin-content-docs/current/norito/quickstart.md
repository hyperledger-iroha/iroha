འདི་གིས་ ང་བཅས་ཀྱིས་ གོང་འཕེལ་གཏང་མི་ཚུ་གིས་ ལྷབ་སྦྱང་འབད་བའི་སྐབས་ ལཱ་འབད་ནི་གི་རེ་བ་ཡོད་པའི་ ལཱ་གི་རྒྱུན་རིམ་འདི་ གསལ་སྟོན་འབདཝ་ཨིན།
Norito དང་ Kotodama འདི་ འགོ་དང་པ་: གཏན་འབེབས་བཟོ་ནིའི་ མཉམ་རྐྱང་གི་ཡོངས་འབྲེལ་ཅིག་ བུཊི་འབད་དགོ།
གན་རྒྱ་ཅིག་བསྡུ་སྒྲིག་འབད་ཞིནམ་ལས་ ས་གནས་ནང་ སྐམ་བཏང་ཞིནམ་ལས་ དེ་ལས་ Torii བརྒྱུད་དེ་ བཏང་དགོ།
གཞི་བསྟུན་ CLI.

དཔེ་གན་ཡིག་འདི་གིས་ ལྡེ་མིག་/གནས་གོང་ཆ་གཅིག་ འབོད་བརྡ་གཏང་མི་གི་རྩིས་ཐོ་ལུ་བྲིསཝ་ཨིན་ དེ་འབདཝ་ལས་ ཁྱོད་ཀྱིས་འབད་ཚུགས།
`iroha_cli` དང་ཅིག་ཁར་ དེ་འཕྲོ་ལས་ ཟུར་ཕོག་འདི་བདེན་སྦྱོར་འབད།

## སྔོན་འགྲོའི་ཆ་རྐྱེན།

- [Docker](https://docs.docker.com/engine/install/) མཉམ་བསྡོམས་ཝི་༢ དང་བཅས་ (ལག་ལེན་འཐབ་ཡོད།
  དཔེ་ཚད་ཀྱི་མཉམ་རོགས་ `defaults/docker-compose.single.yml` ནང་གསལ་ལྟར་འགོ་བཙུགས་ནི།).
- ཁྱོད་ཀྱིས་ཕབ་ལེན་མ་འབད་བ་ཅིན་ གྲོགས་རམ་གཉིས་ལྡན་བཟོ་ནི་གི་དོན་ལུ་ རསཊི་ལག་ཆས་རྒྱུན་རིམ་ (༡.༧༦+) ཨིན།
  དཔར་བསྐྲུན་འབད་མི་ཚུ།
- `koto_compile`, `ivm_run`, དང་ `iroha_cli` གཉིས་ལྡན་ཚུ། ཁྱོད་ཀྱིས་ཁོང་ལས་བཟོ་ཚུགས།
  ལས་ཀའི་ས་སྟོང་བརྟག་ཞིབ་འདི་གཤམ་གསལ་ལྟར་དུ་སྟོན་ཡོད་པ་ཡང་ན་མཐུན་སྒྲིག་གསར་བཏོན་འབད་ཡོད་པའི་ཅ་རྙིང་ཚུ་ཕབ་ལེན་འབད།

```sh
cargo install --locked --path crates/ivm --bin koto_compile --bin ivm_run
cargo install --locked --path crates/iroha_cli --bin iroha
```

> གོང་འཁོད་ཀྱི་ གཉིས་ལྡན་ཚུ་ ལཱ་གི་ས་སྒོ་ལྷག་ལུས་ཚུ་དང་གཅིག་ཁར་ གཞི་བཙུགས་འབད་ནི་ལུ་ ཉེན་སྲུང་ཡོདཔ་ཨིན།
> ཁོང་ཚོས་ `serde`/`serde_json` དང་ནམ་ཡང་འབྲེལ་བ་མེད། Norito གསང་གྲངས་ཚུ་ མཐའ་མཇུག་ལས་མཇུག་ཚུན་ཚོད་ བསྟར་སྤྱོད་འབད་ཡོདཔ་ཨིན།

## 1. མཉམ་པའི་ dev ཡོངས་འབྲེལ་འགོ་བཙུགས།

མཛོད་ཁང་འདི་ནང་ `kagami swarm` གིས་བཏོན་ཡོད་པའི་ Docker བསྡུ་སྒྲིག་བང་རིམ་ཚུ་ཚུདཔ་ཨིན།
(Kotodama). འདི་གིས་ སྔོན་སྒྲིག་རིགས་མཚན་འདི་ གློག་ཐག་འབདཝ་ཨིན།
རིམ་སྒྲིག་, དང་ གསོ་བའི་འཚོལ་ཞིབ་ཚུ་ དེ་འབདཝ་ལས་ Torii འདི་ `http://127.0.0.1:8080` ལུ་ལྷོད་ཚུགས།

```sh
docker compose -f defaults/docker-compose.single.yml up --build
```

དམ་སྦྱིས་གཡོག་བཀོལ་མི་འདི་ (གདོང་ཕྱོགས་ལུ་ཡང་ན་ ཁ་ཕྱེ་ཡོདཔ་)། གེ་ར
དེ་གི་ཤུལ་ལས་ CLI གིས་ `defaults/client.toml` བརྒྱུད་དེ་ མཉམ་རོགས་འདི་དམིགས་གཏད་བསྐྱེདཔ་ཨིན།

## 2. རྩོམ་པ་པོས་གན་རྒྱ།

ལཱ་འབད་སའི་སྣོད་ཐོ་ཅིག་གསར་བསྐྲུན་འབད་ཞིནམ་ལས་ ཉུང་མཐའ་ Kotodama འདི་ གསོག་འཇོག་འབད།

```sh
mkdir -p target/quickstart
cat > target/quickstart/hello.ko <<'KO'
// Writes a deterministic account detail for the transaction authority.

seiyaku Hello {
  // Optional initializer invoked during deployment.
  hajimari() {
    info("Hello from Kotodama");
  }

  // Public entrypoint that records a JSON marker on the caller.
  kotoage fn write_detail() {
    set_account_detail(
      authority(),
      name!("example"),
      json!{ hello: "world" }
    );
  }
}
KO
```

> ཐོན་རིམ་ཚད་འཛིན་ནང་ Kotodama འབྱུང་ཁུངས་བཞག་ནི་ལུ་དགའ་བ། དྲྭ་ཐོག་-ཧོསཊི་དཔེ་ཚུ་ནི།
> ཡང་ [Norito དཔེ་སྟོན་པ་ ](./examples/) འོག་ལུ་ཡོདཔ་ཨིན།
> འགོ་བཙུགས་ས་ཚིགས་ཕྱུགཔོ་ཅིག་དགོཔ་ཨིན།

## 3. IVM དང་མཉམ་དུ་བསྒྲིགས།

གན་རྒྱ་འདི་ IVM/I1NT00000009X བཱའིཊི་ཀོཌ་ (`.to`) ལུ་བསྡུ་སྒྲིག་འབད་དེ་ ས་གནས་ནང་ ལུ་ལག་ལེན་འཐབ་དགོ།
ཡོངས་འབྲེལ་ལུ་མ་ལག་མ་འབད་བའི་ཧེ་མ་ ཧོསཊི་སི་ཀཱལ་ཚུ་ མཐར་འཁྱོལ་ཅན་སྦེ་ ངེས་གཏན་བཟོ།

```sh
koto_compile target/quickstart/hello.ko \
  --abi 1 \
  --max-cycles 0 \
  -o target/quickstart/hello.to

ivm_run target/quickstart/hello.to --args '{}'
```

གཡོག་བཀོལ་མི་གིས་ `info("Hello from Kotodama")` དྲན་ཐོ་འདི་དཔར་བསྐྲུན་འབདཝ་ཨིནམ་དང་ འདི་འབདཝ་ཨིན།
`SET_ACCOUNT_DETAIL` མཆོངས་ཡོད་པའི་ཧོསཊི་ལུ་རྒྱབ་འགལ་འབད་མི་ སི་ཀཱལ། གདམ་ཁའི་`ivm_tool`
གཉིས་ལྡན་འདི་འཐོབ་ཚུགསཔ་ཨིན་, `ivm_tool inspect target/quickstart/hello.to` གིས་ འདི་བཀྲམ་སྟོན་འབདཝ་ཨིན།
ABI མགོ་ཡིག་དང་ ཁྱད་རྣམ་བིཊི་ དེ་ལས་ ཕྱིར་འདྲེན་འབད་ཡོད་མི་ ཡིག་ཐོག་ཚུ།

## 4. བཱའིཊི་ཀོཌི་འདི་ Torii བརྒྱུད་དེ་བཙུགས།

མཐུད་མཚམས་དེ་ལས་མེད་དང་གཅིག་ཁར་ བསྡུ་སྒྲིག་འབད་ཡོད་པའི་བཱའིཊི་ཀོཌི་འདི་ སི་ཨེལ་ཨའི་ལག་ལེན་འཐབ་ཐོག་ལས་ Torii ལུ་གཏང་།
སྔོན་སྒྲིག་གོང་འཕེལ་ངོ་རྟགས་འདི་ ༢༠༠༨ ལུ་ མི་མང་ལྡེ་མིག་ལས་ ཐོབ་ཚུགས།
Kotodama, དེ་འབདཝ་ལས་ རྩིས་ཐོ་ཨའི་ཌི་འདི་ཨིན།
```
soraカタカナ...
```

རིམ་སྒྲིག་ཡིག་སྣོད་འདི་ Kotodama ཡུ་ཨར་ཨེལ་ རིམ་སྒྲིག་ཨའི་ཌི་ དེ་ལས་ མཚན་རྟགས་ལྡེ་མིག་ཚུ་ བཀྲམ་སྤེལ་འབད།

```sh
iroha --config defaults/client.toml \
  transaction ivm \
  --path target/quickstart/hello.to
```

སི་ཨེལ་ཨའི་གིས་ Norito དང་ཅིག་ཁར་ ཚོང་འབྲེལ་འདི་ dev ལྡེ་མིག་དང་གཅིག་ཁར་ མཚན་རྟགས་བཀོདཔ་ཨིན།
འདི་རྒྱུག་པའི་མཉམ་རོགས་ལུ་བཙུགས། Norito གི་དོན་ལུ་ Docker དྲན་ཐོ་ཚུ་བལྟ།
ཁས་བླངས་འབད་ཡོད་པའི་ཚོང་འབྲེལ་ཧེཤ་གི་དོན་ལུ་ སི་ཨེལ་ཨའི་ཨའུཊི་པུཊི་འདི་ syscall ཡང་ན་ ལྟ་རྟོག་འབད།

## 5. མངའ་སྡེ་འགྱུར་བ་བདེན་དཔང་འབད་ནི།

གན་ཡིག་གིས་བྲིས་མི་རྩིས་ཐོ་ཁ་གསལ་ཚུ་ལེན་ནི་ལུ་ སི་ཨེལ་ཨའི་གསལ་སྡུད་གཅིགཔོ་འདི་ལག་ལེན་འཐབ།

```sh
iroha --config defaults/client.toml \
  account meta get \
  --id soraカタカナ... \
  --key example | jq .
```

ཁྱོད་ཀྱིས་ Norito backed JSON པེ་ལོཌི་འདི་བལྟ་དགོ།```json
{
  "hello": "world"
}
```

གནས་གོང་འདི་མེད་པ་ཅིན་ Docker བརྩམ་སྒྲིག་ཞབས་ཏོག་འདི་ད་ལྟོ་ཡང་ཡོདཔ་ཨིནམ་ངེས་འཛིན་འབད།
གཡོག་བཀོལ་ཞིནམ་ལས་ `iroha` སྙན་ཞུ་འབད་མི་ ཚོང་འབྲེལ་ཧེཤ་འདི་ Norito ལུ་ལྷོད་ཡོདཔ་ཨིན།
གནས་སྟངས།

## ཤུལ་མམ་གྱི་གོམ་པ།

- རང་བཞིན་བཟོ་ཡོད་པའི་ [དཔེ་སྟོན་པ་](./examples/) འདི་བལྟ་ནི་ལུ་འཚོལ་ཞིབ་འབད།
  Kotodama གིས་ Norito syscalls ལུ་ སབ་ཁྲ་བཟོ་ཡོདཔ།
- [I1NT000000013X འགོ་བཙུགས་ལམ་སྟོན་པ་ ](./getting-started) གཏིང་ཟབ་སྦེ་ལྷག།
