CI བཟོ་བསྐྲུན་, གི་ཊི་ཧབ་ཤོག་ལེབ་བཀྲམ་སྤེལ་དང་ ལག་ཐོག་ལས་ དུ་ཁའི་བརྟག་དཔྱད་ཚུ་གིས་ དབྱེ་ཚན་ག་ར་ཁྱབ་ཚུགསཔ་ཨིན།
