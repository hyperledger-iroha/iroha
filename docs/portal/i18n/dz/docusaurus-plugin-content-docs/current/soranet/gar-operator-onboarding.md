བསྐྱར་ལོག་འབད་བཏུབ་པའི་ ཨེསི་ཨེན་ནེཊི་-༩ བསྟར་སྤྱོད་རིམ་སྒྲིག་འདི་ གསར་བཏོན་འབད་ནི་ལུ་ ཐུང་ཀུ་འདི་ལག་ལེན་འཐབ།
རྩིས་ཞིབ་མཐུན་སྒྲིག་བྱ་རིམ། དབང་འཛིན་བསྐྱར་ཞིབ་དང་གཅིག་ཁར་ ཆ་སྒྲིག་འབད་ནི་ཨིནམ་ལས་ བཀོལ་སྤྱོད་པ་རེ་རེ་ལུ་ ཆ་སྒྲིག་འབད།
འཇུ་བྱེད་དང་སྒྲུབ་བྱེད་བཀོད་སྒྲིག་ཅོག་འཐདཔ་ཚུ་ལག་ལེན་འཐབ་ཨིན།

## གོམ་པ་།

1. **བསྡུ་སྒྲིག་རིམ་སྒྲིག་**།
   - `governance/compliance/soranet_opt_outs.json` ནང་འདྲེན་འབད།
   - ཁྱོད་རའི་ I18NI000000005X གིས་ བདེན་ཁུངས་བཀལ་ཡོད་པའི་ བདེན་ཁུངས་བཞུ་ཚུ་དང་གཅིག་ཁར་ མཉམ་བསྡོམས་འབད།
     [ཁྲིམས་མཐུན་བསྐྱར་ཞིབ་ནང་](gar-jurisdictional-review).
2. **ནུས་ལྡན་**
   - `cargo test -p sorafs_orchestrator -- compliance_policy_parses_from_json`
   - `cargo test -p sorafs_orchestrator -- compliance_example_config_parses`
   - གདམ་ཁ་ཅན་: I18NI0000008X
3. **ཚད་གཞི་གི་སྒྲུབ་བྱེད་**།
   - `artifacts/soranet/compliance/<YYYYMMDD>/` གི་འོག་ལུ་ གསོག་འཇོག་འབད་ནི།:
     - I18NI000000010X (མཐའ་མཇུག་དང་མཐུན་པའི་སྡེབ་ཚན་)
     - I18NI000000011X (URIs + བཞུ་ཚུགས།)
     - བདེན་དཔྱད་དྲན་ཐོ་ཚུ།
     - མིང་རྟགས་བཀོད་ཡོད་པའི་ PDFs/I18NT0000000X ཡིག་ཤུབས་ཚུ་ལུ་གཞི་བསྟུན་འབདཝ་ཨིན།
༤ **ཤུགས་ལྡན་**
   - བསྐོར་ཐེངས་ (`gar-opt-out-<date>`), reploy orchetrator/SDK རིམ་སྒྲིག་ཚུ་ རྟགས་བཀལ།
     དང་ རེ་བ་བསྐྱེད་ས་ལུ་ དྲན་ཐོ་ཚུ་ནང་ `compliance_*` བྱུང་ལས་ཚུ་ ངེས་གཏན་བཟོ།
5. **ཁ་བསྡམས།**།
   - གཞུང་སྐྱོང་ཚོགས་སྡེ་དང་གཅིག་ཁར་ སྒྲུབ་བྱེད་བསྡུ་སྒྲིག་འབད་ནི།
   - ཇི་ཨར་ དྲན་ཐོ་ཀི་དེབ་ནང་ ཤུགས་ལྡན་བཟོ་ནིའི་སྒོ་སྒྲིག་ + ཆ་འཇོག་འབད་མི་ཚུ་ལུ་ ནང་བསྐྱོད་འབད།
   - ཁྲིམས་དོན་བསྐྱར་ཞིབ་ཐིག་ཁྲམ་ནང་ལས་ ཤུལ་མའི་བསྐྱར་ཞིབ་ཀྱི་ཚེས་གྲངས་ཚུ་ དུས་ཚོད་བཀོད་དགོ།
