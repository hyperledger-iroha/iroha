## 1. ནང་བཅུད་བསྡུ་བསྡུས།

- **Bulletin:** ཡུ་རོ་པིན་ལྷན་ཚོགས་ “བློ་གཏད་ཅན་གྱི་རྒྱལ་དར་དང་ མགྱོགས་དྲགས་སྦེ་ བཏོན་བཏང་ནིའི་འགན་ཁུར་ ཌི་ཨེན་ཨེསི་/ཧོསི་ཊིང་བྱིན་མི་ཚུ་གི་དོན་ལུ་” (Ref. DSA/2026/1219).
- **Key དགོས་མཁོ་:** བློ་གཏད་ཅན་གྱི་ཕེལ་གཱར་འདི་ བཀོད་སྒྲིག་འབད་བའི་ལཱ་གི་རྒྱུན་རིམ་འདི་དཔར་བསྐྲུན་འབད་ནི། ངེས་གཏན་ཆེ་བའི་བརྡ་དོན་གྱི་དོན་ལུ་ <4 ཆུ་ཚོད་ཀྱི་ལན་འདེབས་དུས་ཚོད་དང་ མི་མང་རྗེས་འཇུག་ཆ་མཉམ་ནང་ ཁྲིམས་ལུགས་དམིགས་བསལ་གྱི་ལོག་སྤྱོད་འབྲེལ་བ་ཚུ་ ཕྱིར་བཏོན་འབད།
- **བཙུགས་པའི་ཚེས་གྲངས་:** ༢༠༢༦‐༡༢‐༢༠.
- **དུས་ཚོད་:** 2027‐01‐31 (DG CONNECT ལས་བརྟེན་ བསྟར་སྤྱོད་གསལ་བསྒྲགས་)།

ཨེསི་ཨེན་ཨེསི་ཆོག་ཐམ་དོན་ཚན་༡༡ པའི་ཤེས་རྟོགས་འདི་ ༢༠༢༦‐༡༢‐༢༠ ལུ་ གཞུང་སྐྱོང་རྗེས་འདེད་འབད་མི་ (ཊིཀཊ་ I18NI0000000010X) ལུ་བཙུགས་ཡོདཔ་ཨིན།
