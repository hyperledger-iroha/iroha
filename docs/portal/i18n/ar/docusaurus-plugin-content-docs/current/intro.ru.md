# شكرًا جزيلاً على البوابة الإلكترونية للمصممين SORA Nexus

تقوم بوابة المصمم SORA Nexus بتقديم المستندات التفاعلية والمواد التعليمية وواجهة برمجة التطبيقات (API) الملائمة للمشغلين Nexus والمساهمين Hyperledger Iroha. بالإضافة إلى وثائق الموقع الأساسية، والإرشادات للخطة الأولى، والإرشادات والمواصفات العامة، على سبيل المثال هذا المستودع. تتضمن درجة الإقراض النقاط الموضوعية Norito/SoraFS والملصقات المصورة OpenAPI والتعليقات البث المباشر Norito الذي يمكن للمساهمين من الحصول على عقد للتحكم في التدفق، وليس ضخ المياه المواصفات.

## ما يمكن القيام به اليوم

- **تعلم Norito** - ابدأ بالملاحظة والبدء السريع لتعمل على تسلسل النماذج والأدوات bytecode.
- **تثبيت SDK** - تابع التشغيل السريع لـ JavaScript وRust الآن؛ تتوافق أنظمة Python وSwift وAndroid مع مجرد إيصالات الترحيل.
- **عرض واجهة برمجة التطبيقات (API) الصحيحة** - يعرض الجزء Torii OpenAPI المواصفات الفعلية REST، وتكوينات اللوحة يتم الرجوع إلى معايير Markdown الأساسية.
- **إضافة تحديث** - دليل التشغيل للاستثناءات (القياس عن بعد، التسوية، تراكبات Nexus) ينتقل من `docs/source/` ويتحرك مرة أخرى حول مجرد الهجرة.

## حالة تيكيوسي

- ✅ الإقراض الموضوعي Docusaurus v3 مع مطبعي عادي وبطل/بطاقات متدرجة وموارد تايل، بما في ذلك الماء البث المباشر Norito.
- ✅ الاتصال Torii OpenAPI متصل بـ `npm run sync-openapi`، مع التحقق من الصور التوضيحية وحماية CSP-guard، مثال `buildSecurityHeaders`.
- ✅ يتم تشغيل تغطية المعاينة والتحقيق في CI (`docs-portal-preview.yml` + `scripts/portal-probe.mjs`)، ثم قم بتنزيل المستند عبر البث، وبدء التشغيل السريع SoraFS، وقوائم المراجعة المرجعية قبل النشر قطعة أثرية.
- ✅ Quickstarts Norito وSoraFS وSDK بالإضافة إلى أقسام خاصة موجودة في اللوحات المربعة؛ يتم عرض الواردات الجديدة من `docs/source/` (البث والتنسيق وسجلات التشغيل) منذ أكثر من كتابها.

## كيفية الحصول على السعادة

- سم. `docs/portal/README.md` للأوامر المحلية (`npm install`، `npm run start`، `npm run build`).
- يتم عرض المزيد من محتوى الترحيل من خلال خريطة الطريق المحددة `DOCS-*`. قم بإغلاق الخصوصية - قم بتمرير الأقسام من `docs/source/` وقم بإضافة الجزء إلى اللوحة الخلفية.
- إذا قمت بإضافة قطعة أثرية عامة (المواصفات، جداول التكوين)، قم بتوثيق أوامر القيادة التي قد تكون مفيدة لك تعرف على.
