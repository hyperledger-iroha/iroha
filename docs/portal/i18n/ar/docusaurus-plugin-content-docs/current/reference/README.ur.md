هناك حلقة Iroha تحتوي على مادة "المواصفات المسبقة". هناك عدة صفحات كاملة من المعلومات والمقاطع الصوتية التالية.

## آج دستیاب

- **نظرة عامة على برنامج الترميز Norito** - `reference/norito-codec.md` لنسخ مواصفات `norito.md` المستندية، قم بإدراج شريط الجدول في الجدول.
- **Torii OpenAPI** - `/reference/torii-openapi` Redoc تم تحديث Torii وهو أحدث إصدار من مواصفات REST. سيتم تحديث المواصفات `npm run sync-openapi -- --version=current --latest` (تحتوي الإصدارات التاريخية الإضافية على لقطة شاشة لـ `--mirror=<label>` شاملة).
- **جداول التكوين** - مكمل كتالوج المعلمات `docs/source/references/configuration.md` میں رکھا جاتا ہے۔ عندما لا يكون الاستيراد التلقائي للبورتات صحيحًا، فإن الإعدادات الافتراضية وتجاوزات البيئة مناسبة لميزة Markdown.
- **إصدار المستندات** - شريط التنقل هو القائمة المنسدلة للإصدار `npm run docs:version -- <label>` التي تحتوي على لقطات مجمدة دکھاتا ہے، جس سے تطلق إرشادات جلدية موزعة بسهولة ويسر.
