# Bienvenido al Portal de sarrolladores de SORA Nexus

تجمع بوابة مطوري SORA Nexus الوثائق التفاعلية ودروس SDK ومراجع API لمشغلي Nexus والمساهمين في Hyperledger Iroha. قم بتكملة الموقع الرئيسي للمستندات من خلال عرض الممارسات والمواصفات التي تم إنشاؤها مباشرة من هذا المستودع. تتضمن صفحة البداية الآن نقاط إدخال موضوعية لـ Norito/SoraFS ولقطات OpenAPI ثابتة ومرجع مخصص لـ Norito Streaming لجذب المساهمين إلى عقد خطة التحكم في البث بدون البحث عن المواصفات العالية.

## Lo que puedes hacer aqui

- **اكتشف Norito** - ابدأ بالوصف العام ودليل البدء السريع لفهم نموذج التسلسل وأدوات الكود الثانوي.
- **البدء في تطوير مجموعات SDK** - متابعة عمليات التشغيل السريع لـ JavaScript وRust Hoy؛ تتوافق أساليب Python وSwift وAndroid مع نقل البيانات.
- **استكشاف مراجع واجهة برمجة التطبيقات** - تعرض الصفحة OpenAPI من Torii مواصفات REST أحدث، وتعرض جداول التكوين العناصر الأساسية في Markdown.
- **إعداد العروض** - يتم ترحيل دفاتر التشغيل التشغيلية (القياس عن بعد، والتسوية، وتراكبات Nexus) من `docs/source/`، والتوافق مع هذا الموقع مسبقًا للترحيل.

## الحالة الفعلية

- الوصول إلى Docusaurus v3 بالموضوع، والطباعة المُجددة، والبطل/البطاقات ذات التدرجات، وشرائح الموارد التي تتضمن السيرة الذاتية لـ Norito Streaming.
- البرنامج المساعد OpenAPI من Torii متصل بـ `npm run sync-openapi`، مع التحقق من اللقطات الثابتة وحماية تطبيقات CSP من خلال `buildSecurityHeaders`.
- تغطية المعاينة والتحقق من CI (`docs-portal-preview.yml` + `scripts/portal-probe.mjs`)، والآن قم بحظر مستند البث وبدء التشغيل السريع لـ SoraFS وقوائم المراجعة المرجعية قبل نشر العناصر.
- عمليات التشغيل السريع لـ Norito وSoraFS وSDKs، جنبًا إلى جنب مع أقسام مرجعية، وهي موجودة في الشريط الجانبي؛ استيراد جديد من `docs/source/` (البث المباشر والتنسيق وسجلات التشغيل) يمكن تعديله بهذه الطريقة.

## كومو مشارك

- استشارة `docs/portal/README.md` لأمر التطوير المحلي (`npm install`، `npm run start`، `npm run build`).
- يتم وضع مهام ترحيل المحتوى بجوار العناصر `DOCS-*` من خريطة الطريق. قم بتقييم المساهمة: قم بفتح الأقسام من `docs/source/` وقم بإضافة الصفحة إلى الشريط الجانبي.
- إذا قمت بإضافة منتج تم إنشاؤه (المواصفات، وجداول التكوين)، قم بتوثيق أمر البناء حتى يتمكن المساهمون المستقبليون من التحديث بسهولة.
