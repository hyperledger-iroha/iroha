## خطافات التقارير

- Chaque mercredi, mettre a jour le tableau ci-dessus et l'issue دعوة نشطة مع une note Courte (دعوات المبعوثين، Relecteurs Actifs، الحوادث).
- عند انتهاء الأمر بشكل غامض، قم بإضافة مسار استئناف التعليقات (على سبيل المثال `docs/portal/docs/devportal/preview-feedback/w0/summary.md`) ثم من `status.md`.
- إذا تم إلغاء معيار الإيقاف المؤقت لـ [معاينة تدفق الدعوة](./preview-invite-flow.md)، فقم بإضافة خطوات الإصلاح قبل إعادة عرض الدعوات.
