# تجربة سابقة على بورنغ

##جائزہ

DOCS-SORA جول ويلبر لرحلة لانجري الحربية. يبني المجموع الاختباري بوابات
(`npm run serve`) ومضبوطة جربها فلو سنگ ميل کولتے ہیں: پبلاك برييويو ے وسیع پيمانے پر
يتفوق هذا النشاط على ريوورز الفائقين في مجال الرياضة. لقد تم جمع هذه البطاقة الرائعة،
لقد تم إنشاء شيء ما، وسرد هذا الشيء، وشركة محفوظات من خلال هذه الشركة.
التخطيط المشترك، وإيقاع الدعوة، وصادرات المعادن
[معاينة تدفق الدعوة](./preview-invite-flow.md) مميز؛ بعض المراحل في لعبة فوككس
تم اختياره كمنتخب جديد بعد تتويجه للمرة الأولى.

- **السكوب:** ومعاينة مستندات GA الأحدث من ريوورز (`docs-preview.sora`، إصدارات GitHub Pages، أو حزم SoraFS) تم نشرها.
- **الآفاق:** Torii أو SoraFS المثبطات (مجموعات الإعداد الإضافية) وعمليات نشر البرامج (مميزة) [`devportal/deploy-guide`](./deploy-guide.md)).

## رولز وبيري ريكويز

| رول | عام اداف | مطلوبہ تحف | أخبار |
| --- | --- | --- | --- |
| المشرف الأساسي | اختبارات الدخان الجديدة هي اختبارات الدخان. | مقبض GitHub، جهة اتصال Matrix، موقعة CLA فائل پر۔ | عمومًا، تم الانتهاء من `docs-preview` GitHub؛ يوجد أيضًا درجة أعلى من درجة الائتمان التي يمكن تدقيقها. |
| مراجع شريك | تم تطوير مقتطفات حزمة تطوير البرامج (SDK) أو محتوى الإدارة بشكل سريع. | البريد الإلكتروني للشركة، إثبات المفهوم القانوني، شروط المعاينة الموقعة۔ | ٹیلی ميتر + متطلبات معالجة البيانات هي الهدف المطلوب. |
| متطوع مجتمعي | احصل على تعليقات حول سهولة الاستخدام. | مقبض GitHub، جهة الاتصال المفضلة، المنطقة الزمنية، CoC قبولیت۔ | سيارة إسعاف صغيرة؛ اتفاقية المساهمين |

جميع الأقسام التي تحتاج إليها:

1. معاينة القطع الأثرية ذات الاستخدام المقبول.
2. ملاحق الأمان/قابلية المراقبة
   ([`security-hardening`](./security-hardening.md)،
   [`observability`](./observability.md)،
   [`incident-runbooks`](./incident-runbooks.md)).
3. لقطة منخفضة تخدم الرقم التسلسلي `docs/portal/scripts/preview_verify.sh` يتم قبولها مرة أخرى.

## تناول ورک فلو

1. تمتع بالتجربة
   [`docs/examples/docs_preview_request_template.md`](../../../examples/docs_preview_request_template.md)
   إنشاء مزرعة (أو إصدار نسخ/لصق). كم من بطاقة التسجيل: المعرفة، الارتباط عبر الطريق،
   يتم التحقق من مقبض GitHub وتواريخ المراجعة والمستندات الأمنية.
2. يقوم متتبع `docs-preview` (إصدار GitHub أو تذكرة الحوكمة) بتصنيف وتخصيص الموافق.
3. قبل ريكويز أي لعبة:
   - CLA / اتفاقية المساهم فائل پر موجود ہو (یا مرجع عقد الشريك)۔
   - إقرار الاستخدام المقبول
   - تقييم المخاطر مکمل ہو (مثال: المراجعون الشركاء کو قانونيون لا يوافقون کیا ہو)۔
4. لا مشكلة في تسجيل الخروج ومشكلة التتبع التي قد تؤدي إلى إدخال إدارة التغيير أيضًا
   (مثال: `DOCS-SORA-Preview-####`)۔

## التزويد و ٹولنگ

1. **إنشاء القطع الأثرية** — سير عمل CI أو SoraFS pin هو أحدث واصف معاينة + ملف أرشيف
   (`docs-portal-preview` قطعة أثرية)۔ المؤرخون الذين يرجع تاريخهم إلى ما يلي:

   ```bash
   ./docs/portal/scripts/preview_verify.sh \
     --build-dir build \
     --descriptor artifacts/preview-descriptor.json \
     --archive artifacts/preview-site.tar.gz
   ```

2. **يعمل تطبيق المجموع الاختباري على خدمة البطاقة** — ريورز هي إدارة رأس المال ذات المجموع الاختباري:

   ```bash
   DOCS_RELEASE_TAG=preview-<stamp> npm run --prefix docs/portal serve
   ```

   لقد تم إعادة استخدام `scripts/serve-verified-preview.mjs` من خلال بنية غير مؤكدة لم يتم التحقق منها.

3. **مراجع GitHub (اختاري)** — إذا كانت هناك تقارير في فروع غير منشورة، فما عليك سوى مراجعة مدتها لفترة طويلة
   `docs-preview` GitHub يشمل كريات وعضوية تعدل درجة رغبتك.

4. **قنوات الدعم المفضلة** — الاتصال عند الطلب (Matrix/Slack) وإجراءات الحادث
   [`incident-runbooks`](./incident-runbooks.md) كل شيء جديد.

5. **القياس عن بعد + التعليقات** — يقوم الناشرون بجمع التحليلات مجهولة المصدر
   (ديكا [`observability`](./observability.md)). قم بالدعوة إلى نموذج الملاحظات أو نموذج الإصدار
   والحدث هو [`preview-feedback-log`](./preview-feedback-log) المساعد الذي يوفر ملخصًا للموجة.

##ريوور چيک لٹ

أحدث التقارير السابقة التي تم نشرها مؤخرًا:

1. التحقق من العناصر التي تم تنزيلها کریں (`preview_verify.sh`)۔
2. `npm run serve` (أو `serve:verified`) يعمل حارس المجموع الاختباري لبوابة لانچ بشكل فعال.
3. ملاحظة أولية تتعلق بالأمن وإمكانية المراقبة.
4. لا يتم استخدام OAuth/Try it وحدة التحكم في تسجيل الدخول برمز الجهاز (إذا كان قابلاً للتطبيق) ورموز الإنتاج المميزة.
5. نتائج التتبع المتفق عليها في نطاق البحث (الإصدار أو المستند المشترك أو المزرعة) وعلامة إصدار المعاينة وعلامة التصنيف.

## صيانة وصيانة

| رحلةہ | اجراءات |
| --- | --- |
| ضربة البداية | هذه هي القائمة المرجعية للسحب التي تحتوي على السليكون، المصنوعات اليدوية + بطاقات الائتمان، [`preview-feedback-log`](./preview-feedback-log) التي تحمل `invite-sent` عبر الإنترنت القراءة، وإذا قمت بمراجعة واحدة تتضمن الكثير من الأشياء، فستساعدك على مزامنة نقطة المنتصف. |
| الرصد | معاينة القياس عن بعد (غير عادي، جربه جيدًا، اكتشف الفشل) وإذا حدث خطأ ما، فستكون بمثابة دليل تشغيل للحادث. النتائج التي توصلت إليها أحداث `feedback-submitted`/`issue-opened` لا تعتمد على مقاييس الموجة بشكل صحيح. |
| الخروج من الطائرة | يعرض GitHub أو SoraFS صفحات الويب، و`access-revoked`، وملف أرشيف الملفات (ملخص التعليقات + الإجراءات المتميزة، وتسجيل المراجعين). وبالتالي فإن السجل يبني سجلًا صافيًا و[`docs/examples/docs_preview_feedback_digest.md`](../../../examples/docs_preview_feedback_digest.md) يهضم منسلك. |

تدور الموجات التي تدور فيها الموجات في نفس الوقت الذي يتم فيه استخدام الكرة. الريبو هو مسار ورقي
(الإصدار + القوالب) يتم إجراء عملية التحقق من خلال DOCS-SORA والسجل القابل للتدقيق والحوكمة الذي يتم التحقق منه بعدة طرق
معاينة الوصول إلى عناصر التحكم الموثقة تتوافق مع ذلك.

## قوالب الدعوة والتتبع

- التوعية هي بداية
  [`docs/examples/docs_preview_invite_template.md`](../../../examples/docs_preview_invite_template.md)
  فائل سے کریں. إنه قانون زابان، ومعاينة المجموع الاختباري لليوميات، ومن المتوقع أن يتضمن النص
  ريوورز الاستخدام المقبول باليس تسيليم كريں.
- قالب بطاقة الاتصال `<preview_tag>` و`<request_ticket>` والعناصر النائبة لقنوات الاتصال بدل.
  الرسالة النهائية التي تحتوي على تذكرة دخول واحدة هي أهم المراجعين والمعتمدين والمدققين
  هناك المزيد من المواصلات.
- دعوة جارية بعد تعقب جدول البيانات أو إصدار الطابع الزمني `invite_sent_at` وتاريخ الانتهاء المتوقع
  كل هذه الخطوات [معاينة تدفق الدعوة](./preview-invite-flow.md) تقرير عن المجموعة النموذجية أيضًا.
