## تاريس النشطة (W0)

- تم تحديث عناصر الاختبار المبدئي (تم تنفيذ GitHub Actions `docs-portal-preview` 2025-03-24، وتم التحقق من الواصف عبر `scripts/preview_verify.sh` باستخدام العلامة `preview-2025-03-24`).
- خطوط الأساس للقياس عن بعد (`docs.preview.integrity`، لقطة من لوحات المعلومات `TryItProxyErrors` محمية في الإصدار W0).
- تم حظر نص التوعية باستخدام [`docs/examples/docs_preview_invite_template.md`](../../../examples/docs_preview_invite_template.md) مع معاينة العلامة `preview-2025-03-24`.
- طلبات الإدخال المسجلة للخمسة الأوائل من المشرفين (التذاكر `DOCS-SORA-Preview-REQ-01` ... `-05`).
- أول خمس دعوات تم إرسالها 2025-03-25 10:00-10:20 بالتوقيت العالمي المنسق بعد مرور ستة أيام متتالية من القياس عن بعد الأخضر؛ يتهم الحراس في `DOCS-SORA-Preview-W0`.
- مراقبة القياس عن بعد + ساعات عمل المضيف (تسجيلات الوصول اليومية حتى 31-03-2025؛ سجل نقاط التفتيش حتى الآن).
- تعليقات على آخر تحديثات / مشكلات تم جمعها وتصنيفها `docs-preview/w0` (الإصدار [W0 Digest](./preview-feedback/w0/summary.md)).
- استئناف النشر + تأكيدات نشر الدعوات (حزمة النشر بتاريخ 08-04-2025؛ الإصدار [W0 ملخص](./preview-feedback/w0/summary.md)).
