# المجموع الاختباري سے شروط پریویو پلان

لقد تم إصلاحه واستبداله بكاميرا بيانية وهو إصدار سابق من النسخة التجريبية التي تم نشرها بشكل جيد لموسيقى رائعة. تهدف هذه الخطة إلى إنشاء ألعاب فيديو ودردشة فيديو لريوورز CI، ولا يمكن تغيير سجل المجموع الاختباري للفيديو، والملف SoraFS ذري Norito يمكن توصيله بسهولة.

## اداف

- **البنيات الممتعة (الحتمية):** يمكن استخدام `npm run build` بشكل جيد من خلال إنشاء و`build/checksums.sha256` للخارج.
- **تم إجراء فحص سابق:** تم نشر مقال سابق حول المجموع الاختباري للصناعة اللازمة ولم يتم نشره بعد.
- **Norito شائع الاستخدام:** برامج السكرتير المسبق (التزام بالمعلومات، وملخص المجموع الاختباري، وSoraFS CID) وNorito JSON في ما يتعلق بمهمة الأطفال، فإن أولز ريليز هي لعبة تزلج.
- **آبرولنج:** رحلة استكشاف سكريبت الحربية التي تستخدمها على كل سكاي (`./docs/portal/scripts/preview_verify.sh --build-dir build --descriptor <path> --archive <path>`); المجموع الاختباري للسكربت + الواصف الحقيقي للفكرة هو بمثابة سمة. الإدارة المسبقة (`npm run serve`) مع `docusaurus serve` تعمل على تحسين جهاز الكمبيوتر وتجميعه على أقل عدد ممكن من المجموع الاختباري تحت المجموع الاختباري رہیں (و`npm run serve:verified` معروف بشكل واضح برقرار).

## مرحلہ 1 — CI نفاذ

1. `.github/workflows/docs-portal-preview.yml` هذه العملية:
   - تم إنشاء Docusaurus بعد `node docs/portal/scripts/write-checksums.mjs` چلايا جايے (الإصدار پرپہلے ہی چلتا ہے).
   - `cd build && sha256sum -c checksums.sha256` چلايا جئے ولا يتوافق مع جاب فيل ہو۔
   - قم بإنشاء ملفات `artifacts/preview-site.tar.gz` للملفات النصية والمجموع الاختباري لملفات الفيديو المصغرة وملفات `scripts/generate-preview-descriptor.mjs` وملفات JSON الثابتة `scripts/sorafs-package-preview.sh` الجيل (`docs/examples/sorafs_preview_publish.json`) يوفر لك سير العمل المزيد والمزيد من المزايا SoraFS.
   - الموقع الإلكتروني، وهو عبارة عن عالم إبداعي (`docs-portal-preview`، I18NI00000032)، وSoraFS بنك (`docs-portal-preview-sorafs`) قم بتنزيله في الواقع، سيارة جديدة وخطة لإعادة بناء كل سيارة جديدة.
2. يتضمن طلب السحب المجموع الاختباري لخلاصة البحث ومعلومات CI الكبيرة (يحتوي `docs-portal-preview.yml` على برنامج GitHub Script المفيد).
3. `docs/portal/README.md` (سلسلة CI) يعرض سير العمل ويقوم بنشر قائمة التحقق التي توضح مراحل التحقق.

## توثيق سكرپٹ

`docs/portal/scripts/preview_verify.sh` يمكنك تنزيل المقالة السابقة لأداة البحث `sha256sum`. لقد بدأ استخدام لقطة شاشة لوكل وقت `npm run serve` (أو معروف بشكل واضح `npm run serve:verified`) في استخدام لقطة شاشة لـ `docusaurus serve`. منطقة التوثيق:

1. مناسب SHA ٹول (`sha256sum` أو `shasum -a 256`) أو `build/checksums.sha256` لخلاف چلاتا.
2. قم باختراع النسخة التجريبية الأولى من `checksums_manifest` لتوزيع/توزيع الأسماء الملخصة/الملفات، إذا كانت النسخة الأولية، عبارة عن نسخة تجريبية من المقترحات الملخصة/التفضيلية.
3. لا يتوافق هذا أيضًا مع ما لا نهاية من الختم وما إلى ذلك من التسجيلات السابقة للتزلج الأسود.

مثال للاستخدام (CI آرٹیفیکٹس نکالنے کے التالي):

```bash
./docs/portal/scripts/preview_verify.sh \
  --build-dir build \
  --descriptor artifacts/preview-descriptor.json \
  --archive artifacts/preview-site.tar.gz
```

لقد قامت شركة CI وRilies Engineers التي تعمل في مجال سكريبت الآن بتمويل من بنك ريو برييو أو ريليز كل ما هي لعبة مانسل كريك المجانية.

## مرحلہ 2 — SoraFS اشاعت

1. سير العمل المسبق يشمل هذه الطريقة:
   - `sorafs_cli car pack` و`manifest submit` تم نشرهما مرة أخرى مع SoraFS ووحدة التخزين المرحلية.
   - واپس هذه الملخصات وSoraFS CID كمحفوظات.
   - `{ commit, branch, checksum_manifest, cid }` إلى Norito JSON میں سریلائز کرے (`docs/portal/preview/preview_descriptor.json`).
2. قم بإنشاء سكربت افتراضي لملف محفوظ وسحب طلب يظهر CID.
3. يتضمن هذا الموقع الإلكتروني `sorafs_cli` الذي يمكن أن يؤدي إلى مستقبل واعد في المستقبل وفقًا للقرار المتبع.

## رحلة 3 — گورنس وآڈٹ

1. Norito سكيم (`PreviewDescriptorV1`) شائع جدًا وهو عبارة عن سكاكر صغير وهو `docs/portal/schemas/` تحت الإنشاء.
2. قائمة مراجعة النشر DOCS-SORA للقيام بذلك:
   - `sorafs_cli manifest verify` الذي تم تنزيله CID من خلاف ذلك.
   - ملخص المجموع الاختباري للفيديو و CID الذي يبرز تفاصيل العلاقات العامة.
3. تقوم إدارة السيارات بتدوير ريليز وود من خلال المجموع الاختباري للفيديو الذي يلتقط شيئًا ما.

## كلے سوالات

- SoraFS كا كون سا ماحول النسخة التجريبية السابقة (التدريجي مقابل حارة المعاينة الخاصة)؟
- هل تم نشر هذه النسخة من الإصدار السابق من جهاز سكربت آخر (Ed25519 + ML-DSA)؟
- هل سير عمل CI هو `sorafs_cli` عندما يقوم الأوركسترا بالترتيب (`orchestrator_tuning.json`) فكس كرني هاهئے تاكه مينى فيستس قابل للتخصيص؟

تم الانتهاء من `docs/portal/docs/reference/publishing-checklist.md` من خلال درج آخر وحل أمور أخرى واضحة من أجل تحقيق المزيد من النجاح.
