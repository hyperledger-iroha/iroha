   يبدأ مع زوج الوصف/الارشيف ديم. لم تتطلب اي تجاوزات.
2. **ملاحظات الهجوم** - تم تسجيل مشكلتين بسيطتين في الشريط الجانبي
   (`docs-preview/w0 #1-#2`). كلاهما محالتان الى Docs/DevRel ولا تعرقلان
   بل.
3. **تكافؤ runbook في SoraFS** -طلب sorafs-ops-01 روابط متقاطعة اوضح بين
   `sorafs/orchestrator-ops` و `sorafs/multi-source-rollout`. تم فتح المشكلة
   متابعة؛ عالج قبل W1.
4. **مراجعة القياس** - إمكانية الملاحظة-01 ان `docs.preview.integrity`,
   `TryItProxyErrors` شركات وكيل Try-it الخضراء؛ لم تنطلق تنبيهات.
