# مجلة الهجرة SoraFS

تعرض هذه المجلة سجل الترحيل المتزايد والموثوق في هندسة RFC
SoraFS. قم بكتابة المجموعة في الأعلى والإشارة إلى الوضع والأوامر المزعجة
والنشاط المطلوب. خطة هجرة دبي الجديدة من خلال هذه المنطقة و RFC
(`docs/source/sorafs_architecture_rfc.md`)، لنقل المستهلك إلى المصب
في المساعدة.
