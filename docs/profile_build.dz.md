# ངོ་སྤྲོད་ `iroha_data_model` བཟོ་བསྐྲུན་།

`iroha_data_model` ནང་ བཟོ་བསྐྲུན་གྱི་རིམ་པ་ཚུ་ མགྱོགས་དྲགས་སྦེ་ འཚོལ་ཞིབ་འབད་ནི་ལུ་ གྲོགས་རམ་ཡིག་ཆ་འདི་ གཡོག་བཀོལ།

```sh
./scripts/profile_build.sh
```

འདི་གིས་ `cargo build -p iroha_data_model --timings` གཡོག་བཀོལ་ཞིནམ་ལས་ དུས་ཚོད་ཀྱི་སྙན་ཞུ་ཚུ་ `target/cargo-timings/` ལུ་བྲིས།
བརྡ་འཚོལ་ནང་ `cargo-timing.html` ཁ་ཕྱེཝ་དང་ ཀེརཊིསི་ཡང་ན་ བཟོ་བསྐྲུན་གྱི་གོ་རིམ་ག་ཅི་གིས་ དུས་ཚོད་མང་ཤོས་འགོརཝ་ཨིན་ན་ བལྟ་ནི་ལུ་ དུས་ཡུན་གྱིས་ ལཱ་ཚུ་དབྱེ་སེལ་འབད།
