# Hyperledger Iroha

[![لائسنس](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](https://opensource.org/licenses/Apache-2.0)

Hyperledger Iroha ایک سادہ اور مؤثر بلاک چین لیجر ہے جو
**distributed ledger technology (DLT)** پر مبنی ہے۔ اس کا ڈیزائن جاپانی
فلسفہ "کائی زن" (Kaizen) سے متاثر ہے، جس کا مقصد غیر ضروری بوجھ (*muri*)
کو ختم کر کے مضبوط اور قابلِ اعتماد نفاذ فراہم کرنا ہے۔

Iroha آپ کو اکاؤنٹس، اثاثوں اور آن‑چین ڈیٹا اسٹوریج کو مؤثر اسمارٹ
کانٹریکٹس کے ذریعے منظم کرنے میں مدد دیتا ہے، اور ساتھ ہی بیزنٹائن فیلیرز
اور کریش‑فیلیرز دونوں کے خلاف برداشت فراہم کرتا ہے۔
